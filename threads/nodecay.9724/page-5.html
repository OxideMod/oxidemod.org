    It should work with TwigsDecay, that said, depending on the settings NoDecay could cancel some of TwigDecay's work. If you set Stone to zero in NoDecay and have stone decay in TwigsDecay, Stone will NOT decay. You can however have TwigsDecay decay all twig and just leave NoDecay twig to 1 and you'll be fine.

    NoDecay adjusts decay on ALL entities that get Decay damage. That said, inner foundations (in vanilla) do not decay, so technically it only controls decay on outside foundations. Only exception to this would be if you are using TwigsDecay, I assume this plugin decays ALL twig parts, regardeless of where they are placed.
