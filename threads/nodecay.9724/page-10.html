    That's exact. The plugin does not alter the tick frequency, only the damage itself. The damage is fractional so the multipliers are very precise, there is no rounding to worry about here. Note that the "grace" period before items start to decay is not handled by this plugin. So yes, FTM, you can simply increase the multiplier if you want stuff to decay faster than normal.. but there is no such thing as "everything decays in 3 days precisely".
