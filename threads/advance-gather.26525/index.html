    HOUGAN_Y submitted a new resource:

    AdvanceGather - Custom gathering with some action's and extension drop

    This plug-in should diversify the extraction of resources by players, it also implements a fruit collection system in the game, etc.

    Default config file:
    Chance to get berry from hemp : 0 - 100
    How often players will get berry from hemp (0 - to disable)

    Max berry amount : 0 - 9999999
    How many berries can player get from one hemp

    Max berry stack: 0 - 9999999
    How many berries can be stored in one stack

    Enable broadcast : true/false...
