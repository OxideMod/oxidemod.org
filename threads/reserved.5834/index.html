    Wulf submitted a new resource:

    Reserved - Reserves a number of slots so that whitelisted players can connect


    Reserved is a simple plugin that reserves a configurable number of slots so that whitelisted players can connect.

    Chat Commands
    • /reserved add player (or steamid)
      Add player to reserved whitelist.
    • /reserved remove player (or steamid)
      Removes player from reserved whitelist.
    • /reserved slots #
