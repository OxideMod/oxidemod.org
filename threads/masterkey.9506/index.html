  1. Wulf Community Admin

    Wulf submitted a new resource:

    Master Key - Allows players with permission to unlock anything

    Master Key allows players with permission to unlock doors, storage boxes, and even authorizing when entering restricted cupboards' radius. Usage is logged by default to the dated masterkeys_##-##-####.txt file in the oxide/logs directory.

    Permissions
    This plugin uses Oxide's permission system. To assign a permission, use oxide.grant user <username|steamid> <permission>. To remove a permission, use oxide.revoke user <username|steamid> <permission>....
