  1. Mughisi

    http://oxidemod.org/resources/crafting-controller.695/ said:
    This is a plugin that allows you to change the crafting speed on your server. Admins can change the crafting speed by using the command/craft <speed> where speed is a number between 0 and 100 and stands for the % of the normal speed.

    Examples:
    /craft 0 -> instant craft
    /craft 50 -> half craft
    /craft 75 -> 3/4 craft

    Another function that was added to this plugin is the ability to control which items that are able to be crafted and which aren't.
    You can simply block an item by using the command /block "item name"

    Examples:
    /block "Camp Fire"
    /block "Lock"
    /block "Tool Cupboard"

    When you want the players to be able to craft the item again you simply use the command /unblock "item name".

    Examples:
    /unblock "Camp Fire"
    /unblock "Lock"
    /unblock "Tool Cupboard"
