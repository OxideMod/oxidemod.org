    To change the crafting rate you simply run the command /craft <value:0-2> where 0 is instant crafting and 2 is double crafting duration, so in your case you would simply run /craft 0
    It's actually all explained in the overview here: http://oxidemod.org/plugins/crafting-controller.695/

    The only part that's not yet explained in the overview is the batch crafting while pressing the left shift key (run) or left ctrl key (duck) which you can configure in the CraftingController.json file.
