    Ya I have a HUGE update coming later today! It should fix all of these issues. I also added an Admin menu where admins can blacklist (add and remove) items; as well as put the background image selection process there.

    The items are now "moved" instead of recreated so that solves many of the issues. Stacks are split appropriately as well.

    One thing I also had to add was a check for stack changes. So for example, if you list 1000 wood then remove 500 from the stack it will not trigger the removal of the item from the market (as moving items would) because the hook I am using which checks for removal of items from container does not sense stack changes. Therefore, instead when the item is attempted to be purchased the purchase fails because the amount is different then what was originally listed. The item is then removed from the market.

    Anyway... you will all see. Not saying there will be no issues but overall the entire thing is running much better!
