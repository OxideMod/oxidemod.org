  6. ignignokt84

    Xrl69 said: ↑
    so how do i add a effect to the config or do i just change the 3 it uses?
    Click to expand...
    The triggerEffect can only contain one effect, but effects is a list that can contain as many effects as you wish. You can replace the existing effects, or add/remove as many as you wish.
    Code:
    "effects":[// explosion effect list
  "assets/prefabs/weapons/rocketlauncher/effects/rocket_explosion_incendiary.prefab",
  "assets/bundled/prefabs/fx/gas_explosion_small.prefab",
  "another/effect.prefab",
  "and/another/effect.prefab"
]
     
    ignignokt84, Sep 22, 2017
