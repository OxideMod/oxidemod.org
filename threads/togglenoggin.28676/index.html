    ToggleNoggin
    Allows your to bind a key to a console command that spawns, equips and ignites a candle hat or miners hat.

    Console Command(s):
    togglenoggin

    Example Usage:
    togglenoggin
    togglenoggin 1

    Permissions:
    ToggleNoggin.admin

    Quick Start:
    -Install Mod
    -Set admin permission ie: oxide.grant user username ToggleNoggin.admin
    -Bind key(s) to console command:

    bind k togglenoggin
    - This binds the k key to the candle hat.

  11. carny666

    TheConnor110 said: ↑
    Would be awesome if the miners hat and candle hat had different commands so it wasn't one of the other.
    Click to expand...
    If you pass an argument (anything, true or a number) to either the console command or the chat command the plugin will spawn a miner's hat, with no argument supplied a candle hat is spawned.

    for example,

    '/togglenoggin true' in chat will render a miner's hat, provided the user had the proper privilege, admin or miner.

    just '/togglenoggin' (no argument) in chat will render a candle hat, provided the user had the proper privilege, admin or candle.

    same with the console command.
