  1. OuTSMoKE

    OuTSMoKE submitted a new resource:

    Realistic Weather - Command based weather that realisticly rolls in & out.

    This is a command based plugin designed to mimic real life weather, with whatever weather system you chose to start gradually building and leaving over time instead of a straight on/off. Clouds will roll in, eventually leading to rain slowly building, which in turn brings in fog, etc.

    This is designed to be used with a timed command plugin such as Timed Execute, and replace the built in weather, so I recommend also using Timed Execute to set all weather...
