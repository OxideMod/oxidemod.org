    Delete your 'steamcmd' folder in the server root and then run the update again. After that, start the server and check the version. If the version is 739.98, proceed with the a fresh download of the latest Oxide build. Once you've installed Oxide, check the version again. If the version is 739.88, you somehow got an outdated Oxide build, so rinse and repeat.
