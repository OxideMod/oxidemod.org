    Can I suggest that you add this as a sub-command of the /stakes command? If you are using the livemap.io plugin, the /share command from it shares your position on the map with another player.

    I think it would be more user friendly if this new command was something like:
    /stakes share <playername> [distance]

    This way you understand that you are sharing your stakes. We have had a few players that ask or trick other newbie players into using the /share command with their name, which gives away their position to that player. It would be much, much worse if they manage to trick them into using this version of the share command.

    Matt
