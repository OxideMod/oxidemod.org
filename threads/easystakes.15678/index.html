    EasyStakes - Fill all authorized stakes with amber from your inventory from anywhere

    Permission:
    easystakes.use

    How to Add Permission:
    /grant user UserName easystakes.use

    To grant permission for everyone use
    /grant group default easystakes.use

    Commands:
    /stakes
    This will show you many stakes you have authority on and how much amber you need to fill them all.

    /stakes fill

  19. Noviets

    To extend how much time the amber gives you change the ServerConfig variable called: StakeDeauthTime

    The default value is: 432000f (which is 5 days in seconds).

    What ever you set that value too, it will be divided by 5 for each amber.
