    ConnectMessages - Provides connect and disconnect messages.

    Provides configurable connect and disconnect messages.

    Default config:
    Code:
    {
  "Settings": {
    "ShowConnectMessage": true,
    "ShowDisconnectMessage": true,
    "ShowDisconnectReason": false
  }
}

    English:
    Code:

  "DisconnectMessage": "{0} has disconnected.",
