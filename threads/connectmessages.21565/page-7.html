    Set the permissions in the config of NTele

    "Home": {
    "HomesLimit": 2,
    "VIPHomesLimits": {
    "nteleportation.vip": 5 <--- This is the Permission for set homes
    },
    "Cooldown": 275,
    "Countdown": 15,
    "DailyLimit": -1,
    "VIPDailyLimits": {
    "nteleportation.vip": 5

    then oxide.grant group <groupname> <permission>
