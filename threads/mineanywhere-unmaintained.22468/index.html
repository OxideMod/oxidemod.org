  1. Calytic Community Admin Community Mod

    Calytic submitted a new resource:

    MineAnywhere - Mine all the rocks, all the wood, all the junk, even the ground

    MineAnywhere will turn almost everything into a minable resource.

    Things that are minable with MineAnywhere includes, but is not limited to:
    • Logs
    • Mountains
    • Rocks
    • Monuments
    • Caves
    • Buildings
    • Junk Piles
    • Player Structures
    • Deployables
    • Grass
    • Snow
    • Sand
    How does it work?

    Anytime you hit anything, a resource node is created. You can configure the depletion rate, gather rate, node decay, distance between nodes, and specific resources...
