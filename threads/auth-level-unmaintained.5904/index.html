    Wulf submitted a new resource:

    Auth Level - Temporarily sets the auth level for players via command


    Auth Level is a basic plugin that temporarily sets the auth level for players via command. The player that you set the auth level on will only retain that level during that sessions, as disconnecting and reconnecting will reset their auth level to their normal level.
