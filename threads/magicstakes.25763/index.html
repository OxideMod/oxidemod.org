    Magic Stakes - Set which stake you spawn at via command.

    A simple plugin that allows you to set which stake you spawn at via command.

    View attachment 37683

    Usage:
    /setspawn - Will show you a list of stakes you're authorized on with IDs.
    /setspawn id - Will set your spawnpoint to stake of specific ID.

    Permissions:
    Code:
