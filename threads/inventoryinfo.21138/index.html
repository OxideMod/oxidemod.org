    InventoryInfo - Saves player inventory info to file

    This plugin saves player inventory info to oxide/data/InventoryInfo files. Admin/owners can access server files can look players inventory there. If a player loses items while they are offline, inventory information can be found there. Every time when a player logs out, the information is overwritten.
