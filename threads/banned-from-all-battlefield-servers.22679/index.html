  2. sqroot

    This is the wrong place to ask to get unbanned.
    No, all server bans are local, unless the server owner uses something like my BanSync plugin to share bans with other servers (there's probably other alternatives too). This usually implies that the server owners using the same ban database all trust each other.
    The only thing that will ban you globally is VAC.
    It would be terrible if server bans were global, as you do not even need to rent a server to host a server, anyone can just download the server files and host a server themselves.
