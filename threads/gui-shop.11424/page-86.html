    Click to expand...
    ONE SOLUTION:
    Install this mod TimedPermission and replace the command line in the GUIShop.json with something like:

    old line: "cmd": ["o.grant user $player.id nteleportation.vip"],
    new line: "cmd": ["grantperm $player.id nteleportation.vip 45d"],

    In my example, it gives the $player.id 45 days of the permission nteleportation.vip.

    Hope this inspires anyone out there?
