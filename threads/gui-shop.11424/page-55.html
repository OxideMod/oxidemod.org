    I have found and fixed one of the issues you had on your config file. Basically. Pretty much every item in the file was put incorrectly.

    For example, you had this in your config file: "item": "Hunting Bow",

    The correct way to do it is this: "item": "hunting bow",

    Under the item part, you should never use capital letters. I have done you a massive favour and gone through the WHOLE thing and changed them all. You should now have no issues getting your items.

    HOWEVER, there is a second issue in your config file that I will not touch as I will leave it up to you to figure out what you want where.

    You're only supposed to have one "chat" shop in your config file. You have 2 or 3. Items you want to appear in /shop all need to be merged into one "shop" rather than 2 or 3.

    You can still have as many NPC shops as you'd like with as many different items in each shop as you want.

    Upload your new config file and do oxide.reload GUIShop in the console and then /shop should work (when you've set it up properly that is).

    If you need help getting it into one then I can help you a bit more.
