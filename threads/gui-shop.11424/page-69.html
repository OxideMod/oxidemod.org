    No Name Brand said: ↑
    I would like to request a feature, an option to remove the 'buy all' button. Occasionally a player hits it by accident and loses all their money buying something they don't want. I implemented it on my own server by changing one line, in the if statement before the button is made.

    If you could make this an option set in the configuration file, it would be great. I think a lot of people would find it useful.

    Code:
                if (!cooldown && sell)
            {
                container.Add(new CuiButton
                {
                    Button = { Command = $"shop.{(sell ? "sell" : "buy")} {shop} {item} all", Color = color },
                    RectTransform = { AnchorMin = $"{(sell ? 0.775 : 0.5) + steps.Length * 0.03 + 0.001} {ymin}", AnchorMax = $"{(sell ? 0.805 : 0.53) + steps.Length * 0.03 - 0.001} {ymax}" },
                    Text = { Text = "All", FontSize = 15, Align = TextAnchor.MiddleCenter }
                }, ShopContentName);
            }
            return container;
