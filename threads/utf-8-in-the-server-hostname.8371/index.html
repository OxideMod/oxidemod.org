  3. BasKoning

    If you sort modded servers to show the bottom of the nameslist: you can see chinese-like characters. So I think it might be possible.

    I am trying to insert a white-hand-pointing-to-the-right symbol myself in the hostname.
    You could do this is by typing it directly into the shortcut-target.

    So: create shortcut to RustDedicated.exe
    then rightclick - properties that shortcut
    make target something like this:
    RustDedicated.exe +server.hostname "type your unicode characters here"

    Or you can create a folder named cfg in the same map where your RustDedicated.exe is, with in it a file named server.cfg. Open it with notepad, and insert something like:
    Code:
    server.secure true
server.maxplayers 100
server.worldsize 4000
server.port 28015
server.hostname "type your unicode characters here"
server.identity "YourIdentity"
load
    Save as: coding: utf8

    then create a shorcut (you have to startup in batchmode and give a seed) to:
    RustDedicated.exe -batchmode +server.seed 9999
    The config file will be loaded as default values. Including your unicode name.

    Then, if you run that shortcut, it will show up in unicode in the serverslist.
    (the serverconsole will still display ?? I hope that is not a problem for you)
