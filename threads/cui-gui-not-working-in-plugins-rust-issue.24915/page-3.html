    How do i update the client only? Last night i had to reinstall the whole server because my server didn't update? My server is hosted at my.gameservers? (Sorry for the noob question, new admin here)
    Click to expand...
    When a weekly update hits it's usualy a Server & Client update. They first deploy the server update so as a server owner you can download it and install it before the client update hits. Once FacePunch has update all their official servers they deploy the client update.

    If you client version is different from the server version you can not join the server.
    However, sometimes they can deploy small client-side only patches that don't require a server update.

    If your server is hosted by gameservers and it's a managed server, you can not do this update manually and gameservers do it for you (usually a lot slower than unmanaged servers because they have multiple customers). If you had to reinstall the whole server something must have went wrong, it's best to contact your server provider or host the Rust server yourself (unmanaged or dedicated), that way you have full control of the update process.
