  2. Wulf Community Admin

    The "Compiler disconnected" message generally means that either your CSharpCompiler is outdated, or you are missing a dependency such as Visual C++ 2013 x86, but likely the first. When updating and installing Oxide, make sure that you are replacing ALL files with those from the latest Oxide zip. If you are using a host and uploading via FTP, make sure you have full permission to upload ALL files, otherwise you'll end up with some not being updated.
