  2. Wulf Community Admin

    You can use an RCON tool such as RustAdmin or use Oxide and plugins with Modded set to false in the oxide.config.json.

    Keep in mind that Facepunch only allows "admin tools" under the Community section though, so anything other than that could get your server blacklisted. I don't see an issue with a player list command though, but game mechanic changes and visual changes I'd avoid.
