  10. JMJ

    If anyone is still having issues with the bindings not working, you can set them manually in the f1 console with these commands

    bind q craftui.toggle to open the ui

    bind escape craftui.closeoverlay;craftui.close to close the ui as well as the quick craft that gets stuck.

    Hope this helps!
