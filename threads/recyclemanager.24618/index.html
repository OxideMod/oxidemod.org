  1. redBDGR

    redBDGR submitted a new resource:

    RecycleManager - Easily change features about the recycler

    A simple plugin that lets you increase the usefulness of recyclers.

    Permissions:
    • recyclemanager.admin - used for any admin commands
    Commands:
    • /addrecycler - spawns a recycler at your feet (need admin priv)
    Configuration:
    Code:
    {
  "Settings": {
    "Blacklisted Items": [
      "explosive.timed"
    ],
    "Recycle Output Multipliers": {
      "*": 1,
      "metal.refined": 1
    },
    "Recycle Time": 5.0
  }
}
