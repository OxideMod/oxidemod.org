    I really wasn't going to reply, but I see you're having difficulty. Quoting a post isn't just for you, it's also for everyone else. In this way we easily follow conversation replies and comments. If you read the overview or understood this plugin, you would understand amount per spawn location is not an embedded feature. It would be odd and obscure to have multiple animals spontaneously spawn in one spot. You can do that manually however. The appropriate syntax for the plugin is as follows:

    /cas add <number of animal>

    That's it; there is no amount of animals variable. None of the methods call or use a "amount of animals" variable within the code. If you had the capacity to figure this out, you would have. You do /cas, no clip to the spot, left click, and repeat. An animal promptly appears after clicking. Creating a single spot with multiple spawns requires clicking the mouse however many times (in that one spot). You could have also copy and pasted the entry generated in the data file multiple times over. I.e. location: 1000.7 -23.3 200.2 prefab/boar. The plugin would then spawn the same prefab in the same spot as many times as you'd like. Before you're done, see if there's a plugin to remove the stick out of your ass.
