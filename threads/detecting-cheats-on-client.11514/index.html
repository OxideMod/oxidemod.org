    You would have to write server-side checks that detect the behavior the cheat itself is creating. For instance if the cheat allows to fly, you could check for player position vs a straight down raycast, or check a player's speed, etc. Most of these are already checked by the FP hack checks though so you'd need to figure a unique thing this cheat does that you can pick up on the server side.

    It's not possible for now to create client-side plugins unfortunately.
