    Yes, go to the docs in the top section of the website then navigate down to Styling Messages then follow those steps. To use a custom color where it says color= put a color code in.
    EXAMPLE:

    Code:
    Wounded_Info: "<color=green>test</color>"
    It is manditory to always end a color with </color>
