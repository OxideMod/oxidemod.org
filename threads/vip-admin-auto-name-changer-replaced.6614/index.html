    TheDoc submitted a new resource:

    VIP / Admin - Auto Name Changer - VIPs, Admins, Auto, Name, Change, Changer

    This plugin will auto-rename any player connecting to your server found in the VIP or Admin list

    Here an example Config file. Make sure that you add your Admin's and your VIP's SteamID's to the lists below.

    Code:
    {
  "Admins": [
  "99999999999999991",
  "99999999999999992"
  ],
  "Vips": [
  "99999999999999991",
  "99999999999999992"
  ],
  "Settings": {
  "AddVipTag": "true",
  "AdminTag": "(Admin) ",
  "VipTag": " ]ViP[",
  "AddAdminTag": "true"
  }
}
    Enjoy...
