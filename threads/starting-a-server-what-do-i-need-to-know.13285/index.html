  2. Ankawi

    Use NFOServers, they're friendly and have almost instant support. You must learn how to manually create a server however if you intend on purchasing a virtual computer from them. I do believe that they portforward it automatically so that will not be necessary. As for editing plugins, that is not also necessary, most plugin owners are quite active and are quick to fix their plugins. Read more about Rust servers and prepare yourself on how to create and manage a Rust server. I could probably create one easily within 10 minutes or so, although the first time it took me was about 2 days to get it fully working. To edit the plugin's configurations files, simply go to the configs folder and see what you want to change. But you must be careful in not creating errors through misplaced quotation marks or commas. You must also know how to update your server manually, which is also quite easy to do. A lot of time and effort goes into a Rust server. Good luck.

    P.S Look for tutorials on how to setup your server, or just add me on steam and if I have some time I'll help you out.
