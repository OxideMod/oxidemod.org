  9. Visagalis

    Visagalis updated Sticky Chat with a new update entry:

    0.0.3

    • Now it's really ready for other plugin's integration.
    Author of other plugins (which are affecting chat) must use this call:
    Code:
               var StickyChat = plugins.Find("StickyChat");
            if (StickyChat != null)
            {
                if ((int)StickyChat.Call("PlayerStickyState", player) != 0)
                    return false;
            }
    Returned state shows in what chat toggle player currently are.
    • GENERAL = 0
    • CLAN = 1
    • PRIVATE =...
