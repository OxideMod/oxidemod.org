    I don't think the salt is needed, but I believe the seed is if you don't want a random map each restart.
    Server owners can specify a server salt in addition to the server seed, which is unknown to people who connect to the server and is used to scramble the locations of mining resources. By default the salt, similar to the seed, is calculated from a unique hardware ID, but I recommend setting it via the startup parameters (server.salt) and changing it regularly, for example with every wipe.
