  2. Mr. Tea

    Hey 2dles, there sure is a plugin that allows you to change the minimum and maximum air drop frequency. I've listed it below:

    Airdrop Extended for Rust | Oxide

    In game simply type this in to the console:
    aire.minfreq <frequency_in_seconds> - The minimum amount of time to wait before an airdrop
    aire.maxfreq <frequency_in_seconds> - The maximum amount of time to wait before an airdrop
    aire.save - To save your configuration
