  3. King Bob Bare Foot

    @BlueEyes425 This plugin does change the gather rate for the game. so if you want to change the amount of wood you get from a wood log or coal from a coal node.
    Code:
    "GlobalStackSizeMultiplier": 1.0,
  "Version": 48,
  "VersionConfig": 3,
  "LootConfigs": {
    "None": {
      "RollCount": 1,
      "RollWithoutReplacement": true,
      "LootResult": null,
      "Children": []
    },
    "OwnershipStake": {
      "RollCount": 99999,
      "RollWithoutReplacement": true,
      "LootResult": {
        "ItemId": "Amber",
        "StackSize": 2,
        "RandomVariance": 0

    so this is the first line of code, if you want to change the loot drop of every single thing in the game to 6x, you will change the "GlobalStackSizeMultiplier":1.0, to "GlobalStackSizeMultiplier":6.0, (Assuming you are playing HurtWorld not sure about other games)
    [DOUBLEPOST=1491617288][/DOUBLEPOST]
    Code:
    "Key": 1.0,
                "Value": {
                  "RollCount": 1,
                  "RollWithoutReplacement": true,
                  "LootResult": {
                    "ItemId": "Wood",
                    "StackSize": 24,
                    "RandomVariance": 6
    this is another line of code for a SPECIFIC item in the game (again that game is hurtworld) So the "Key" is the Percentage drop of the item. "Stacksize" is the amount you want the item to drop (in my case I have it at 24 wood per log) "randomVariance" is probability of a number of logs you get (so in my case I have it "6" so it will drop between IDK 24-30 logs or something like that). That really all that you need to mess with. The rest of it does not need to be messed with.
    (Note: FYI this is just the way I know how to do it if my information is wrong please feel free to correct me.)
