  16. Dragosani

    To answer a couple of the posts above - it's not LootConfig that is changing people, it's HURTWORLD. The game has patched itself a few times over the last few weeks. When that happens YOUR LOOTCONFIG will reset to a default setting.....The lootconfig/json (that you HAD working) with have the older version number in it and it will be replaced by a NEW LOOTCONFIG file with the newest version number......

    What admins NEED to do when Hurtworld is patched, is to LOOK at the new LootConfig.json file and make a note of the version number (last patch went from version 37 3 to 38 3). Then you should also see the LootConfig.json.old file alongside (which has all your your specific configs in for 3x 5x whatever)....So, change the lootconfig.json.old to reflect the new version number, save it, delete the .json file then rename the .json.old file to .json

    Problem solved.

    No need for sexism here - it is frustrating when hurtworld patches, but it's not the creators of the plugins that are at fault here - they're working on new fixes as soon as they see a hurtworld patch.....

    If you're still having problems with your lootconfig after trying the above, post it here and I'll have a look.
