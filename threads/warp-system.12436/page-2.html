    All Oxide commands for Reign of Kings work in the chat, just prefix them with / like any other chat command. The information the plugin author posted a few posts up was incorrect. Oxide provides console commands via the chat due to the lack of in-game console.

    You also do not need to prefix the commands with "oxide", you can use them without it.
    Example: /grant user Bob warp.admin

  4. CaptFoxX

    First and foremost i would like to thank the developer for making this beautiful plugin, so with a little trial and error i finally have to admit that this was no easy task. Yes this is a great feature for VIP access players but i managed to do it for all just to simply and make it simple.

    When you have this plugin uploaded onto your server go in-game and type:

    /grant user (username) warp.admin

    This grant you access to to actually adding the warp as an admin wise. Next, you would want to go to your designated area like (im using example from Stormwall Map) the oil fields behind the gorge. So when you have a desired location type this

    /warp add OILFIELD O O all

    that is not an "O" it is a zero value therefore it should be instant teleport there and then proceed to add the warp additions to desired locations. Hope this helps.
