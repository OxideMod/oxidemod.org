    Scorpyon submitted a new resource:

    Alliance Tracker - Form alliances with other guilds. Yes, you and THIS army...

    With this plugin, players and their respective guilds can form lasting alliances with other guilds.
    This is extra helpful when declaring wars!

    Player Commands
    Use /alliancecommands to view all plugin commands below in-game:

    /requestally <player_name> Request to start an alliance with this player's guild
    /viewallies <guild_name> - View all allies for this guild
    /viewallyrequests - View all ally requests for your guild
    /acceptally <guild_name> - Accept an ally request from this guild...
