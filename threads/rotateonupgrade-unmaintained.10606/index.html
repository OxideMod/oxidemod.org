  1. KeyboardCavemen

    KeyboardCavemen submitted a new resource:

    RotateOnUpgrade - Allows you to rotate and/or demolish your building part after upgrading it from twigs.

    Since the suggestion to be able to rotate building parts after upgrading them has been up on reddit a lot recently, we decided to create a plugin that allows this but only if you upgrade from twigs. This way this mechanic can't be abused by raiders and gives you acces to this nice feature that Garry himself wil hopefully implement soonâ„¢.

    You can edit the plugin to your preferences by editing the following lines in RotateOnUpgrade.json:
    Code:
      "allowDemolish": true,...
