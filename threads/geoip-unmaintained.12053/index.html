    Wulf submitted a new resource:

    GeoIP - Provides a geographic information API for IP addresses

    GeoIP provides an API for plugins to obtain IP address geographic information. The plugin downloads and uses a JSON-formatted MaxMind GeoLite2 database locally, so no web requests are needed beyond the database download and updates.

    Keep in mind that alone this does nothing, as it's an API meant for plugins to utilize. To use this with a supported plugin, simply install like you would any other plugin!
