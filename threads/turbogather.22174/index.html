    TurboGather - Give a short gather rate boost with a cooldown afterwards

    This is my first "bigger" plugin that I have written, so any feedback / ‘constructive criticism’ would be much appreciated.

    This plugin will allow players, with the appropriate permissions, to type a chat command and get a resource gather rate boost for a certain amount of time with a cooldown afterwards.
