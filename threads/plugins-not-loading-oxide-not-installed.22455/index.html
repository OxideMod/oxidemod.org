    These are the steps I'd take normally for updating/installing:
    1. Shutdown the server entirely, kill it if needed
    2. Run SteamCMD or whatever updater your host provides for the game
    3. Make sure your server is shutdown still, other stop it again
    4. If manually updating/installing Oxide, make sure ALL files are uploaded
    5. If updating/installing via a control panel, use "reinstall" if available, else use "uninstall" and "install" for Oxide
    6. Start the server, and then check the oxide.version command (no version = not installed properly)
