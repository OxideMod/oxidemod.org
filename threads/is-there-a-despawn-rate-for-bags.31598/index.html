  2. Mr_Jeff

    5 minutes is the default for corpses and 5 minutes for normal items.
    You could try changing corpsedespawn lower, this should make only corpses despawn faster.
    If you change itemdespawn lower, backpacks should despawn with the time you set but this only works for lower tier items. I think some higher tier items have different despawn times so backpacks that have say '556 ammo' that won't despawn in the time you set.

    In console use the bold text:
    Itemdespawn = 300 (default - 5mins)
    Corpsedespawn = 300 (default - 5mins)
