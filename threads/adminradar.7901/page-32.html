  13. nivex

    AK47 said: ↑
    when the radar is turned on, it causes too much server delay. is there a way?
    Click to expand...
    Use a refresh rate at or above 1 second and use fewer filters if your server can't handle the load.

    We use a high end machine and even it would experience a delay some times (from the very first AdminRadar to this version). This is usually only the case when your server is already experiencing latency issues, or it can't handle the load at the moment. Often restarting the server helped due to other performance issues.
     
    nivex, Jul 24, 2017
