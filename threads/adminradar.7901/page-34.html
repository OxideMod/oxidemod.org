  12. Wulf Community Admin

    @nivex, just a reminder that you can't have both of these right now:
    Code:
    bool IsRadar(BasePlayer player) => activeRadars.Any(x => x.player == player);
bool IsRadar(string id) => activeRadars.Any(x => x.player.UserIDString == id);
    Plugins will assume the first, which breaks compat with old I believe. Seeing that issue with AdminPanel right now. There is no signature handing with Oxide in plugin calls or hooks yet, so until there is you may want to only provide one method.
