    On the steam client: (your friend does this)
    1. Right click the steam icon by the system clock
    2. Click Servers
    3. Goto the Favorites tab
    4. Click Add Server
    5. Type in the WAN IP Address and Port to your server
    6. Click Add
    7. Does it appear?
