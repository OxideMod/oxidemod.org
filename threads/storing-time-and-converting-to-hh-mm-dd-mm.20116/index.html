  4. sqroot

    Code:
    var s = new DateTime(1970, 1, 1, 0, 0, 0, 0, DateTimeKind.Utc).AddSeconds(d).ToString("your format string");
    where "d" is your double and "your format string" is your formatting syntax as described by Custom Date and Time Format Strings.

  6. Caffeine

    You already got the answer you were after but if you wanted to store datetime in a MySQL database you could use

    Code:
    string GetMySQLDateTimeUTC()
{
       return DateTime.UtcNow.ToString("yyyy-MM-dd HH:mm");
  }
    this returns the dt as UTC which i've found is the best way to store it as you can then convert it to any timezone for display purposes. How to convert DateTime in Specific timezone?
