  17. Smeag

    @Funian - I don't know russian. You can make a new lang file for it, then russian users should see all messages as you customize it.

    @Mr.Gatsby™ - Just add the person as admin in the server. Currently, the plugin uses no permissions.

    @q1LLa - Yes, but then you need to define your own fall sites! The current default fall sites are defined for DiemensLand.

    @snack95 - As SouZa said, the drop chances are calculated with the count of online players. The calculation for each items is as follows:

    Code:
    {[(count of online players)² + 0.2f)] * (Meteors Loot Factor Config)} * (Item drop chance);
    So, in your case, you can set the (Meteors Loot Factor Config) to "5" with "/setmeteorslootfactor 5" to get fixed drops.

    @Vinerra - As SouZa said, the drops will be even better.


    Looks like the drop rate calculation need to be simpler.
