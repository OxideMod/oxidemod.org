  1. k1lly0u

    k1lly0u submitted a new resource:

    StorageCleaner - Removes left over UI imagery stored in file storage

    Since the last major Rust update stored imagery for UI has been saved as files to your servers disk. This has caused some issues since quite a few plugins utilize this method of image storage and now images are left behind every time the server is restarted leading to increased storage size.

    Now that plugins are taking those images and re-using them it simply deleting them each restart is not possible as the first plugin to remove them would stop other plugins from retrieving their own...
