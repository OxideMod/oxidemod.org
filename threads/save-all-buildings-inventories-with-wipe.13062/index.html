  3. Deicide666ra

    You could probably go from Copy/Paste and create a save of all structures using the same methods quite easily. That said, like Wulf said, most forced wipes come with slight to major changes in map generation so the map can be totally different after a wipe even if it's using the same seed, from landmass shape to elevations, river and rock placement, even rad towns, roads and mountains can move around when they change the code.

    The wipe we just had is a good example of this, a same seed generates a totally different map now than before the wipe.

    So even if you did save/reload all buildings, it would probably be terrible, you'd have bases in rad towns, floating bases, bases underground, bases in water, etc. You could of course make up a bunch of rules in the paste part to adjust elevation, move buildings away from rad towns / rocks, etc. but ultimately you're definitely swimming against a very strong current with this idea.
     
    Deicide666ra, Dec 4, 2015
