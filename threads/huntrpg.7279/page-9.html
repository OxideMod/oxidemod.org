  17. ViTaliY_VS

    gerrmanman said: ↑
    how do i reset everyones hunt levels?
    Click to expand...
    1 Way:
    oxide.unload HuntPlugin
    In oxide/data delete file Hunt_Data.json
    oxide.load HuntPlugin

    2 Way:
    Stop Your Server
    In oxide/data delete file Hunt_Data.json
    Start Your Server.

    It'll delete all info about players levels. They all will be 0 lvl.
