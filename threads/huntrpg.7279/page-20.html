  19. Nogrod

    EviL iNsiDe said: ↑
    as a feature request : it will be nice to integrate in the config file an option to multiply the xptable , for the moment manually need to change each (of 200) level xp . thanks .
    Click to expand...
    you can generate the xp table with console command 'hunt.genxptable 383 1.105 10 0.005' (BaseXP, LevelMultiplier, LevelModule, ModuleReducer) the given example are the default values to generate...the XP for every level is calculated with BaseXP * LevelMultiplier^Level and every 'LevelModule' the 'LevelMultiplier' is reduced by 'ModuleReducer'...

    yes you can try 0.1 or something else...multiply the value by 100 and it is the % chance, so 0.15 is 15% and 0.1 will be 10%
    for gaining levels i would adjust the xp table...for example increase LevelMultiplier or reduce ModuleReducer value to make it harder to get high levels
