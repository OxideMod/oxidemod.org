    @KHCloud You will need Economics in order to use a Plugin like GUIShop. This merely holds an 'account' for each user, which other Plugins then hook into. To earn money take a look at Rewards for Rust | Oxide, MoneyTime for Rust | Oxide, BarrelPoints for Rust | Oxide, GatherRewards for Rust | Oxide, Playtime Tracker | Oxide, Quests for Rust | Oxide, Event Manager for Rust | Oxide, Voter Rewards for Rust | Oxide, Also placing items 'for sale' in GUIShop is a good way for users to earn coins. Hope this helps
