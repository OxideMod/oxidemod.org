  1. Absolut

    Absolut submitted a new resource:

    ResetCodeLocks - Simple Plugin that allows players to reset the code lock (guest/main) on all their locks

    This plugin requires a user to have ResetCodeLocks.allow

    This plugin provides users with the ability to reset all CodeLocks they own from a simple chat command. This can reset Guest and Regular Codes.

    When you reset a Regular Code it will ensure the locks are also locked; whereas with the Guest it will only change the code.

    So if you want it locked, reset the main then the guest.
