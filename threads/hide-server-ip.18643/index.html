  2. sqroot

    No. Players need the IP to connect to your server, so it's trivial for any player to retrieve the IP of your server.
    Making it more difficult to find is also impossible since the IP is provided by the master server, which redirects players to your server, and for it do that, you have to fulfill its contract, which includes you giving the master server your IP to distribute it to players.
    You could buy a domain for the IP, but that wouldn't make any difference, as a domain is basically just an alias for a set of IPs.
