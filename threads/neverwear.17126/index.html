  1. k1lly0u

    k1lly0u submitted a new resource:

    NeverWear - Simple plugin to stop items from losing condition

    As requested by @Rebel 2

    A simple plugin to stop item wear. There are 3 config options, each will disable item wear for that category of item.

    Config
    Code:
    {
  "useAttire": false, // Used for armour
  "useTools": true, // Used for tools
  "useWeapons": false // Used for weapons
}
