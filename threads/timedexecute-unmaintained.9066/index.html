    TimedExecute - Execute commands every (x) seconds.

    What is "PaiN TimedExecute":
    PaiN TimedExecute is a very useful plugin that allows you to set in config which commands you want to execute every [x] seconds.

    EXPERIMENTAL VERSION:
    http://oxidemod.org/plugins/timedexecute.919/

    Default Config:
    Code:
    {
  "ShowTimedCommands": "true",
  "TimedCommands": [
    [
      "airdrop.drop",
      300
    ],
    [
      "say 'hello world'",
      60...
