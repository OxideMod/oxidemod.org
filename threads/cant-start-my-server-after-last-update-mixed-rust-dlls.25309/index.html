  8. Mughisi

    AlexTu said: ↑
    If oxide is outdated the server starts nevertheless. Or not ?
    Click to expand...
    Oxide patches some of the server files so every server update requires an update to Oxide. If you would use an older Oxide release you are technically reverting some of the server files back to their previous version which sometimes causes the server to spam errors, sometimes it appears to work and sometimes it won't start at all.
    The general rule is to wait before updating until a new Oxide update is out for the new version of the server.
