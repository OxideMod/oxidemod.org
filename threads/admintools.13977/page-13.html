    Hey mate, so TempBans work in Minutes, not seconds. So just double check that you've not added the ban in seconds.
    Secondly, editing the ban file isn't recommended. There's two commands in addition to the TempBan command which you can use to check and remove temp bans. /removetempban [IP/Name/SteamID] (You use their name, and can see their IP when they join,if you are unsure on their SteamID).
    Those same params can be used to check a temp ban /checktempban [IP/Name/SteamID] to see how long they have left.

    If you edit the list directly, you'll need to "/reload AdminTools" to reload the edited list as it will be stored in memory.
     
    Noviets, Apr 12, 2016
