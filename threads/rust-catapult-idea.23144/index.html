    Just Something I have been working on.
    Catapults for Rust !!! what do you guys think? (my video skills suck.. so be nice )



    Looking for ideas on what kind of features would be good.
    Geared mostly for primitive or medieval type servers. but anyone can have fun

    I was thinking of making is so whatever you put in the Box, it will "throw".
    You can put 500 stone, and it will throw the default rock.
    Or put a Human skull in there and it can throw a body? lol.
    Maybe if player is standing on the arm, it can throw that person. even better

    Currently you build a foundation, then just spawn the catapult on it. from there you can rotate it left or right.

    Current Damages are a little OP for testing and FUN factor at MAX ... LOL

    EDIT : (1/21/17)
    - Added a Build mode or Spawn mode for admins. Admin can make a foundation a Buildable catapult by anyone. Buildable foundations have a red Hammer floating/spinning over them. they can run up and hit it with hammers until all the pieces are built. Which will show up a few at a time when hitting until fully built. Or admins or player with perms, can just spawn complete catapult on foundation without the need for building it.
