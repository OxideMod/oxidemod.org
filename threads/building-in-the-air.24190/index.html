  2. Diesel 42o

    You would need to use Zones Manager for Rust | Oxide, and setup a zone with No Stability, explained on the overview, or you can turn server.stability false, but I do not recommend this option.

    If you use server.stability false, anybody on the server can build in the air, but with ZoneManager and a NoStability Zone, it is only within that small area.

    Untrue. You can use build and do something such as this /spawn foundation 10 2 (Height 10, building tier 2). Once you have the foundation, you'll be able to use /build foundation 0 2 to build next to the foundation.
