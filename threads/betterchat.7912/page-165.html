  6. Riddle

    For people that want to disable the Logging to Console, because maybe they also installed plugin Logger (thus, one chat message outputs two times). Since there is no config option in BetterChat and Logger provides more info throughout the game in Console, you can disable logging to console from BetterChat by commenting out line 1427 in BetterChat.cs (the plugin)

    //Puts(Group.Format(player, message, true));

    Maybe it's a good option to create a configuration line to enable/disable logging to console for this plugin?
