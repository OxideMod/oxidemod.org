    You have to open your old groups file with something like Notepad++ and replace following:

    HideIfNotHighestPriority => HiddenIfNotPrimary
    PlayerName => Username
    {Name} => {Username}
    Formatting => Format

    and then save the file in the new location: .../oxide/data/BetterChat.json
