    BetterChat used oxide groups. BetterChat group "owner" is the oxide group "owner". Meaning if you already have an oxide group called owner, it will apply the betterchat group to all of them.
    [DOUBLEPOST=1460550690][/DOUBLEPOST]To everbody having problems with BetterChat 4+

    Here the important informations once more as I am tired of telling everybody twice.

    Please ALWAYS read the update logs when updating. That is very important as you see. They contain important informations about what changed.

    Nothing changed about how BetterChat groups work. They use oxide groups. The BetterChat group "owner" will just be the oxide group "owner".
    BetterChat just uses permissions and adds that permission to the oxide group. You can only no longer customize the permissions of the groups. They are still there. For the "owner" group for example, it will be 'betterchat.group.owner'.

    The groups are no longer in the config file! They were moved to another file located at oxide/data/BetterChat/Groups.json to have a cleaner config file!

    You can edit that file just how you edited the config file before. Remember to reload the plugin after making changes.
    Only the formatting of the groups changed a bit and there is only one new setting called "HideIfNotHighestPriority"!

    HideIfNotHighestPriority hides the Title of that group if the groups does not have the highest Priority rank of the groups the player has.

    The Priority replaces the old Rank. Priority is used to order the groups and define which one is higher than the other one. It is not supposed to be set to 0 on every group! The smaller the actual value, the higher the Priority rank. For example:

    owner: 0
    admin: 1
    moderator: 2
    vip: 3
    player: 4

    I also created the BetterChat Group Manager to make editing grops even easier! It can be found on the overview!
    Simply load in the new Groups.json file, edit your groups and then save the file.

    Also the /chat command is there so you could also edit groups from your mobile per console. You can choose how you edit your groups. There are enough ways!

    • Groups.json file
      • Notepad++
      • BetterChat Group Manager
    • Command
