    In the oxide/data/betterchat/groups.json file, make sure you set hideifnothighestpriority to true. Make sure that Owner is Either 0 or 1 and then set the other ranks to correspond. So if Owner is 0, set the next group to 1 and so on. This makes it so that it will hide the default player tag if another tag they have is a higher priority.

    You can also download the better chat group manager for easier editing via http://laserhydra.com/downloadfiles/BetterChatManager/BetterChat Group Manager Installer.exe
