  16. Resistance

    nickname. said: ↑
    Can I delete part of cs file BetterChat where player with admin permissions get admin prefix ? Where I can find this part?
    Click to expand...
    Don't edit or delete part of the script just set it to hide the title from your groups.json in the data/betterchat folder
    Code:
    "GroupName": "admin",
    "Priority": 0,
    "Title": {
      "Hidden": true,
      "HideIfNotHighestPriority": true,
      "Size": 15,
      "Color": "#269ec3",
      "Text": "[Admin]"
    },
    "PlayerName": {
      "Size": 15,
      "Color": "#DCFF66"
    },
    "Message": {
      "Size": 15,
      "Color": "white"
    },
    "Formatting": {
      "Console": "{Title} {Name}: {Message}",
      "Chat": "{Title} {Name}: {Message}"
    }
  },
