  3. GR3AVE5Y

    It's via a control panel... It's got the plugins listed and you just click the ones you want to be installed and then it has a config area where you can edit all of the settings for the plugins.

    You click add plugin.
    Press save and this makes the plugin live.
    You then edit the plugins config.
    Press save again.
    Finally in the console you type oxide.reload PluginName and it obviously updates the plugin.

    I hope that's the answer you were looking for.

  13. Heatst0rm

    jackherroor said: ↑
    how is the consol command to make my moderators..to mod for name and no player?
    Click to expand...
    Type in RCON this

    To create a new group for betterchat
    oxide.group add <groupname>

    Give Permission for a group (for permission look at the json permission line)
    oxide.grant group <groupname> <permission>

    Add user to a group
    oxide.usergroup add <STEAMID64> <groupname>

    Note : You can find your SteamID64 at STEAMID I/O
