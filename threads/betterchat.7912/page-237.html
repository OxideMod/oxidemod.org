  13. Resistance

    Hider said: ↑
    the username's color can't change, always standart blue
    Click to expand...
    You can change it in the betterchat.json file found in the data folder
    "GroupName": "default",
    "Priority": 25,
    "Title": {
    "Text": "[Player]",
    "Color": "#55aaff",
    "Size": 15,
    "Hidden": false,
    "HiddenIfNotPrimary": true
    },
    "Username": {
    "Color": "#DCFF66", <------ Change this one
    "Size": 15
    },
    "Message": {
    "Color": "white",
    "Size": 15
    },
    "Format": {
    "Chat": "{Title} {Username}: {Message}",
    "Console": "{Title} {Username}: {Message}"
    }
    },
