  20. LaserHydra

    Mario m9m9 said: ↑
    Hello I am using my new server with this plugin and have all of the groups sorted out but for some reason I allways appear as admin and owner at the same time with countless saves and also I have the clans io plugin but does not seem to appear when I have better chat in my server
    Click to expand...
    As you're an admin on your server, oxide adds you to that group each time.
    Therefor you can use BetterChat's HiddenIfNotPrimary setting.

    chat group set admin HiddenIfNotPrimary true

    Make sure you have set your Priorities correctly.
    Priority: 0 > 1
