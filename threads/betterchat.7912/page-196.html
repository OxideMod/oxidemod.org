    Alright, so the file location was relocated and renamed, so here's the fix
    - Go to where your old Groups.json file was "oxide\data\BetterChat\"
    - Copy (or cut) it and paste it in "oxide\data\"
    - Delete BetterChat.json
    - Rename Groups.json to BetterChat.json
    - Reload plugin

    If this helped please like this post <3
