  13. Evilyn Corporated

    Lots of questions about getting clan tags to work with the latest version of BetterChat. I've run some tests and have good news and bad news.

    There are THREE clans plugins that I know of, and TWO of them support clan tags with the latest version of BetterChat.

    Rust:IO Clans by dcode - Does NOT support clan tags in the latest BetterChat.
    Clans by k1llyou - DOES support clan tags in the latest BetterChat.
    Clans Reborn by Fujikura - DOES support clan tags in the latest BetterChat. (And IMO the better of all three.)

    The problem would appear to be that if you're using Rust:IO clans, you don't get any clan tags. Until dcode updates Rust:IO clans to support the latest version of BetterChat, you're not going to see them.

    So stop asking LaserHydra to update BetterChat as there's nothing wrong with it. Instead address all your requests to the Rust:IO Clans support thread and you might get your problem resolved.

    OR, use one of the other two clans plugins. Problem solved.
     
    Evilyn Corporated, Feb 13, 2017
