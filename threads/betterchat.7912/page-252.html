  10. SuperCrazyBars

    Hey all,

    Trying to setup SDonate to add users to a group and give them a tag based on the package they buy. As best I can make out, I need to use something like "chat user add -username- VIP+" to add them to the VIP+ group. My issue is, how do I get the username or steamid of the person making the donation?

    Edit: Figured it out. For those having a similar issue, as an example to add the user to the group "developer" using SDonate, use the command chat user add {{VAR=STEAMID}} developer and to remove upon package expiration/revokation, use chat user remove {{VAR=STEAMID}} developer
