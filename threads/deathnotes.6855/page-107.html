    Seriously guys it's not hard to read the overview and discern the required steps... But here you go!


    Permissions:
    • deathnotes.see needed for /showdeaths and for seeing the messages

    How to grant permissions:
    Use following CONSOLE commands to grant permissions
    • grant user <player> <permission> grant permission to player
    • grant group <group> <permission> grant permission to group

    Commands:
    • /deathnotes show infos about the plugin
    • /showdeaths toggle if you see messages in chat (needs permission deathnotes.see)
    In other words

    in console (rcon or f1 in game) type

    grant group players deathnotes.see

    If your players are in a different group type that instead of "players"...

    If you have a vip group type that group in the same command in addition to the player group...

    The plugin works fine if you follow how to set it up... (Which is ONE step for each group!)
