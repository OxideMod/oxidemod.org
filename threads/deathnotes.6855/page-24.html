    @Antec your issue has been solved. Once again thank you for reporting.

    For a further explanation, plugins like Zone Manager kill players with a game built-in function which triggers the Generic death type, Death Notes did not support this type until now, and also didn't have the right checks to avoid errors. The plugin should now be flawless whenever this death type is triggered in any case.

    I also added messages for this type, but it's obvious there are no attackers, weapons used or whatever whenever a player dies of Generic, so its messages can only use the {victim} name format, and they are pretty basic, here are the default messages:

    '{victim} died.'
    '{victim} has been killed by the gods.'
