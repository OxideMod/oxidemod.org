    LaserHydra updated Death Notes with a new update entry:

    5.0.0

    Please delete your config file before updating!
    Please read the whole change log, as there are important informations included!

    Changelog:
    Code:
    * completely rewritten
* now has a lot cleaner code
* improved way of handling deaths
* improved way of handling data of players who die after being wounded
* plugin no longer writes to config every death if the data is already in the config (You should be able to edit the config file without unloading the plugin)
* due to the new way of handling, Wall deaths etc. will no longer appear
+ added Permission deathnotes.see to allow players to see death messages
If you want all of your players to be able to read them, use the console command 'grant group player deathnotes.see', 'grant group moderator deathnotes.see' and 'grant group admin deathnotes.see'
+ added command /showdeaths to toggle if you can see the death messages
