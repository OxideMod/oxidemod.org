  4. Kenjico

    Fix for all multiplay users with version 3.3.3

    1. Load up Clanforge and navigate to the Edit Profile > Configs > Death Notes and select DeathNotes.json
    2. Open up the Death Notes Overview page on here and find the # ORIGINAL CONFIGURATION FILE section.
    3. Copy and paste the entire piece of code from this section into the DeathNotes.json on Clanforge overwriting the original on there. Press Save to prevent a restart of the server.
    4. Once saved, using RCon type oxide.reload DeathNotes
    5. Death Notes works
    For some reason, deleting & reinstalling v3.3.3 still imported the old layout config, this is the issue.
    Mines been working fine since I did this.
     
    Kenjico, Sep 16, 2015
