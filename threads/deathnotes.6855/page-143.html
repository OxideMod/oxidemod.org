  12. Steenamaroo

    Camomano99 said: ↑
    Is there any way to remove the death messages for certain weapons and or weapons with certain attachments? Like have no death message for melee weapons, bows and or silenced weapons.
    Click to expand...
    All the message formats go to your config file, so you can edit them in there then reload the plugin.
    If the config is moved or deleted, the plugin will republish the original message lists.

    You can do the first thing you asked, but not the second.

    Change
    Code:
    "{attacker} killed a {victim} with a {weapon}{attachments} from {distance}m."
    to
    Code:
    "{attacker} killed a {victim} from {distance}m."
    for example.
