    What are the items in the kit? You need to keep in mind that the amount of an item you specify in a kit will not necessarily give the user one item with that amount of the item. It will still obey the stack limit of that item so if you would have something like a kit which gives 1000 torches, you need to keep in mind that torches only stack up to 15 so it will attempt to give around 68 stacks which will most likely result in a crash.

    I currently plan on adding more checks to Kits when creating them and when adding items to keep the amount of stacks within the limits of an inventory but I have some other things I need to get done first.
