  10. CaptainNebulous

    Did you restart the server or type /reload Stacksize in chat after changing the values in the config file? No changes you make will take place until you do that. This goes for any oxide mod.

    /reload plugin name (case sensitive) will work on most plugins. However, with SpawnConfig you will break the spawns if you simply use /reload so with that plugin always need server restart to get changes to take affect.
