    Nothing. It checks each item a bunch of times before it removes anything.
    For example if you have the default settings it will check once every two hours, for a total of 24 times (48 hours) Once you server has been online for 48 hours it will start cleaning objects.

    This is so it will clean things 48 hours after they are placed so it's progressively cleaning instead of removing everything in one go, to give that "decay" effect.

    One thing I would like to point out though, your "UpdateIntervalSeconds" seems to be 10 seconds?
