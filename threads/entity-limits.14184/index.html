  12. PWARS

    The entities problem its a Rust itself problem, not a hosting problem.

    Of course move to a dedicated server its exactly what you need, not a virtual server. With a dedicated server at least you make sure you can use all the resources you want. Look for a server with at least 8 or 16 RAM, the processor doesn´t need to be really powerfull.
    Then try to keep the entities count low. Install some plugins like TwigsDecay ot TwigsRemover and clean up the map everyday of raided bases.

    The gather rate doesn´t affect that much to the entities number, the worst thing its the amount of players in the server. I used to run a 60x with 200 players all the time. Then I reduced the gather to x5 and still having the same issues. Lower the player cap to 100 or 120 its what really helps.
