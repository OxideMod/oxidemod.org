    PaiN submitted a new resource:

    PaiN Prefix - Simple Prefix system.

    PaiN Prefix is a simple plugin that allows you to add a Prefix to a player's name.

    What if player has two permission (admin and vip for example):
    So the plugin check which one of the two permissions is at the top in the config file.. so if the AdminSettings are first and VipSettings second and only the AdminSettings counts.

    Default Config:
    Code:
    {
  "AdminSettings": {
    "Permission": "adminprefix",
    "Prefix": "[ADMIN]"
  },
  "Custom1Settings": {
    "Permission":...
