  2. Reynostrum

    You have to make an SQL connection beetween you server, paypal, and a mysql database.

    You need to know C#, MySQL and PHP and use the Paypal IPN (if you use Paypal) with it's external reference (to get players in-game name, or steamid64) and order number, and make a command like /vip ordernumber, and also an HTML/PHP "pasarela de pago". I don't know the translation for that, I think is payment gateway or payment form".

    The /vip ordernumber should search into a mysql database if there is an order with that ordernumber and if it is payed, then redeem it.

    Form -> IPN -> Database <- /vip ordernumber

    1. Player complete form (to get the external reference) and send him to Paypal. (search for paypal form)
    2. Player use Paypal, and if he pay send info to IPN.
    3. The IPN send a HTTP post to an url, for example: yoursite.com/ipn.php. Theipn .php file send the data of the payment to the MySQL database.
    4. Player use /vip ordernumber. And that search into the database if exists that ordernumber.
    5. If exists then redeem it and mark it as "used", for avoid use it multiple times.

    This is how I did that on my server. It's really hard to me for explain it better,because, as you can see, my english is really bad.

    You should find someone who know PHP, MySQL and C#.
