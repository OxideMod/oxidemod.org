    DropRemover - Remove selected world items from the map with commands

    This plugin allows you to remove world items from the map with no effect on server performance. You can remove all items on the map via the name of the item, via the category of the item or you can also remove every world item from the map if you wish. This is useful if you have some map spammers and you want to remove them from your map without restarting the server.

    Commands
    • drops.remove - Removes ALL world items from the map
    • drops.itemremove <item name>...
