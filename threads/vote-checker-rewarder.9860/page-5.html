    You will get the unknown command when the plugin doesn't load. That looks like you are updating the .cs file, which is the plugin itself. Re-install the default file and use the /rewardconf api <YOURAPITKEY> and /rewardconf serverid <yourserverIDnumber> to add them to the config .json file. You can get your API key and Server Number by logging into reign-of-kings.net and selecting Manage My Servers. API = 32 digit alphanumeric , Server ID = 4 digit number. Don't forget to run /rewardconf <Interval> <Period> like /rewardconf month 1 will track rewards on a monthly basis.

    The VoteChecker.json in the config directory will look like the following with the hashes filed in.

    {
    "autoGive": "true",
    "serverApi": "################################",
    "serverId": "####",
    "trackingInterval": "1",
    "trackingType": "month"
    }
