  6. Calytic Community Admin Community Mod

    It is not likely we will get a web browser or anything even resembling a web browser in Rust. The web platform is waaaay too complicated and scope creep. If you are going to ask for anything I would ask for a server-side instruction using the (already integrated) SteamAPI to open a web-page on the client. We know this works given the "Web Page" button on a server listing, we just need to be able to do that ourselves with the Rust client-server protocol.

    Personally I would like to see more advanced controls(text boxes, drop down lists, etc) in the UnityUI CUI before anyone start worrying about web integration.
