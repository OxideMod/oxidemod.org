    Advertise Me - Lets players place eye catching advertisement messages.

    AdvertiseMe Lets players use a command "/advert" to send an advert to advertise their shops, items to trade, bounties and everything their heart desires with an eye catching configurable color message.

    Permissions:
    advertiseMe.allow     - Allows the user to use the /advert command if allowAdvertToAll option is false.
    advertiseMe.admin - Allows the user to configure the settings via the console. Admin is not enabled by default.
