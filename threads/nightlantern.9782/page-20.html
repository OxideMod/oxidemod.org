  14. UFOLorD

    Bill Murray 2 said: ↑
    Does this plugin work? I'm getting mixed results from people.
    Click to expand...
    Yeah, it does work, so long as you make sure that the TunaLight: true has a comma after it! The config was borked because of a ruddy comma!

    It should look like this:

    Code:
    {
  "ConsumeFuel": true, // Require's fuel to start lights
  "LightTypes": {
    "Campfires": true,
    "CeilingLights": true,
    "Furnaces": false,
    "JackOLanterns": true,
    "Lanterns": true,
    "TunaLight": true,
  },
  "SunriseHour": 7.5, //Time lights are disabled
  "SunsetHour": 18.5 // Time lights are enabled
}
