  1. k1lly0u

    d3nnib submitted a new resource:

    NightLantern - Automatically turn ON and OFF lanterns after sunrise and sunset

    NightLantern automatically turns ON lanterns after sunset and then turns them OFF after sunrise without consuming wood.
    Sunset and sunrise hours can be set in configuration file, default values are 7 and 18:

    CONFIGURATION FILE:
    Code:
    {
  "Settings": {
    "sunriseHour": "7",
    "sunsetHour": "18"
  }
}

    CHAT COMMANDS:
    The plugin also includes chat commands to manually turno ON or OFF all lanterns:

    /lantON -> Turn...
