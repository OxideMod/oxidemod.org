  2. Wulf Community Admin

    The 1st is a plugin, the 2nd is a tool not a plugin. If you want something that checks and updates, use the tool alone or the Catalyst extension under the Extensions tab. Plugins are sandboxed, so they do not and should not have access to run programs on the server or write to files outside of what is available via the Oxide API or game itself.
