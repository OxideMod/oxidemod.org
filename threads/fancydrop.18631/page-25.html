  9. k1lly0u

    exactly, pc has its own flight controller, therefore making it unusable with standard plane behavior. the allplanes option was added per request to turn any plane into a crash plane, so it will override any standard plane behavior and run its own. if you want planes to be a different speed via options in fancydrop then disable the allplanes option. the chances of hitting a fast moving/high altitude plane is extremely small
     
    k1lly0u, Jan 29, 2017
