    set this in your BetterLoot Config "useCustomTableSupply": false to true too

    Code:
      "SupplyDrop": {
    "includeSupplyDrop": true,
    "maxItemsPerSupplyDrop": 8,
    "minItemsPerSupplyDrop": 2,
    "randomAmountSupplyDrop": true,
    "useCustomTableSupply": true
  }
