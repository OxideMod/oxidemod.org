    Zanderwar submitted a new resource:

    StatLogger - A plugin designed to send statistics in JSON format to defined endpoints

    The goal of this plugin is to simply send information about specific events that take place on your server, to an endpoint on your web server for the sole purpose of recording statistics and monitoring player activity

    Whilst it is currently in it's infancy, what you can create from it will definitely set your server out from the rest.

    In short, it will dispatch JSON encoded information to the baseUrl that you define within StatLogger.cs

    bREeeEeEeeEeeEZaaaHhH

    It doesn't really need a config file as everything will be pushed to an external URL (you have to edit the .cs file). As an example:
    Code:
    $json = file_get_contents("php://input");
$data = json_decode($json, true);
$SteamID = $data['SteamID'];
$PlayerName = $data['PlayerName'];
$Address = $data['Address'];
    With this little PHP code you can do "something" with the people that join. Save their ID, playername and IP address for later use (stats, join counts, unique players, etc)

    Basically if you don't know how this works, you wouldn't (or shouldn't) use it.
