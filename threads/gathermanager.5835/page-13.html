    Quarry resources are incredibly low so the but can be brought more on par with normal gathering by using a bigger modifier. And the dispenser.scale changes the default amount of resources that can be gathered from a dispenser, ex. disp scale x1, wood gather x1000. When hitting a tree you'll get something between 700 and 1000 resources and the tree is destroyed. If the dispenser.scale for tree would be 2 you'd get around 2000 wood for that same swing.
