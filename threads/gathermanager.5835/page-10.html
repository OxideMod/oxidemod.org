    Thanks for the reply!

    I am not a programmer, but I also am not dumb hehe, and because I really want the Happy Gathering Hour event, I did these things:

    - Created a new folder in the FTP of the server, with 2 sub folders: Normal and HappyHour.
    - Made a timed command using my FTP software to copy the file from HappyHour to oxide/config, thus replacing the current file for m-GatherRate.
    - Used TimedCommand to issua an oxide.reload m-GatherRate command from the RCON, this activated the newly copied config file.
    - Then, when the happy hour is off, the FTP software copies the config from the Normal folder, into the oxide/config folder.
    - TimedCommand issues another oxide.reload command to activate the config.

    And voila! It works
