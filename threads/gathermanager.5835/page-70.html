    @Mintyy, I have had a similar issue before as well.. Firstly, make sure you dont type gather.rate ALONE, type in for example, /gather.rate pickup Mushroom 5 See if that works, of course you can change it once it does, but for the most part, try it. Secondly, if you already did this step correctly, try using it in the server console rather then F1, I dont know if it will make a difference but I know the server console method worked for me so, try it out as well. (That may have sounded stupid if they are the exact same but I have only been doing rust for 2 weeks lol). Now I did have an issue the first time where my plugin didnt exist but it was in the folder? Try removing it completely, restart the server, then download a completely new copy and use that instead, << Reason you should do it in a step sequence like this is so no matter what, you know its not the plugin file(s). Restarting it refreshes every single plugin while reloading just refreshes that single plugin, depending on the other plugins, and a new copy means latest version, and the chance you downloaded a corrupt version of code? Not sure if that second one is possible but who knows, I could see it happening.
