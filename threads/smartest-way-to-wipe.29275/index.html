  5. Manvaril

    @Pascal 6 I'm currently hosted through Fragnet, This is my routine.

    Normal Wipes
    1. Shutdown Server
    2. Wipe World
    3. Steam Update
    4. Update Oxide Mod
      - If Rust.IO needs to be updated --> Mod Manager -> Uninstall -> Install
    BP Wipes
    1. Shutdown Server
    2. FTP into server or use the File Manager
    3. Navigate to /server/fragnet/
    4. Delete "player.blueprints.1.db"

    Map Seed Changes
    1. Command Line Manager -> Custom Commandlines -> Edit
      Under +server.seed put in a random number or find a map you like on Map Gallery - Rust:IO
    2. Save
    Changing the seed number wipes the map and generates a new map
    DO NOT MESS WITH THE COMMAND LINE MANAGER WHILE THE SERVER IS RUNNING YOU WILL CORRUPT YOUR SAVE FILE!
