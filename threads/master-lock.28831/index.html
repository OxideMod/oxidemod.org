    S0N_0F_BISCUIT submitted a new resource:

    Master Lock - Gives players the ability to control all code locks in their base from the tool cupboard.

    Master Lock gives players the ability to have all the code locks in their building sync with the code lock on the tool cupboard, a.k.a. the master lock.

    All linked code locks will share the same code as the master lock, as well as share the same authorized player list. Upon enabling master lock, all currently unlocked code locks will link with the master lock.
