    I usually just delete the save folder in my server directory

  7. [X] Meep

    Be sure that when you change your map and seed you delete teleportation data file. Else players could teleport into the ocean when they want to go home.

    Changing the map and seed however do not delete blueprint data.

    One of the other common things is clearing the storage.db file. Helps a bit on performance after changing the map.
