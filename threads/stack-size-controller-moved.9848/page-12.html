  3. AllHokage

    Hello,

    This is a fix for anyone using FTP who can't add stacks or see permissions folder!

    So Basically I rent a server and have to use FireFTP (Mozilla Firefox addon) to add plugins and so on. So my problem was that I couldn't see the permissions folder to add stacks in the game or in the permissions folder. So to get the permissions folder working, all you have to do it make a fake json folder (a.json) and leave it blank and place it in your server config file and poof, all your permissions folders from plugins will appear! This is due to there being bugs in FTP clients and simply uploading a file will update it!

    P.S (Make sure to deleted that file you created)

    Hope this helped and people won't spend hours like I did
