  13. PaCe

    Do you want to set everything on a specific stacksize? Then it will be easy. Give yourself as admin all permissions with type following in the CONSOLE System:
    oxide.grant user YourName *


    Then use the command
    /stackall 10000
    INGAME, you can change the number to another one.
