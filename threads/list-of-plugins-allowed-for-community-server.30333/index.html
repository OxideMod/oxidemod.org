  2. Wulf Community Admin

    Facepunch has stated that anything other than "admin tools" is not allowed, so anything that visually alters the game for players or changes game mechanics is not allowed, everything else would be.

    Examples of NOT ALLOWED:
    • Any in-game or external maps that show positions of things (ex. Rust:IO, LustyMap)
    • Any GUI plugins that players can access or use
    Examples of ALLOWED:
    • Ban/kick plugins such EnhancedBanSystem, SecureAdmin, etc.
     
    Wulf, Apr 20, 2018
