    I've seen this issue occur with ZoneManager's "NoCollect" flag. If you have any zones with that flag, I'd recommend checking to see if there's any animals there trying to eat. If so, the only solution is not using the NoCollect flag or the flag (I think there's one) that kills NPCs that enter the zone.
