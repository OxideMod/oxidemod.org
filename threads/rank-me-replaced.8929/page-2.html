    Rank-ME | Top List fixes, and much, much more!

    [ CHANGELOG ]
    • Re-worked Top List due to the amount of issues it had previously.
    • Rank Command will now check whether the player has any kills rather than the ranking itself.
    • Commands description can now be seen with /rankme help, instead on SendHelpText hook.

    • Added Longest Shot, Sleepers and Animals keys.

    • Fixed wrong configuration version.
    • Fixed not checking Announce Database Wipe.
    • Fixed Show Top In Chat and Show Top In Console settings not working.
    • Minor tweaks and fixes.
