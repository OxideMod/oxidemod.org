  12. NobodyFTW

    Hi everyone,

    I have finished a new version for Rank-ME, this version is a complete re-write to improve the old version in overall, also I have added a bunch of new stats and a new dynamic system for top lists.
    Since I do not own any server at the moment I am not able to test this in full to make sure any major and perhaps minor bugs are fixed before the release. I was hoping any of you could help me and test the version out for me.

    [ REQUEST ]
    The task is simple, to test the plugin, use it at your own will, mess with it, abuse of it any way you can to find any leaks, if any. So far I've tested everything on my own, which looks fine at first, but you never know.
    If any errors come up, please post them in this discussion.

    * Note: If you're using any older version of Rank-ME, then you must delete any older files.

    For more details, here is the current change log of this version, though, it is not final until the its release:
    [ CHANGELOG ]
    • Complete re-write of the plugin
    • Improved calling for top lists: /top [list name]
    • Replaced Admin commands with:
      • /rankme <save>
      • /rankme <resetdb>
      • /rankme <del> <player name>
    • Added new statistics:
      • Suicides / Death ration
      • Resources Gathered (In general)
      • Times Wounded
      • Rockets Fired
      • Bullets Fired
      • Arrows Fired
      • Explosives Used
      • Human NPC Kills
      • Barrels Destroyed
    • Renamed Range List to Distance List
    • Top Distance will now be checked only on PVP
    • Fixed Auto-Save timer not working properly
    - This version is a complete re-write of the plugin, and requires you to remove any older files.
    Click to expand...
    For more info about Rank-ME: Rank-ME for Rust Experimental | Oxide

    Thanks in advance.
