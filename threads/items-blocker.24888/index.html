    Vlad-00003 submitted a new resource:

    Items Blocker - Prevents some items from being used for a limited period of time. With UI.

    This is a very simple plugin, that prevents players from putting items onto thier belt for a limitied period of time.

    Basicly it was made to block usage of weapons for some time after wipe.
    If player tries to put something from the list to their belt - GUI with timeleft shows up.
    You can change it's text in the lang file under
    Code:
    lang/en
lang/ru
    By default it supports englist and russian trranslation.
    Default config file:
    Code:
    {
  "Block end time": "10.04.2017 03:38:42", <--...
