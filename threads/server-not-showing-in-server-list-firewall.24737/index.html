  11. logikgear

    to edit the file manual you will need to install a text editor like notepad++.

    Stop your server. You are going to want to go to the rust program folder and edit the file "rustserver.*" I put a * because I run a Linux server and am not sure what the file extension is. Once you open that in notepad++ you will scroll down till you fine the ###Server Settings### section. In that secion there is a line ip="0.0.0.0". Change it to your servers local IP. Restart the server and give it about 30 mins before you start looking for it to show up.

    If you want to edit your Rustide Config instead you will want to add a line before the port number it should look something like "Set _ip="192.168.0.xxx"

    The problem I was having is Steam doesn't like you using "localhost or 0.0.0.0" when behind some routers. Not sure why but I fought it for about an hour the first time before changing it and it working.

    Also do you have those inbound rules mirrored in outbound?
