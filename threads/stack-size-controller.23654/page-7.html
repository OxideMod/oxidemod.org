  3. Canopy Sheep

    Canopy Sheep updated Stack Size Controller with a new update entry:

    1.9.4

    - Stackall commands no longer change stacksize of items with health
    - Changed example item from "ammo_hv_rocket" to "ammo.hv.rocket"
    - Added error message when trying to change stack size of items with health
    - Added config check for items with health, will change stacksize to 1 if value is added to the config manually

    If you wish to disable the config check for whatever reason, comment out lines 38-43 in the plugin itself. Regardless, items with health will not stack in game.
