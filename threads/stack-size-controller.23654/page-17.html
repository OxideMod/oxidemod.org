  4. jackherroor

    im not sure i have undertsand what u mean but what i think is you cant change the stacksize?
    for this go to oxide\config\StackSizeController.json ..open it..than you see this:
    {
    " Cooked Bear Meat": 2000,
    "12 Gauge Buckshot": 500,
    "12 Gauge Slug": 500,
    .......
    in this you can change the stack at every item you want.at this example is bear meat stackable to 2000..you can change this how ever you want make 100 than stack to 100.
    after this save this. and reload plugin whit this command in console : oxide.relaod StackSizeController
    after it must work....
    or do the shortway:
    in chat : /stackall 50000 or in console : stackall 50000
    than all items stackable to 50000

    but you can also change all stacks from singel items per command
    example for chat: /stack wood 100000
    /stack stones 30000
    example console: stack wood 100000
    and so on.
    hope this is what you need.or is the plugin not load on your server?
