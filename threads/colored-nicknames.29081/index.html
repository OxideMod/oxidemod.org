    ColoredNicknames - Simple plugin to make nicknames coloured.

    Makes nickname coloured using ROK registry system.

    Command:
    /color     "name of color" or "hexcode" or "reset"

    Examples:
    /color     blue
    /color ff0000
    /color reset

    Config:
    Code:
    {
  "AdminOnly": false
}
