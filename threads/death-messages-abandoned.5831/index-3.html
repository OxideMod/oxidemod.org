  1. Reneb

    Reneb submitted a new resource:

    Death Messages - simple plugin that will handle players deaths messages and animals.

    Death messages that will show PvP kills and PvE kills.
    Doesn't support suicides / bleeding / radiation / etc yet

    Configs:
    Code:
    {
  "ChatName": "Death",
  "Settings": {
    "animalkills": true,
    "players": true,
    "animaldeaths": true
  },
  "deathByEntityMessage": "@{killed} has died by a @{killer}",
  "wildlifeDeathMessage": "@{killer} killed a @{killed} (@{weapon}) at @{distance}m",
  "playerDeathMessage": "@{killer} killed @{killed} (@{weapon}) at @{distance}m"
}
    ...
