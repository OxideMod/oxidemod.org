    You need to modify the value thats being replaced into the message in the loop, otherwise it won't help. Basically you need to check the replacement value for % before its's replaced.
    Code:
       for k, v in pairs(tags) do
     v:gsub("%%", "%%%%")
     customMessage = string.gsub(customMessage, "@{".. k .. "-}", v )
   end
