  14. XsV=Tool

    I had an issue with this script working on SmartFTP. Although I was able to fix it once I determined the source, it was an overall easy fix. I hope it helps anyone having trouble using this.
    So what I noticed was this plugin was in the wrong script format for my FTP to transfer. So after a little research I learned about line conversions done with a program called Dos2Unix. It has a program within its folder called Unix2Dos, all I did was simply drag and drop the plugin on top of the executable and then click run when the startup window launched. Then it automatically converted all the lines from Unix to Dos in the correct format for SmartFTP to transfer correctly. Now all the lines are in order and it doesn't look like a bunch of jibberish.
