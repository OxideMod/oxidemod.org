    Smoosher submitted a new resource:

    Decay Everything - Disables standard decay and creates a Custom Decay with custom rates and timers

    There are a few plugins that decay twig/walls etc, there are a few plugins that remove entities not in cupboard range, this plugin combines the 2 into one large decay plugin that lets you configure what you want to decay, how often and by how much.

    Config
    The config is Meaty, so feel free to ask questions if your not sure
    Code:
    {
  "DecayAllEntities": false, -- Decays ALL entities in the list
  "DecayBuildings": true, -- Decays all building foundations that are marked as true...
