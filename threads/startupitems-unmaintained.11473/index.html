  1. RedNinja1337

    RedNinja1337 submitted a new resource:

    StartupItems - Let you select the startup items when a player respawn from dead.

    When a player respawn from dead it starts with only a Rock and a Torch. This plug-in use a configuration file where you can set the items you want the players to have after they respawn. The configuration file by default is set for players to start with a Rock, Hunting Bow, 25 HV Arrows, Hide Pant, and Hide Boot as an example but you can remove this on the config file.

    Example of adding an item on config file:
    {
    "Amount": 1,
    "Container": "Wear",
    "itemID": -148163128,...
