    PaiN submitted a new resource:

    Warp System - warp system, warp, teleport

    Warp System is a plugin that allow any admin(2) or moderator (1) to create, delete and list Warp Points.

    Examples:
    /warp add WarpExample - This create a warp point with the name WarpExample on the admin current location.

    /warp del WarpExample - This delete a warp.

    /warp go WarpExample - Use the warp and teleport the player to the Warp location.

    /warp list - Lists all available warps

    /warp back - Teleport the player back before he got warped to his previous location.
