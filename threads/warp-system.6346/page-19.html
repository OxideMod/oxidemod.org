  4. PaiN

    PaiN updated Warp System with a new update entry:

    1.6.0

    [Changed]

    Commands:
    The warp add command looks like: "/warp <add> <WarpName> <WarpTimer> <WarpRange> <WarpPermissionGroup>" now.
    I just added the <WarpRange> argument because its useful for arenas, death matches and other events.

    If you want the player to teleport on the position that you created the warp just simply set it to "0"
