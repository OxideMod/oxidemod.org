    oxide.grant command ? Where did you see that.
    type
    a) oxide.grant user <PlayerName> <permission>(In this case permission should be warpsystem.admin)
    OR
    b) oxide.grant group admin <permission>(Permission warpsystem.admin again)

    You should also know that if you are an owner on your server Oxide automatically assigns you to the PermissionGroup named "admin"(Option B)
