    I'd recommend Streamline as far as shared hosting goes, however there have been some CPU issues mainly. The main reason you're experiencing more lag than usual is because most server hosters overcrowd their boxes, before that wasn't an issue as they used very little CPU. Now each server is using 50%+ on startup due to navmesh changes. So when a hoster has say 20 servers on one machine it doesn't end well. The only way to escape that is dedicated, I'd take a look at the OVH game line or NFO if you can afford it. (NFO shared is experiencing the same issues, you'd need a NFO dedicated server.)
