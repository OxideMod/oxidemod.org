  14. Skye12977

    Well, I found a way that fixed it for me. No modifying of the script needed.

    1. Connect to your server
    2. revoke admin group rights for admintoggle.use and admintoggle.* permissions
    3. server.writecfg
    4. Delete admin toggle plugin and other folders
    5. remove yourself from the admin group
    6. server.writecfg
    7. Disconnect from server
    8. Reconnect to server
    9. server.writecfg
    10. add yourself to the admin group
    11. server.writecfg
    12. Disconnect from server
    13. Connect to server
    14. Verify status
