    Hardwithporc, if its working, its going to stop printing around that point. Any output from the rust server would only be shown in the GUI, and xvfb-run basically blocks the GUI from actually showing (because you don't have one). The GUI isn't really necessary, but you're going to have to try an rcon client or try telnetting to the port after its been running a minute to see if its accepting connections. Basically, if you look at top and look at the CPU usage of the process, its going to be about 100% while its booting up, and about 50% if its running correctly after boot up. If you see it lower to almost nothing, it might indicate you have a problem.

I was still having problems so I installed wine-gecko2.40 and it finally started working properly. Tested with vanilla and oxide.


    -------------------------------------------------------------
    As a side note the basic steps to take on a new stock ubuntu system:


    #set up wine 1.7
    sudo add-apt-repository -y ppa:ubuntu-wine/ppa
    sudo apt-get update
    sudo apt-get -y install wine1.7 winetricks
    sudo apt-get -y install wine-gecko2.40
    sudo apt-get -y install xvfb


    #set up steam
    sudo apt-get -y install libc6-amd64 ;#steam requires this
    sudo useradd -m steam
    sudo usermod -aG sudo steam ;#Add steam to user group sudo
    sudo passwd steam ;#set a password so you can sudo later
    sudo su - steam
    mkdir ~/steamcmd
    cd ~/steamcmd
    wget https://steamcdn-a.akamaihd.net/client/installer/steamcmd_linux.tar.gz
    tar -xvzf steamcmd_linux.tar.gz
    ./steamcmd.sh +quit


    #downloading rust server is something like
    /home/steam/steamcmd/steamcmd.sh +\@sSteamCmdForcePlatformType windows +login anonymous +force_install_dir /home/steam/s/rust +app_update 258550 validate -beta experimental +quit

    #getting and installing oxide is something like
    cd /home/steam/s/rust
    wget -O Oxide-Rust.zip https://github.com/OxideMod/Snapshots/blob/master/Oxide-Rust.zip?raw=true
    unzip     -o Oxide-Rust.zip


    #starting a server is something like (with correct IP)
    cd /home/steam/s/rust
    wineboot --update
    WINEARCH=win64 WINEPREFIX=~/.wine64 /usr/bin/xvfb-run --auto-servernum --server-args='-screen 0 640x480x24:32' /usr/bin/wine RustDedicated -batchmode -load +rcon.ip 1.2.3.4 +rcon.port 5558 +rcon.password mypassword +server.hostname 'Test Server' +server.port 5557 +server.identity myserver +server.maxplayers 100 +server.seed 5551212 +server.globalchat True +server.worldsize 3000 -autoupdate
