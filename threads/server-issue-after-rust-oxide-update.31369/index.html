    It could be plugin related.
    1. Starting the server without any plugins enabled. (Add .disabled to the end of all the file names) Wait to see if the server freezes after 10 minutes. If it does the issue is Oxide or any other extention related. Try disabling them and test them. Also update oxide if you havnt already.
    2. 1 by 1 enable a plugin (remove .disabled from file name) and wait for it to full load and test they are functionally working, since your server freezes after 10 so minutes, this is the long part due to how many plugins you have. Make sure to load depencies first, Rust:IO, Image library etc etc.
    3. Once the server crashes, Disable that one plugin and restart the server again. Wait to see if it crashes without it loaded. If it does, go back to step 1 disable them all again and repeat. If it doesnt crash again. Skip the one that caused the crash and carry on enabling plugins 1 by 1
    4. Once your confident the server is running and you figured out what plugin it is see if there is an update for it. If not, contact the developer of the plugin with your issue.
    1 thing that pops out, is you say the server crashes after 10 minutes, and you advert plugin is set to run every 10 minutes. So try that first.
