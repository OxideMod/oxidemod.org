    Compiler disconnected generally means that either the CSharpCompiler/CSharpCompiler.x86 is outdated (check that you have permission to update it when uploading it manually) or you're missing a dependency such as libgc6.

    The autosave is not something Oxide controls, but it should happen automatically if your save is not corrupt.
