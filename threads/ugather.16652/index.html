    DylanSMR submitted a new resource:

    IGather - Set a gather rate for each player

    This plugin adds the ability to your server to add a gather rate to each player. For example, if "tommy" is a VIP or something and gets 5+ more gather, then you can add 5 gather to his current gather rate.

    Configuration Settings:
    DefaultQuarryRate | The amount a quarry produces multiplied by the normal amount
    DefaultResourceRate | The amount you earn from a node, tree, etc when you harvest it multiplied by the normal amount.
