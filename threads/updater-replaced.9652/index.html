    Updater - Automatic update checking and notifications for plugins

    Updater automatically checks all supported Oxide 2 plugins for updates on server start or by chat command. This is useful to see if you are using an outdated, older versions of plugins. It will also automatically check for updates every hour, or at the time configured. Updater also supports the Email API and Push API plugins for instant notifications!...
