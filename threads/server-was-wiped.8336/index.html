    As someone much more experienced in the topic said, plugins are go-to excuses when it comes to customer service of game servers providers

    AFAIK Oxide is NOT touching your .sav files, where your map and everything on it is saved. In the rare events where the save file gets corrupted, there are few older versions of it saved in the same folder.
