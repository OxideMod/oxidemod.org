    Rust:IO Clans - Allows your players to form and manage clans.

    This is an addon for Rust:IO, so in order to use all features of this plugin, you need to install Rust:IO first!

    Rust:IO Clans provides your players with an extensive clan system. It offers:

    Code:
    Available commands:
  /clan - Displays relevant information about your current clan
  /c Message... - Sends a message to all online clan members
  /clan create "TAG" "Description" - Creates a new clan you own
  /clan join "TAG" - Joins a...
