  10. dcode

    The plugin is adding a non-breaking-space character between the clan tag and the player's original name to have a reliable way to determine what is an actual tag from the plugin and what is not. In an UTF8 capable environment like the game, this just shows as a space. Maybe set your console to UTF8.

    If you are using plugins that do not look up player names by partial inputs, these will probably need to be improved in that regard.
     
    dcode, Mar 31, 2015
