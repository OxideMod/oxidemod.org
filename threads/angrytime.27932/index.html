  1. Tori1157

    Tori1157 submitted a new resource:

    AngryTime - Check, add & set time via commands

    View attachment 41815

    This plugin is made for people who want other staff members that do not have access to console, to be able to control the time via commands. There are 2 ways to change time, one is setting time and the other is adding time.

    Currently there is only one permission which is angrytime.admin

    Available commands:
    • /time set 10 (This will set the time to a whole number (10:00))
    • /time add 1 (This will add one hour to the current time...
