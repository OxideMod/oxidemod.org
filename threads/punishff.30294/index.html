  1. Vliek [NL]

    Vliek [NL] submitted a new resource:

    PunishFF - Punish player for attacking his friend by giving the attacker the damage he has done to his friend..

    @alkaris his plugin request

    With this plugin players who attack a friend will be punished by giving damage to the attacker instead of the victim (If used with FriendlyFire)

    If it's not used with FriendlyFire both the attacker and victim get damage.

    Default config:
    Code:
    {
  "Settings": {
    "% of punish damage given (Default 50%)": 0.5
  }
}
