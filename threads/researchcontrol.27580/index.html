    Vlad-00003 submitted a new resource:

    ResearchControl - This plugin allows you to manage all what can be done with the research table. Speed and cost.

    So, this plugin is basicly made to allow you to adjust the price for research for any item. Also it allows you to customize the speed for every player in game!

    You can create as much custom permission as you want.
    basic examples are included.
    So, how does the custom permission looks like?

    Code:
        "researchcontrol.vip": { <-- The permision itself
      "Reserach cost modifier": 70,
      "Reserach speed modifier": 70.0,
      "The speed is absolute value, not modifier": false
    }...
