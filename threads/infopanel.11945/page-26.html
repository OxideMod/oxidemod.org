  4. Pix3l

    MR.STEAL said: ↑
    Yeeepp
    Click to expand...
    Open the InfoPanel config file and search for "BottomDock" after it says "Panels".
    Change BottomDock to "TopDock".

    Eg.
    Code:
      "Panels": {
  "AirdropEvent": {
  "AnchorX": "Left",
  "AnchorY": "Bottom",
  "Autoload": true,
  "Available": true,
  "BackgroundColor": "0 0 0 0.4",
  "Dock": "BottomDock",
  "Height": 0.95,
  "Image": {
    to
    Code:
      "Panels": {
    "AirdropEvent": {
      "AnchorX": "Left",
      "AnchorY": "Bottom",
      "Autoload": true,
      "Available": true,
      "BackgroundColor": "0 0 0 0.4",
      "Dock": "TopDock",
      "Height": 0.95,
      "Image": {
