  8. DroplulGaming

    Pouncer012 said: ↑
    How do I even get this up and running. It's already installed to the server, but not sure how to get it started. Help please?

    Looks like we're all having the same issues.
    Click to expand...
    Just Remove the old files from plugin, config and data folder, and download th folowing 2 posted files from "RAK":
    InfoPanel | Page 66 | Oxide

    Iff your server is already running, first put the config in the configs, and then the plugin under plugins, so it will keep the settings in the post, and to be sure the plugin wont generates his own settings.

    It will be automaticly shown on the bottom of the screen.
    To get it on top, you have to search for the following 2 things:


    "Docks": {
    "BottomDock": {
    "AnchorX": "Left",
    "AnchorY": "Bottom",
    "Available": true,
    "BackgroundColor": "0 0 0 0.4",
    "Height": 0.02,
    "Margin": "0.005 0.005 0.001 0.005",
    "Width": 0.99
    },
    "TopDock": {
    "AnchorX": "Left",
    "AnchorY": "Top",
    "Available": true,
    "BackgroundColor": "0 0 0 0.4",
    "Height": 0.03,
    "Margin": "0.005 0.005 0.005 0.005",
    "Width": 0.99


    And change it to:

    "Docks": {
    "BottomDock": {
    "AnchorX": "Left",
    "AnchorY": "Top",
    "Available": true,
    "BackgroundColor": "0 0 0 0.3",
    "Height": 0.03,
    "Margin": "0.005 0.005 0.001 0.005",
    "Width": 0.99
    },
    "TopDock": {
    "AnchorX": "Left",
    "AnchorY": "Top",
    "Available": true,
    "BackgroundColor": "0 0 0 0.4",
    "Height": 0.03,
    "Margin": "0.005 0.005 0.005 0.005",
    "Width": 0.99


    This works on my server, also took me few days of adjusting and putting back the origional config because I crewed it again
     
    DroplulGaming, Jul 19, 2016
