  5. Ghosst

    [-Pix-] Slodrein said: ↑
    how to move my picture : left, right, up etc
    Click to expand...
    You need to make a custom panel.

    For example:
    "MyLogo": {
    "AnchorX": "Left", <- Left or right side
    "Autoload": true,
    "Available": true,
    "BackgroundColor": "0 0 0 0.4",
    "Dock": "BottomPanel", <- BottomPanel or TopPanel
    "Height": 0.95,
    "Image": {
    "Height": 0.8,
    "Margin": "0 0. 0 0",
    "Order": 1,
    "Url": "Your Image url",
    "Width": 1
    },
    "Margin": "0 0 0 0.0005",
    "Order": 6,
    "Width": 0.02 <- Width
    },
