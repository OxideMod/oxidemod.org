  7. FireStorm78

    Memesiss,

    It really helps to understand the box model in CSS web design. Everything is a container and the width is a percentage. The way it is implemented in this plugin is a bit wonky but it will eventually make sense. Pay extra attention to the image at the bottom of the overview that shows how margins and widths add up to 100% (1). The unfortunate thing is if the width of the parent container changes it changes the children as well. So if you get it just right then expand the dock width then it all stretches. I've spent hours tweaking settings and you will too. To stack items vertically in the same dock I THINK you'll want to pay attention to AnchorY and Align. I haven't tried it yet. Good luck.
