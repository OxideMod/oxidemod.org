  17. MongerVog

    RibsGrowBack said: ↑
    what did you do to make it work please?
    Click to expand...
    I created a zone, then edited it using zonemanager. first thig I did after creating it was give it a name.
    Then I logged into ftp and went into my oxide folder and into data folder and viewed the ZonesManager.json file
    I had something like this
    Code:
    {
  "ZoneDefinitions": [
    {
      "Name": "Town",
      "Radius": 250.0,
      "Radiation": 0.0,
      "Location": "-2334.557 0.3051544 908.9743",
      "Size": "0 0 0",
      "Rotation": "0 0 0",
      "Id": "87446157",
      "EnterMessage": "Welcome to Town, This is a NoKill zone! Visit our website for more info.",
      "LeaveMessage": "Come back any time by typing '/Town' in chat. (limted 5 uses a day)",
      "Enabled": true,
      "Flags": "AutoLights, PvpGod, PveGod, SleepGod, UnDestr, NoBoxLoot, NoPlayerLoot, NoBleed, NoDrown, NoPickup, NoDrop"
    }
  ]
}
    Your going to want the location, radius and zone id.
    Now go into your zonedomes.json file in same folder and do add it in.
    Example:
    Code:
    {
  "Zones": {
    "87446157": {
      "Position": "-2334.557 0.3051544 908.9743",
      "Radius": 250.0
    }
  }
}
    Of you could do it the easy way and grab that same info from sonesmanager and insert all the info using your ownerid access on your server and using the chat commands from here for ZD.
