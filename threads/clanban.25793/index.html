    ClanBan - Ban an entire clan with one command!

    ClanBan is a plugin that helps you ban an entire clan, this works with both RustIO Clans and ClansReborn.

    • Works with both version of clans.
    • Choice of 3 ban systems! (Vanilla, EnhancedBanSystem, DiscordMessages)
    • Customization of messages
    Commands
    () Required <> Optional
    • /clanban /cb (clan) <reason>
