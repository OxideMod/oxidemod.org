    I was looking for more people to test my new way of giving players the ability to Noclip without giving full admin rights.

    Note: By default, it does disable flyhack and noclip antihack protection. (working on a workaround)

    I have this plugin, which gives players a flag of IsAdmin on wake up. From my testing, it works well and allows non admins to use noclip just as a admin would. I have also added my Fauxclip command /noclip as well.
    As this Admin flag is not set BEFORE players log in, its a temporary admin that only seems to allow client side admin checks to work. (noclip and Shift-N camera)

    Permission : fauxclip2.allowed

    What the plugin does :
    1 - when players wake up (front intitial connection or respawn) they get flagged as a admin with a IsAdmin player flag.
    2 - players only have access to use noclip and shift - n (dev camera) while under the temp admin just as normal admins would. They can Bind a key for it as well.
    3 - currently flying into a building blocked will turn off noclip for players. For testing, will add a bypass permission.
    4 - players will have a Green admin color in chat, but no other powers. (used better chat and they are seen as regular players by default.)
    5 - the temp admin flag is automatically removed when players disconnect or log off. The flag does not stay. There was no need to manually remove flag with plugin.

    I have tested it with my players, just wanted other input to see if this would be a viable way allow noclip flying without giving players full admin/mod rights. Plus I want to make sure there are no secret commands that I missed in testing.

    Note: Any plugin that is loaded that does check for a player flag of IsAdmin, will allow fauxclip admins access. FYI when testing

    And any suggestions will be greatly appreciated !!
