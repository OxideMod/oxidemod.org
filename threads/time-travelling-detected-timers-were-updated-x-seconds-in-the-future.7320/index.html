  9. bawNg

    That is a warning, not an error. The latest build should not spam console but it will warn you if the clock suddenly jumps. A warning about a late clock could simply mean the server froze for over a minute. The timer system will attempt to automatically correct itself so that timers continue to function normally instead of either firing immediately or never firing again as would happen in the past.

    The clock in question is the internal performance counter provided by the windows kernel. Although it is meant to always be accurate, faulty hardware or a bugged BIOS can cause the counter to change depending on CPU.

    If you are experiencing clock jumps, please try limit your servers affinity to a single core and let us know if that helps.
     
    bawNg, Mar 19, 2015
