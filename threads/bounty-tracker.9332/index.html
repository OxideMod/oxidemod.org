    Bounty Tracker
    This plugin does exactly what it says on the tin! You can now set bounties on other players (in a stack of harvestable resources up to 1k)!

    Got someone who has raided your base? Set a bounty on his head!
    Don't like the current king? Set a bounty on his head!
    Want to see if anyone has the guts to take you on? Set a bounty on your own head!!

    Commands
    /setbountyname <player_name> - sets the name of the bounty target
    /setbountyresource...
