    He's trying to run a Rust Server from a linux machine. Linux uses IP tables as a firewall. Your posts should also be forwareded if you're behind a router. Allowing them via IP Tables is not enough. However if you're on a dedi you can get away with just iptables. You can also use this tool: AdriaanBoshoff/GetServerAtAddress to actually check if your server gets listed with steam. If it doesn't then your ports are closed or something is blocking it.
