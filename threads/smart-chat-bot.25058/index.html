  1. i_love_code

    i_love_code submitted a new resource:

    Smart Chat Bot - I send chat messages based on some triggers or time

    Allows you to easily create recurring timed messages and automatically have a bot respond to in-game chat messages from players. You can configure this bot to trigger when a player says any sets of words.

    Installation:
    Your bot can just be dropped in and loaded, which will generate configuration files in the data folder for you.

    The default configuration is setup as follows:

    Code:
    {
  "ChatDisplayName": "<color=#00FF00>[Chatbot]</color>",
  "BotCooldownInSeconds": 15.0,...
