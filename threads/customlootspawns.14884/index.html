    CustomLootSpawns - Create additional custom spawn points for loot boxes of your choosing

    Create custom loot spawn points, using boxes of your choosing. Boxes are on a refresh timer, and will respawn in the same place when you restart the server. Authlevel access only

    Chat Commands
    /loot - Displays commands
    /loot boxes - Puts available loot box types and ID's in your console
    /loot add ## - Create a new spawn location for loot box ID ##
    /loot remove - Removes the box you are looking at
    /loot near ## - Shows all custom loot boxes in a...
