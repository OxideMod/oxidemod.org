    The log you originally posted was just compiler warnings, not errors, which can be ignored. Your main log file for Oxide is the oxide/logs/oxide.*.txt files with dates, not the compiler one. Your server "crashing" or being outdated wouldn't be in any of the Oxide logs though, as Rust generates crash logs under your server's main directory and its own log files.

    If your server is giving you that protocol message, then your server isn't fully updated. It may be that your host is providing a link to an old Oxide build for installing instead of the latest, which would downgrade your server as Oxide patches Rust DLLs (mods them).

    on my Source servers I use the following commands:

    ::=======================::
    :: End of variables ::
    ::=======================::

    :: This will keep the window clean and easy to read
    @echo off

    :: Sets the title of the window
    title Anti Crash %servername%

    :: Clears the window incase there is anything there
    cls

    :: Prints to the window what we are doing
    color 02
    echo Servidor iniciado No More Room In Hell !
    echo.
    echo *************************************************************************
    echo Para fechar o Servidor, feche esta janela!
    echo *************************************************************************
    echo.
    echo.
    echo %servername% Server Iniciado!
    >> "%servername%.log" echo.
    >> "%servername%.log" echo.
    >> "%servername%.log" echo (%date%)(%time%) SRCDS Guardian 3.0 has been started!
    >> "%servername%.log" echo (%date%)(%time%) %servername% is now starting...

    :: This is a return point in case the server crashes or is closed
    :restart
    color 03
    echo.
    echo (%date%)(%time%) %servername% is now ONLINE
    echo Watching %servername% for crashes...
    >> "%servername%.log" echo.
    >> "%servername%.log" echo (%date%)(%time%) %servername% is now ONLINE
    >> "%servername%.log" echo (%date%)(%time%) Watching %servername% for crashes...

    ::Start the actual server
    start /wait %runcmd% | echo N
    color 02
    echo.
    echo.
    echo.
    echo.
    echo =============================================================================
    echo (%date%)(%time%) Crash or Close detected!
    echo %servername% is now restarting...
    >> "%servername%.log" echo.
    >> "%servername%.log" echo (%date%)(%time%) Crash or Close detected!
    >> "%servername%.log" echo (%date%)(%time%) %servername% is now restarting...

    ::Server crashed or closed, so we point it to the return point to start the server again
    goto restart
