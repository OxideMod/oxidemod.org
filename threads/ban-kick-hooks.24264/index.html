    Kinesis said: ↑

    Click to expand...
    You could use the OnServerCommand hook for both of those. I used them to detect when a player gets kicked for a plugin I made.

    Code:
    void OnServerCommand(ConsoleSystem.Arg arg)
        {
            if (arg.cmd.FullName == "global.kick")
            {
                arg.GetString(0); // Name/SteamID of the person who got kicked
                arg.GetString(1); // Reason for the kick
            }
        }
    For unban it'd be exactly the same I'd presume but just check if the command is global.unban
