  2. wuzziefuzzy

    Hello Praxam,

    I hope i can help you with this problem,
    First thing that you need is this better chat plugin, you can download it from here: Better Chat for Hurtworld | Oxide
    Next u need to add that plugin to your plugin folder of oxide.
    Then u need to use this command in your game,: /usergroup add Friendsname(Fullname) admin
    After that you will see that your friend is admin, its not the same as the other admin but you can add permissions to that to make it almost the same , You do that with this command /grant user <player>(Or Group) <permission>
