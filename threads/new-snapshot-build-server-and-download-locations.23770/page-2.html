  12. Wulf Community Admin

    If anyone is still using the old location:
    • https://dl.bintray.com/oxidemod/Oxide/Oxide-Rust.zip (any game)
    Make sure to update to the new location:
    • https://dl.bintray.com/oxidemod/builds/Oxide-Rust.zip (any game)

    We've also switched things up a bit so that the Oxide-GAME.zip filenames will be the latest build from our master branch on GitHub (a release), and Oxide-GAME-SNAPSHOT.zip will be the latest build from the latest commit from our "develop" branch on GitHub (a snapshot). I'd recommend not using the SNAPSHOT builds unless otherwise instructed or if you experience an issue that is known to be fixed in a SNAPSHOT build, as these builds may contain breakage and WIP changes intended for testing and development purposes.
     
    Wulf, Feb 22, 2017
