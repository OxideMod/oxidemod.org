    Wulf submitted a new resource:

    SlackNotices - Sends connection and disconnection notices to Slack channel

    SlackNotices sends connection and/or disconnection notices to the configured Slack channel.

    Configuration
    You can configure the settings in the SlackNotices.json file under the oxide/config directory.
    Code:
    {
  "Channel": "",
  "Connections": true,
  "Disconnections": true
}
    Localization
    The default messages are in the SlackNotices.en.json under the oxide/lang directory, or create a language file for another...
