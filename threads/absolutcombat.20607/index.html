    Thanks for the feedback. I pushed this out fairly quickly so let me know if you see any issues.

    I understand what you want but my understanding is we are not allowed to make plugins that allow selection of custom skins (I actually made it that way initially. You mentioned Deathmatch which is setup in a way where you can type the skin id into the data file. This plugin works exactly the same. By default I am writing the skin ID and URL of the skin you had on the equipment when making the set; however after that set is made the data is stored in the AbsolutCombat_Data File. You can go into it and changed the URL to a different picture and the skin ID to a different value. The plugin will automatically update the picture as well.

    Hopefully this helps!
