    Can't make videos but, This may help
    1. Stand were you want the center of your zone to be
    2. Type /zone_add
    3. Type /zone radius XXX "any size you want it to be, biggest i have made is 200 meters"
    4. Type /zone Value True/false i.e. /zone name "whatever" /zone pvpgod true "A list of values is on the overview page"
    a very powerfull plugin, you can even use /zone size "## ## ##" to make zone a box shaped zone You can see how big it is if you fly away alittle bit from it and see the wireframe of the zone.
    to select different zones you have made ie. type /zone_list and use the number id to select it using /zone_edit 461523 or whatever number the zone is you want to edit.
