  8. Bootloop

    Globru said: ↑
    this plugin is way too confusing, it would be nice if someone could explain how the options work and how to make/read the id of a zone you created.
    Click to expand...
    It's actually pretty straight forward, but you have a point about the ID. You need to pull that form the config file. I've found the easiest way to create a zone is use Rust:IO to fond the centre of the location you want the dome or stand in the centre of a building you want to protect. Then type these one after the other in game chat...

    Code:
    /zone_add
/zone location here name *NAME*
/zone enter_message "*WELCOME*" leave_message "*GOODBYE*"
/zone radius *30*
    Then go to the config file and find the Town you have just created and just add the flags you want to use.

    for e.g

    Code:
          "enabled": true,
      "flags": "PveGod, Pvpgod, Sleepgod, UnDestr, NoDecay, NoPlayerLoot, NoCorpse, NoSuicide, NoBleed, NoStability"
    At the same time make a note of the ID and if you want to edit use

    Code:
    /zone_edit *ZONEID*
    I would also recommend using 'Zone Domes'. this will give you a visual of the zone and show you the actual boundary.
     
    Bootloop, Jul 4, 2016
