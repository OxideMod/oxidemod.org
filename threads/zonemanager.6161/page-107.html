    /zone_reset = clears all zones

    /zone_add = stand in center of zone and type this make sure you are at ground level

    /zone name "SAFEZONE"

    /zone radius 500

    /zone autolights true

    /zone enter_message "YOU HAVE ENTERED THE SAFE ZONE"

    /zone leave_message "YOU HAVE LEFT THE SAFE ZONE"

    /zone pvpgod true

    /zone pvegod true

    /zone sleepgod true

    /zone undestr true

    /zone noplayerloot true


    Now that my zone is setup I type:


    /zone_list


    The zoneid is the number on the left of my zone name "SAFEZONE", next I take this number and place it in the "zoneid" and the permission you want in "permission" associated with that zone, example /pz add <zoneid> <permission> or in our case replace the xxxxxxxx with the number you got on the previouos step:

    /pz add xxxxxxxx zones

    "zones" being the specific zone kicker tag.


    Next I added the kick players chat command:


    /zone eject true


    Next I added an sob group with oxide permission system:


    group add sob

    grant group sob zones

    usergroup add username sob
