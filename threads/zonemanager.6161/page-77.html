    Read the overview in full. It's all there.

    "radius XXX => set the radius of the zone (default is 20)"

    /zone edit zoneidorname
    zone radius 25'
    25 being the distance from the centre to the desired edge, in meters.


    I hope I'm not patronising you but you do /zone edit zoneidorname first
    From that point on you are editing that zone, so you just need
    /zone radius 25
    /zone undestr true
    /zone sleepgod true
    etc

    I assume from the overview you can do
    '/zone radius 25 undestr true sleepgod true' in one go
    but I haven't tried that.
