    As long as this is listed in your flags 'autolights' in /data/ZoneManager.json and you have set the time in /config/ZoneManager.json

    Code:
    {
  "AutoLights": {
    "Lights Off Time": "7.0",
    "Lights On Time": "18.0"
  },
    If you put fuel in it will be consumed but just won't run out at night. If you turn the lantern off and there is no fuel in there it won't turn back on unti there is, unless of course autolights kicks in again.

    I've found manually editing your flags is easier as if you add them via rcon or console it only adds the ones in the commend and wipes all the others.
