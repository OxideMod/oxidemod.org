  20. sami37

    Hello everyone,

    I am happy to help this community so.
    New personal thing i added today for some rust community players.

    Some player called for a notifier about zone under attack like a bank getting robbed.
    I am happy to announce you that i just finish this feature.

    How it work:

    You must edit the zone you want get notified in chat with /zone_edit xxxxxxID
    and do /zone notify true
    This new value is a flag like pvpgod/pvegod/ejectsleepers etc and got only 2 possible values False and True.

    Code:
                        if(HasZoneFlag(zone, ZoneFlags.Notify))
                    {
                        if(zone.Info.UnderAttack == false){
                            zone.Info.UnderAttack = true;
                            PrintToChat($""+string.Format("{0} is under attack.", zone.Info.name));
                            timer.Once(10f, () =>
                            {
                                zone.Info.UnderAttack = false;
                                PrintToChat($""+string.Format("{0} is under attack.", zone.Info.name));
                            });
                        }
                    }
    This code part block spam from chat, and put a message every 10s on chat if you are constantly attacking a building part in the zone.

    You will see new option called UnderAttack inserted in your data file associated to your zone, this option should never be set manually.

    @Rened, if you got better way to do it or think it will cause lags, just told me, i am not an expert with c#.

    @All, feel free to use it or not, if any problem with it, just told me in pm
