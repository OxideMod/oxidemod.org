  10. Alphawar

    BlackShadDOW said: ↑
    everytime i try to add a zone it says type /zone_edit XXXXXID but it does nothing what do i do ?
    Click to expand...
    Ok so when you create a new zone the mod will create a zone with a random ID#, you can view this using /zone_list.
    After you use /zone_add you are normally already set to edit that zone, I highly recommend that you then type "/zone name XX" <-- Replace the XX with a name that will tell you what that zone is when you see all your zones under /zone_list. for example "/zone name ArenaIsland" or "/zone name AdminIsland"
    If you then want to edit later after placing another zone you can go /zone_list, find the one with the name you want and type its id like this "/zone_edit IDGoHere"
     
    Alphawar, Mar 6, 2016
