  14. Earth2Luke

    This seems more-so a "preset warps" rather than "random warps" - I was searching for a plugin like this in hopes that someone could simply type "/rwarp" and it would just randomly teleport them somewhere on the map. Instead, they are presented with a list that requires them to type in the name of the predetermined warp.

    My suggestions for this plugin, take em or leave em. (They're just suggestions)
    1. Make /rwarp randomly teleport the player using the command to one of the predetermined warp locations.
    2. Add the command /rwarp list (disallowing someone to name a warp "list"), showing them a full list of the preset warps.
    (currently this is a limited size, and should be sorted into pages, as it only shows 15 of the latest created warps)
    3. Have more permissions for things like "who can see the list?" "who can specify their warp destination from the list?"
    4. Have the ability to determine which usergroup can use what warps. (For example, I'd only like certain people to be able to warp to a location, but currently everyone can use the warp)
    5. Also, I believe you should be allowed to cancel the teleport with /tpc as most people are already used to that with the NTeleportation plugin.
