    You are automatically added to the admin group if you are auth level 2 (ownerid) in Rust on connection. You do not need to do any of those steps though, as none of Rust's users.cfg stuff was touched by Oxide's folder location change. So unless you deleted Rust's users.cfg, then there'd be no reason to follow any of those steps.
