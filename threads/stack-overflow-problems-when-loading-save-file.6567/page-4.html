  4. Nogrod

    You need Visual Studio installed, or someone who has it...open the Visual Studio command prompt (a normal system one won't work) and run:
    Code:
    editbin.exe /stack:8388608 RustDedicated.exe
    you first need to navigate to the folder where the RustDedicated.exe is in (cd "path") or give the full path to the exe. This one increases to 8MB you can also try higher values...ex. 16777216 (16 * 1024 * 1024) for 16MB until the server starts...or you'll let upload someone a patched one, but i recommend to patch by your own, in case of an update you wont rely on somebody to do it for you
