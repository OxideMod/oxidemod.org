    Click to expand...
    The plugin uses the time of the server that Rust is running on, so just change:
    "Restart At (HH:MM:SS)":[
    "08:00:00",
    "20:00:00"
    So if you've set it to restart at 2am EST and your server clock is EST then set that to 02:00.
