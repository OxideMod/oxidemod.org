    Old School Quarries - Makes resource output from quarries better

    Old School Quarries makes quarry output closer to what it was before the August 3 wipe.

    General Information:

    When a survey charge is used, this plugin will check the resource deposit.

    If a deposit has only stone, there is a chance to add either sulfur, metal, or both ore deposits.

    If a deposit has high quality metal ore, there is a chance to add either sulfur, metal, or both ore deposits.

    If the deposit has sulfur ore, it has a chance to add a metal ore deposit.
