  1. Wulf Community Admin

    Wulf submitted a new resource:

    SlackChat - Sends all chat messages or keyword-based chat to Slack channel

    SlackChat sends all chat messages or only select chat messages based on configured keywords to a Slack channel.

    Configuration
    You can configure the settings in the SlackChat.json file under the oxide/config directory.
    Code:
    {
  "Channel": "general",
  "Keywords": [
    "admin",
    "cheat",
    "report"
  ],
  "KeywordsOnly": false,
  "Style": "fancy"
}
