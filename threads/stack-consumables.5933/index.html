    Use this Plugin http://oxidemod.org/plugins/stack-sizes.812/ But you will need to change the config, I have done the stack sizes you wanted.
    Code:
            "name": "ammo_rifle",
        "stackSize": 120
      },
      {
        "name": "ammo_pistol",
        "stackSize": 120
      },
      {
        "name": "ammo_shotgun",
        "stackSize": 120
      },
      {
        "name": "largemedkit",
        "stackSize": 5
      },
      {
        "name": "syringe_medical",
        "stackSize": 10
      },
      {
        "name": "bandage",
        "stackSize": 10
      }
    ]
  }
}
