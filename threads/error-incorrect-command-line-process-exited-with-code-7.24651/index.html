  2. Wulf Community Admin

    Oxide itself does nothing with how your server starts. If you are getting an error when starting, that would most likely be related to the Rust DLLs being used as Oxide includes specific versions of Rust in its DLLs.

    Also looking at your startup command-line, -maxplayers should be +server.maxplayers, and +datadir and -cfg should be removed as they aren't valid.
