  9. Kuro 123

    For anyone that doesn't have Notepad ++ I highly recommend you download it. It is much easier to edit the files than regular notepad and helps keep things organized. If you open the loot table configs in this program and set the language to Java you can see that the brackets are essential. Deleting the brackets isn't recommended and could possibly be luck that the server is adding them back in once uploaded.

    Here is a basic example of editing 2 different items in the loot table, order doesn't matter as long as the information matches the defaults.
    Code:
    # -- StormWall Loot Tables --

Barrel.Items                           {
    - '90% 1-25 x Water'
    - '90% 1-25 x Oil'
    - '90% 1-25 x Grain Seed'
    - '10% 10-100 x Grain Seed'
    - '10% 10-500 x Oil'
    - '10% 10-500 x Water'
}

# Example comment, this line won't get used for anything.

Washed up Crate.Items                  {
    - '10% 1-1 x Light Leather Boots'
    - '10% 1-1 x Light Leather Bracers'
    - '10% 1-1 x Light Leather Helmet'
    - '10% 1-1 x Light Leather Helmet'
    - '10% 1-1 x Light Leather Pants'
    - '10% 1-1 x Light Leather Vest'

}
    If you don't include anything it will refer to the default list, so in this example only those 2 sources of loot will be changed. Make sure to follow the format used when including new items. Copy and paste existing lines and simply change the item name.

    I personally leave spaces between each section so I can leave myself comments if needed. To make a comment use the # symbol before your line of text. I included an example in the code.
