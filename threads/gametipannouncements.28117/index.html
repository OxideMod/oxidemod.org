  1. redBDGR

    redBDGR submitted a new resource:

    GameTipAnnouncements - Send notifications to players as gametips

    A basic plugin that allows you to send gametips to players

    Permissions:
    • gametipannouncements.admin - Players with this permission are able to use to commands for this plugin
    Chat Commands:
    • sendgt <name / id> <message> <length> - This command sends a gametip to the specified player
    • sendgtall <message> <length> - This command sends a gametip to all players
