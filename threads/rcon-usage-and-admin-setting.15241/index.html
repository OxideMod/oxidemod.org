  3. Wulf Community Admin

    RCON is provided by Rust, not Oxide. You'd need a third-party tool like the one Resistance mentioned.

    You can always find the latest commands using "find ." via RCON, your server console, or in-game if you have permission.

    You can find a detailed thread stickied in this section about Rust's auth levels: Admin and moderators in Rust | Oxide.
