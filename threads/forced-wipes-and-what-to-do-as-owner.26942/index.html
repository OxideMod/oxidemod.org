    We generally try to get an update out in under an hour if we're around, but unless you want to run as vanilla you'd want to wait for an Oxide update to be released for that Rust update, otherwise you'd likely be breaking your server by trying to update it before it is compatible.

    For the seed, that's something you'd need to talk to your host about, but you should be able to set a new one via your server.cfg file or command-line.

    Wipes are generally only forced once per month, but pretty sure there are methods to restore the map after a forced wipe. Any other wipes are done by you generally.
