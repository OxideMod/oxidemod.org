    RecoveryItems - Wearing a registered item restores your HP

    If you wear an item it is recovered.
    Wearing a registered item to restore HP.

    Config
    HpRecoveryTime = 5 <- HP recovery interval

    Permission
    oxide.grant [group/user] [name] recoveryitems.use

    ChatCommand
    /rt [recovery interval]
    /ri add [ItemID] [Amount] <- Item register
    /ri del [ItemID] <- Item delete

    Example
    /rt [recovery interval]
    /ri add 2007564590 1 <- Item register
    /ri del 2007564590 <- Item delete
