  13. Pawlemann

    Write this in Server Console:
    usergroup add "youringamename" admin

    Then write in Server Console:
    server.writecfg.

    Just to make sure, write this in the Server Console too:
    ownerid <steamid> "player name" "reason".
    Don't forget the: server.writecfg after.

    You might have to restart the server for it to update and give you the admin status / permissions.
