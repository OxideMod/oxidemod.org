  7. _Trusty_

    The DATA-file HumanNPC.json lists all your NPCs.
    There you find: "protections": { .... , "Bullet": 0.0, ..... }

    "message_hurt": null,
    "message_kill": null,
    "protections": {
    "Generic": 0.0,
    "Hunger": 0.0,
    "Thirst": 0.0,
    "Cold": 0.0,
    "Drowned": 0.0,
    "Heat": 0.0,
    "Bleeding": 0.0,
    "Poison": 0.0,
    "Suicide": 0.0,
    "Bullet": 0.0,

    Change to some value (maybe 2.0), load changed file on server, reload HumanNPC.cs and you will see if it does what you want.
    I never tried it, but this is definitly ment to do exactly that.
    would be glad to hear if it works. cheers
