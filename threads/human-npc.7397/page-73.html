  2. Steenamaroo

    I found that I had to create the kit then apply the various parameters one by one.
    Also, name must be one word, lower case.
    So...

    First, kit yourself out as you want the kit to be. Weapon in the bar, clothes where they should be...Make it perfect.

    Do /kit add nameofkit (let's say it's NPC) - /kit add npckit

    Then /kit NPC authlevel 2 (redeemable by admin only)
    /kit NPC npconly (redeemable by npc only)
    /kit NPC items (copies your inventory to kit)

    Each one of these should give you confirmation output.

    Now go to your npc, edit him, and type /npc kit npckit (applies kit to that npc)


    PS: I'd love to know how you made them not lootable. That variable doesn't work for me.

    I've also found that modifying defend always defaults back to true and adding a kill message manually or in game causes the npc to stop responding correctly.
