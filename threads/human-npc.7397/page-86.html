    Not sure what to tell ya man. However, these are the steps I would follow on testing.

    Completely delete your HumanNPC config file from your server and restart with a fresh copy.
    If you are copying and pasting...stop, go manual.
    In game I would spawn a NPC and begin adding properties
    /npc name name
    /npc hostile true
    /npc damage 40
    /npc kit kitname
    /npc invulnerability true
    /npc health 100
    /npc enable true
    /npc_end

    If you go with those strings you can at least monitor the reactions first on just one NPC. If it goes well, you know where you stand.
    If it does not work I would look into deleting all files of HumanNPC from your server (Config-Data & Plugin)and start over clean.
