    The correct way that works for me:

    /npc_add
    after this, tweak your NPC with settings u like , name, radius, respawn, speed etc (i will put my setup for zombies i use on my server if someone wants to take a look how to do it , you will find it at the bottom) ... DO NOT EXIT NPC EDITOR.
    /kit add "random name" "random description" -authlevel2
    where "random name" is a kit name but, use all small letters and one word (thats how i do it) , "random description" is description about kit, i also use:
    /kit hide true
    so other players cant see it
    Add the clothes and weapons (if you want) to the NPC and do next:
    /kit items
    This will copy all the items to the kit.
    /npc kit "random name"
    where "random name" is the name of your kit.Your NPC will have the kit u have created.

    OPTIONAL

    If you want to make NPC move on the map:
    /waypoints_new
    With this you will start editing waypoints.Now stand at the place where your NPC will start his journey
    /waypoints_add SPEED
    So the SPEED, tweak it how u like, 1 is very slow, 2 i think is normal walk, 3 a bit faster, 4 faster ...
    For example:
    /waypoints_add 2
    Just copy this command and walk to other position on the map and add it again.Than walk to another position and add this command again and so on.After you have finished the walking path for the NPC do next:
    /waypoints_save NAME
    Where NAME would be the name for the waypoint that you want your NPC to use and follow.
    I like to choose the same name for npc, kit and waypoint, because its easy to remember it and use it later.
    After this you just need to close NPC editor with:
    /npc_end

    This is it, your NPC will have clothes, weapons and he will walk on map.

    THIS IS MY SETUP FOR ZOMBIES ON MY SERVER (maybe not ideal but it works, you can tweak it better)

    If you want to edit a NPC but you are not in edit mode (if you are in edit mode, skip this part to TWEAKS), do the following:
    /npc_list
    This will list all your NPC's, look for NPC's ID (few numbers near NPC's name), and when u have found the NPC u want to edit, do next:
    /npc_edit ID
    and with ID you put the numbers for NPC u want to edit.

    TWEAKS:

    /npc attackdistance 50
    /npc damageamount 20
    /npc damagedistance 3
    /npc damageinterval 3
    /npc enable true
    /npc radius 60
    /npc health 100
    /npc hostile true
    /npc invulnerable false
    /npc lootable true
    /npc maxdistance 100
    /npc respawn true 900     (I use 900=15min to respawn)
    /npc speed 5
    /npc stopandtalk false
    /npc needsammo true     (if your NPC use gun, also give him ammo)
    /npc defend true
    After this one more command left:
    /npc_end

    This is how i setup zombies on my server and they do attack players and also animals so its fun
    I hope this is helpful for you guys.
