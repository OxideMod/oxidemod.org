    Hey, I was having this same question, but I think I figured it out. When you make the kit, make sure you have all the items in the positions that you want the NPC to be using them. So if you want them wearing clothes, you must wear the clothes as well. This is from the description of the kit plugin:
    How to create kits:
    1) Empty your inventory
    2) Add in your inventory the kit that you want players to have (blueprints, weapons in the belt, armors in the clothing, etc)
    3) use /kit add "kitname"
    4) set the options via: /kit option1 value1 option2 value2 etc
    ex:
    /kit items max 10 cooldown 3600 description "Every hour kit, max usage: 10"
