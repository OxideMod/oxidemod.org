    that is something which could be changed in the plugin itself. Try to figure out how its done or ask Nogrod to change it. it would actually make sence to unrandomize these phrases.

    Line 1680 & 1681 :
    private static string GetRandomMessage(List<string> messagelist) => messagelist[GetRandom(0, messagelist.Count)];
    private static int GetRandom(int min, int max) => UnityEngine.Random.Range(min, max);

    could be tricky to change it, since its solved pretty nicely with this unity-randomize-method
    for showing msgs in order to each player, you would need a playerspecific counter. (player x used npc1 3 times, player y used npc1 5 times...)

    or you could use a npc specific timer, starting on first use, sending the messages in order, until either last message or timerend is reached => then resets timer and message to 0 (first)
