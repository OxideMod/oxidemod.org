  2. iDeath

    Rust.IO has nothing to do with running a rust server, they're a 3rd party who provides map integration.

    To wipe the server simply delete the following files/folders in:
    /serverfiles/server/server-identity/
    Code:
    user
storage
mapseed.sav
mapseed.map
Log.txt
     
    iDeath, Oct 30, 2017
