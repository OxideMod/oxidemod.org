    AbsolutGifts - Daily Gift Reward System - PlayTimeRewards.. Time Resets Daily

    This is a Gift Reward system to incentives players playing on your server for prolonged amounts of time.

    Admins setup "Gifts" which can consist of up to 10 items.

    A gift consists of 3 parts.

    1. A time requirement in minutes. A Number Pad is presented to the admin when creating the gift which has various times with the final button being 24 hours in minutes (1440).

    2. Gift items. These items are selected from visual buttons presented on the screen.
