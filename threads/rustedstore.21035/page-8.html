    You have your main language files in /Oxide/lang/en folder, if you wanna have bulgarian + english you should make another folder with the name of bl (bulgarian language) and translate all that you want there. Players could easily change the language using /o.lang bl

    Otherwise, you can just change the english files to bulgarian ones.

    After changing a language file, you have to restart the server
