  2. Ryan

    • Map customisation - Removing monuments / caves - have terrain editor
    • Loading custom Unity assets
    • Client side modding, playing custom sounds, images, assets
    • Crafting recipes server side
    • Sphere entity opacity / color customisation
    • Community UI optimised, allowing client side UI / more responsive UI from server - making it so we don’t have to destroy UI’s to refresh an element.
    • UI animations / effects / better hover effects
    • Server side tool cupboard placement
    • Item names / descriptions / behaviors all server-sided
    • Fix “craftMode” RPC to actually prevent the need of Workbenches.
    • Allow editing of native UI, such as removing an item from the crafting menu.
    • Ability to turn “graphics.branding” to 0... (it interferes with plugin UI’s)
    • Ability to override no build radius’ (inside radtowns for example)
    • Ability to allow clickable links in-game / chat / community UI
    There’s many more I’m sure I missed. But they’re the ones that first come to mind. I doubt Helk will listen, we’ve been nagging the developers for all sorts for ages now. Most of us are waiting until Sandbox releases / goes into Beta.
