  5. Maurice

    If the forced update doesnt change anything at the map generation and stuff, you just need to change the map name.

    If you have a map named this: proceduralmap.3000.1173436566.134.sav
    Just go ahead and change it to: proceduralmap.3000.1173436566.135.sav
    after the update. (Increase the number in front of the .sav by one)

    That only works if there hasn't been a map generation update (afaik - not sure tbh - could also work if there was a map generation update).
    With that you can keep your old map on the new update.
