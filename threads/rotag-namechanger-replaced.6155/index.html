  9. TheRotAG

    TheRotAG updated RotAG-NameChanger with a new update entry:

    Support to moderators and more players using at the sametime

    Added support to moderators and more players using at the sametime. Also, the plugin will print info in the console about who changed the name to what it has been changed, that way all changes will be loged. Yet, it will create a data file to record all the information about the "Default Name" that the player choose, this one that will be loaded everytime he reconnects or uses the command /nameoff. You can also, change the Authentication Level needed to use the plugin at the config file.
