    I recently have been getting a lot of people PM'ing me asking where they can find some basic plugins, like gather manager, help text, among many others.

    So its a fact that the plugins section its getting bigger and bigger over time and it's probably a good idea to start thinking in plugin categories for a better search, where people can select categories to find specifc plugins.

    So Plugin Devs should give their plugins the categories they see fit for their plugin's purpose so it's easier to sort the list.

    That's my suggestion, maybe someone else has some better ideas for this matter?

    Thanks in advance.
