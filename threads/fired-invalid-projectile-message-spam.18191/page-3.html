  13. Slut

    Every client=>server action requires verification because if they didn't, cheaters could kill people from the other side of the map by telling the server they're next to the player when they're not. Those messages just indicate that they failed a verification on an action they did. This is common when people have high ping, low FPS or the server lags. You shouldn't need to worry unless these values "0.1704503m > 0.1m in 0.3050159s" are significantly different.
     
    Slut, Apr 26, 2017
