  3. Colon Blow

    the entity is the BaseEntity or player gathering the resources.
    Quick code example to get player who harvested the resource and a quick chat reply to them.

    Code:
            void OnDispenserGather(ResourceDispenser dispenser, BaseEntity entity, Item item)
         {
            var player = (BasePlayer)entity;
            SendReply(player, "You harvested:");
         }
    But if the dispenser is a resource node like trees and rocks, I don't think they link to player at all. other than who is "harvesting" them at the moment.
    But if its like a Corpse it has a parentID I think. Depends on what type of resource dispenser you need to narrow down to?
