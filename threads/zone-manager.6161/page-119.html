  19. Steenamaroo

    The editing is done parameter by parameter, and most of them are listed in the overview.

    /zone_edit 12345 means any parameters you change will be for that zone.

    /zone_edit 12345 then /zone nostability true, means that zone will have no stability, for example.

    You can string the edits back to back
    /zone notrade true nostability true nogather true notp true .....for example.
