  15. Steenamaroo

    Hey Laithrum.
    I set up what you're describing with no prior knowledge and minimal experience.

    Kits are handled by GunGame cfg. I manually edited the json to have the kits I wanted.
    I recommend leaving them vanilla just to get the thing working and tested, though.

    Zones were set up using zone manager. It's fairly intuitive and the overview is helpful.
    Create zone - set radius (visual representation in game), then set parameters from the list in the overview.

    I edited Event manager json to have event 0 as 'GunGame' instead of the preset - deathmatch, I think?
    I created a spawn file with the spawns plugin and put the name of it in the Event Manager json too (or was it Gungame json? Can't remember).

    From there I just do event.launch 0 and gunagame automatically runs.

    If you have specific questions I can look up my configs for you.
    Most of this wont be relevant to Zone Manager, though, so maybe just start a conversation instead of replying here?
