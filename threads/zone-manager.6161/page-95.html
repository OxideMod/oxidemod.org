    With 94 pages of comments I might be asking for something someone else already has, if so I apologize...

    I am looking to make a small mod that will save a Zone Profile and be used to recreate a new zone with the same attributes. I usually do this via the .json file directly but felt others might want a UI based method to do it quickly.

    All that said, I was wondering if you could add a Hook to produce a list of each flag from the provided ZoneID.

    Not sure if this will work but maybe something like this:

    Code:
            private List<string> GetZoneFlags(string zoneId)
        {
            var flags = new List<string>();
            foreach (var entry in storedData.ZoneDefinitions.Where(kvp => kvp.Id == zoneId))
                foreach (var flag in entry.Flags)
                    flags.Add(flag);
            return flags;
        }
    Thanks!
    Click to expand...
    I changed my mind and decided I will just store the ZoneID then recall it. The problem with this approach is you need to have the zone forever to keep it as a template... honestly I dont care that much.

    But FYI to make my code work I added the following to Zone Manager. If you have any interest in adding it as a hook that would be awesome...


    Code:
            private bool UpdateZoneFlags(string targetZoneID, string sourceZoneID)
        {
            ZoneDefinition sourcezonedef;
            ZoneDefinition targetzonedef;
            if (ZoneDefinitions.TryGetValue(sourceZoneID, out sourcezonedef))
                if (ZoneDefinitions.TryGetValue(targetZoneID, out targetzonedef))
                {
                    targetzonedef.Flags = sourcezonedef.Flags;
                    SaveData();
                    return true;
                }
            return false;
        }
