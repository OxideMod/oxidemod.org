  3. Itchy Flea

    Daddy Cally said: ↑
    this might sound nooby but I am having trouble setting up the zone manager plug in. I have it installed on the server but I just cannot figure out in which order and what to input the commands to make the zone manager work. I am trying to place it around 2 event arenas on my server to stop decay and damage to buildings but I don't know how to make the area fit all around the rectangular arena any help would be cool please guys.
    Click to expand...
    Stand in the center of the area and run this command:
    • /zone_add
    Then run
    • /zone size "X Y Z" (The "" marks are required)
    To create a rectangular zone, or use
    • /zone radius XX
    to create a spherical zone. Replace the XYZ/XX with numbers. The mod draws on the screen a representation of the area covered by the zone. Once you've got the area covered, use
    • /zone nodecay true undestr true
    and that will protect the covered area from decay and damage.
    I also recommend using
    • /zone name "Zone Name Here"
    so you can easily see which zone is which once you have a few zones up and need to modify them at some point in the future.
     
    Itchy Flea, Nov 20, 2015
