  3. perfect_root

    A little trick ...
    To copy a single structure from the World separately, just commands:
    /copy NAME e false m building r 0.1
    Then when you insert, there will be only one structure you looked at.
    It can be very useful for non-standard construction. Or when it is undesirable to copy surrounding elements standing very close to the object to be cloned.
