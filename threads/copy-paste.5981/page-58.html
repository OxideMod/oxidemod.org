    I suspect you're correct, which is why I went ahead and figured out a way around the problem. it's more work, but is probably the easiest way to do it:

    1. Download your working serverrewards_rewards.json file to your computer;
    2. After the server wipes, delete the same file on the server;
    3. Create new NPCs in the correct locations for your new town or stores;
    4. copy the location information for each NPC from the newly created serverrewards_rewards.json into the old file.
    5. Replace the serverrewards_rewards.json on the server with the updated local file.
