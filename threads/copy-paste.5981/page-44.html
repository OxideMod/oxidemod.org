  9. VinylFresh

    sisko11 said: ↑
    command not found
    [DOUBLEPOST=1471893779][/DOUBLEPOST]can you give me what i must write , with the name or the steamid ?
    Click to expand...
    type this into your console under F1 in game or in Rusty admin if you use it
    oxide.grant user "yoursteam64ID" "copypaste.copy"
    and
    oxide.grant user "yoursteam64ID" "copypaste.paste"

    or if you want to give the whole server access do this
    grant group default copypaste.copy
    or
    grant group default copypaste.paste
    permissions like these are on a lot of plugins but use this format for most of them just changing the permission name relevant to the plugin ..hope this helps
