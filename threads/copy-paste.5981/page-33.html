  5. Reneb

    all of what you are saying can be overcome with all the options i've added.

    If parts don't copy, juste increase the radius of check: /copy filename radius 4
    If you made parts of the arena with copy and paste or build it's possible that the building doesn't have the same buildingid, so use proximity check /copy filename mechanics proximity
    so for your copy just do: /copy filename mechanics proximity radius 4

    As for the paste part, you can add an option to check if a part was already placed so do /placeback filename checkplaced true, this way it will check if the part is already there before placing it again, and this way it will only complete your building instead of placing back everything and make duplicate walls and stuff

    As for undo, undo only works for the last thing you placed, you can't use /undo 3 times to undo what you do 3 paste ago. Also if you reload the plugin it will reset the undo.
     
    Reneb, Jun 20, 2016

  16. Steenamaroo

    Hey Doctros,
    I don't know if you're using an older version or what, but I couldn't get your copies to work. It just said 'file is empty'.

    The zone thing is done through ZoneManager plugin.
    Stand where you want the zone centre (although you can move it later..this is just easier)
    and type /zone_add - That'll create a default small sphere zone around you

    do /zone size "20 200 20", which is suitable for my lighthouse, and it will create a tall cube zone. It shows the creator a black outline for a short period.
    From there just do /zone notability true, then look down and paste the lighthouse.

    Zone manager is very cool - LOTS of flags for PVPgod, PVE god, no looting, automatic lights, no stability...all sorts of stuff.
