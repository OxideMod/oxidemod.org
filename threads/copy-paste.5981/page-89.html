    You are overcomplicating things... .cs and .json are two different files, so DO NOT RENAME! Delete all copypaste-files, and start with a clean slate. upload your .cs (plugin) file to: /oxide/PLUGINS. This is where all of them go. The .json in config will be created automatically on first load of the plugin, ie. when you upload it, and the data folder is where copypaste will store the copied buildings as seperate files for easy access. This is the folder where you download builds you wan't to tweak in fortify, or upload builds from fortify to paste in-game.
