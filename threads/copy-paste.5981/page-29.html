    Click to expand...
    Corrosive said: ↑
    Just clear the tool cupboard or have no cupboard at all.
    Click to expand...
    Cupboards only set building rights, not ownership. Ownership is always set to the person who built the item (even after pasting). The only way you could do this would be setting up homes to not check for owners. Keep in mind this option WILL allow people to set homes in other peoples' bases if they get the cupboard as well, since building block will not allowed setting home (unless feature is disabled in the homes config).
