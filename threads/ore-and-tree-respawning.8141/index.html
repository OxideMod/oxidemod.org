    There was an issue early on with the latest update that caused a lot of resources to spawn underwater, you might want to try to run the following console command which should delete all ores and then they should respawn after a few minutes:
    Code:
    del autospawn/resource/ores
    Make sure your server is on the latest Rust update (protocol 1249 iirc)
