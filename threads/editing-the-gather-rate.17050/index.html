    Ok
    1: I would (extremely) recomend that you to some research and tutorials on how to set and run a game server.
    2: If you are hosting the server on your computer I would recomend to get a hosted server through a company (it will make things a bit easier)
    3: Get the server and the bare minimum working before doing anything.
    4: Once you get a bit of a better understanding then start looking at expanding with SIMPLE mods, ones that have no config.
    5: Then start looking at adding a few mods that allow config settings.
