    it is IMPOSSIBLE to run it without a graphical interface.
    most hosts provide at least 1 graphical interface.

    ex:
    soyoustart provide:
    ubuntu desktop 12.04
    and
    xubuntu desktop 14.04

    but you need at least the ubuntu (or xubuntu) 14.04 version to run rust with oxide.
