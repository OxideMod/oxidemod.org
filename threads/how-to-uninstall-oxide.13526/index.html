  2. Wulf Community Admin

    Just update your server with Steam. Oxide is a mod, so any updates will uninstall it. You'll likely have to force Steam to update though if you are already updated to the latest Rust version. The main file that makes Rust "modded" and loads Oxide is the Assembly-CSharp.dll, so simply replacing that with an unmodded version would work as well.

    Also, gates and turrets are not controlled or changed at all with Oxide alone, so you may want to check your plugins if it continues. If you don't experience the issue with vanilla Rust, it's possible that you're using an outdated Oxide build that contains and older Rust Assembly-CSharp.dll.
