  12. Wulf Community Admin

    n0stalgia said: ↑
    Ok im going to remove everything and start again following the tutorial on here then ill let you know how it goes
    Click to expand...
    RCON or server console method:
    • Run "ownerid STEAMID" in your RCON or server console
    • Run "server.writecfg" in your RCON or server console
    Manual method if server is running:
    • Add "ownerid STEAMID" to your server/identity/cfg/users.cfg file and save it
    • Run "server.readcfg" in your RCON or server console
    Manual method if server isn't running:
    • Add "ownerid STEAMID" to your server/identity/cfg/users.cfg file and save it
    • Start server
    All of these are vanilla commands and entirely a vanilla Rust feature unrelated to Oxide.
