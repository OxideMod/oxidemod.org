  1. DumbleDora

    DumbleDora submitted a new resource:

    Popup Message - Admins can send a popup with /pop playername title message

    This plugin allows admins to use /pop playername title message to display a popup message on a character's screen. This way admins KNOW players have seen their warnings!

    Note: the title of the popup can only be one word long, try the command on yourself first to be sure you're getting it right

    There's really not much else to it!
