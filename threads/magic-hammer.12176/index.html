    Magic Hammer - Fix anything instantly

    What is Magic Hammer?
    Magic Hammer is a simple plugin that allows users with permission to restore any fixable entity to it's maximum health points.

    Default Configuration:
    Code:
    {
  "bMessagesEnabled": true,
  "bUsePopupNotifications": false,
  "iProtocol": 1323,
  "tMessageRepaired": "Entity: <color=#F2F5A9>{entity_name}</color> health <color=#2EFE64>updated</color> from <color=#FF4000>{current_hp}</color>/<color=#2EFE64>{new_hp}</color>."

    Added modes: Mode 1 = Repair, Mode 2 = Repair. I intend on adding quite a few eventually.
