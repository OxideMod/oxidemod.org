    Pay special attention to the following 2 config options, because I just tested and it was working fine. If it's immediately saying destroy mode disabled when you try to use it, the second setting needs changed:
    Code:
    "bDestroyCupboardCheck": true,
"nModesEnabled(1=repair only, 2=destroy only, 3=both enabled)": 1,
    By default, only repair mode is enabled, and it requires you to have building privs.
