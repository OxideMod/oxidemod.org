    redBDGR submitted a new resource:

    ExplosiveRounds - Gives explosive bullets effects on impact

    This plugin gives explosive bullets a little bit more "kick". Makes it so there is an explosion effect where the bullet lands.

    Notes:
    A list of a few other explosion effects are listed on lines 18 - 21 in the plugin source. Change these via the "Explosion Effect" setting in the config.

    Permissions:
    • explosiverounds.use
    Configuration File:
    Code:
    {
  "Settings": {
    "Ammo Types": [
      "ammo.rifle.explosive"
    ],
    "Explosion Effect":...
