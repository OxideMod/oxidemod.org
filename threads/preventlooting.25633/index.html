    CaseMan submitted a new resource:

    Prevent Looting - Plugin provides protection for entity of player from looting by other players.

    PreventLooting provides protection for entity of player (boxes, furnaces, careers, etc.) from looting by other players. Ideal for PVE servers, but can be used for PVP.

    The plugin can use FriendsAPI, ZoneManager and use permissions.
    Using FriendsAPI allows you to allow your friends access to your entities.
    Using ZoneManager allows you to disable the plugin in certain zones.
