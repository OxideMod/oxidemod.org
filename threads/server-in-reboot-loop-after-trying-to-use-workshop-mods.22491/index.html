  5. CaptainNebulous

    If you start using workshop mods you need to change the format of what is typically the first few lines of your autoexec. The only format that has worked for me for the first line is the following (for example)

    -exec host 12871;queryport 12881;loadmod 12345;loadmod 12345;maxplayers 50;servername <color=#E85900>AZTEC WARFARE</color>3x/lowping/mets/clan/car/kit

    Also several workshop mods CAN ONLY BE INSTALLED, UPDATED/MODIFIED, OR DELETED at the exact same time you wipe your server. Once the savestate has begun, in many cases, any change to the loaded assets will break the save permanently and cause a restart loop. And if you have a restart loop it usually means you either attempted to do this.... or that you did not reconfigure the first lines of your autoexec into the wrong format.

    Not important but funny Note - I have noticed that in logs it detects "-exec" as an "unknown command" but without it it doesn't work at all. Try making sense of that! XD

    Good luck
