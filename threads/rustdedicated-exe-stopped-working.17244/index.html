    Timeouts are generally network issues. The server crashing could be from many things, but your best bet would be to try a vanilla install first, if it works fine then go from there by adding back to it slow. The server crashing could also be be related to hardware issues or issues with your host. There's no indication that your install has any real issues though, so I'm leaning more toward host or hardware issues.
