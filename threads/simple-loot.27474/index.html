  1. Kappasaurus

    Kappasaurus submitted a new resource:

    Scrap Multiplier - Modify the amount of scrap dropped.

    Scrap Multiplier allows you to set a multiplier on how much scrap is dropped via config file.

    Configuration
    You can configure the settings in the ScrapMultiplier.json file under the oxide/config directory.
    Code:
    {
  "Settings": {
    "Scrap multiplier": 1
  }
}
