    C# rewrite

    Entire plugin has been rewritten in C# and quite some things have changed.

    The plugin no longer uses a player's AuthLevel to determine the player is allowed to use the command or not. It now uses the Oxide Permission system. The permission that the player requires is CanUseGodmode

    The plugin now offers two commands for players with the above flag, /god and /godlist.

    /god is used to toggle godmode on the player or when specifying the name of another player it will allow you to toggle...

    Click to expand...
    In the in-game console or rcon/server console you simply run the command:
    oxide.grant user <name or steamid> CanUseGodmode
    or if you use usergroups, like Loup-des-Neiges mentioned:
    oxide.grant group <group> CanUseGodmode
