  7. Foxy

    32kneegrows said: ↑
    Im a noob at thos, i have rusty connected to my server and everything and downloaded the plugin, but have to clue on how to use it, please help
    Click to expand...
    Do you have FTP access to your server? You should do and you need to add the plugin file to the \server\IDENTITY\oxide\plugins folder, the IDENTITY bit is defined in your command line options (or should be) but its likely the only folder under \server\.

    Once you've done this (you can do it while the server is running) you should see some stuff in Rusty about a plugin being loaded. You then need to give the permission to use it to whatever group you want (probably just admin) as defined in your oxide.groups.data folder (don't edit this file use the permissions command all detailed here http://oxidemod.org/threads/using-oxides-permission-system.8296/)

    the permission is canusegodmode

    good luck!
     
    Foxy, Oct 27, 2015
