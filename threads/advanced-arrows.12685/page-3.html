  6. Wulf Community Admin

    Agonos said: ↑
    Hello. Could we have the exact command to type in the console to get the permissions please ?
    Click to expand...
    Using the Oxide permission system | Oxide

    grant group GROUPNAME PERMISSIONNAME
    or
    grant user USERNAME PERMISSIONNAME

    If you want to grant all arrow types, you can use a wildcard like:
    grant group default advancedarrows.*
