  11. sami37

    A temp solution that can open the security door and close it after 5s.
    Code:
            void OnDoorKnocked(Door door, BasePlayer player)
        {
            if (door.PrefabName.Contains("door.hinged.security"))
            {
                door.SetOpen(true);
                timer.Once(5.0f, door.CloseRequest);
            }
        }
    It will open for everyone, so that mean you don't need the card and such thing, it destroy the puzzle concept, so i recommand to use it only if your puzzle is already broken.

    This solution can be used and edited to check player inventory if necessary.
