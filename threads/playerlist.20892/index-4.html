  1. Wulf Community Admin

    Wulf submitted a new resource:

    PlayerList - Shows a count and list of all online players unless hidden

    PlayerList allows players to see a count and list of all online players that aren't hidden.

    Permissions

    This plugin uses Oxide's permission system. To assign a permission, use grant user <username|steamid> <permission>. To remove a permission, use revoke user <username|steamid> <permission>.
    • playerlist.allow (allows player to use the players list)
      Ex. grant user Wulf playerlist.allow
      Ex. revoke user Wulf...
