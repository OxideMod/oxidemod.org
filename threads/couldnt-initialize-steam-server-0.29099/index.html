  5. Wulf Community Admin

    When you see a "Couldn't initialize Steam Server", it often means that either the port you are seeing is inaccessible or the IP address you are using is not actually one that is assigned to that machine. This could be because of a common port being set in your startup .bat or because of your external IP being set when you are behind a router, so make sure you aren't doing either of those.
     
    Wulf, Jan 19, 2018
