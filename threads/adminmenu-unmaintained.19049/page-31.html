  11. Balls0fSteel

    First time the Admin Menu didn't work. Then I noticed that the python files needed the libraries so I followed the instructions. I shutdown the server, I loaded the libraries and when I started the server it failed to start. I knew there was a problem and I knew it was me so I contacted my host and they gave me a list of sensitive files that need to be checked before being uploaded.

    To everyone: be sure your host is available to upload those files.

    In the end I used FileZilla to transfer the .dll files one by one and it took a while because the host checked every single file while uploading them. It took few minutes to upload all them but when I started my server again it worked.

    About the use of the plugin and the available commands: your "group" (moderator or admin) needs to have ALL the permissions of a plugin to use it. For example I gave myself the "helicontrol.admin" for the HeliControl plugin and it was not enough so I gave myself all the permissions and now I can control it. About group permissions I'm still checking
