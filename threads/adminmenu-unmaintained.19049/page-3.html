    You don't have to use the console command, I just had a look and you can already call the function to check straight out. I also just updated the ClearPlayerData function to return either a success or fail message.

    Code:
    object CheckPoints(ulong ID) // returns (int) user point count, or null if the user has no data (already existing)
object ClearPlayerData(BasePlayer target, ulong ID) // returns either a success or fail message. You can use either of those 2 parameters (you don't need both), if you use the BasePlayer it will have the target users displayname in the message, or if you just use the ulong it will only show the ID in the message
    Just so you know, you can call any function from other plugins whether there is a [HookMethod()] attached to it or not. The only time it won't work is when the function is public without a [HookMethod()] attached.
