  2. Emi226

    temporary solution until the author will continue his work at this plugin..

    Here is the fix.. replace entire old class TextBox (lines 756-770) with :
    Code:
    class TextBox:
    def __init__(self, position, text, textColour, fontSize, align):
        self.ID = str(Guid.NewGuid()).replace('-','')
        self.pos = position
        minPos = "%f %f" % (position[0], position[1])
        maxPos = "%f %f" % (position[2], position[3])
        self.textbox = Cui.CuiButton()
        self.textbox.Button.Command = "null"
        self.textbox.Button.Color = "0.1 0.1 0.1 0.7"
        self.textbox.Text.Text = text
        self.textbox.Text.Color = textColour
        self.textbox.Text.FontSize = 12
        self.textbox.Text.Align = align
        self.textbox.RectTransform.AnchorMin = minPos
        self.textbox.RectTransform.AnchorMax = maxPos

                msg = lang.GetMessage('godPlayer',self.Plugin,admin.UserIDString)

                rust.SendChatMessage(admin, msg, None, '0')
        elif arguments[0] == 'ungod' and plugins.Exists('Godmode'):
