    Whisper - Message in-game players from console

    Allows the console to send messages to a player on the server. Useful for donation notices or a way a simple way to "whisper" players on your server.

    You can also use this as a command in-game if you want. But it requires the player to be an admin or have the permission.

    This plugin uses Oxide's permission system. To assign a permission, use grant user <name or steam id> <permission>. To remove a permission, use revoke user <name or steam id> <permission>....
