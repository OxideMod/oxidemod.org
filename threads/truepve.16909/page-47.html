  6. ignignokt84

    @Shark-A-Holic - If you're using the correct directory, then there must be an issue with your Rust or Oxide installation on the server. There's no way the latest version of TruePVE can throw an error regarding BaseResource, because BaseResource doesn't exist in it. The most likely possibilities are either your server is picking up an old version of TruePVE from a different directory, or your server is hiding or caching the plugin somewhere and using that. But I think the first option is the most likely. You could also double check that you don't have the Oxide path (+oxide.directory) defined in your startup - maybe it's pointing to another location?
