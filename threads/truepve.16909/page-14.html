  6. ignignokt84

    ignignokt84 updated TruePVE with a new update entry:

    0.4.4

    - Added configuration entry: heliDamagePlayer - handles heli damage for players

    This option, coupled with heliDamage allows the heli to be customized to damage players but nothing else, or vice versa. Setting heliDamage: true and heliDamagePlayer: false will allow anything except players to be hurt by the heli. To enable players to be damaged, but nothing else, set heliDamage: false and heliDamagePlayer: true.
