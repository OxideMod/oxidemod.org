    Hey just wanted to let peps know about a custom plugin im creating. Its nowhere near finished but should be ready for a live test within a week ( hopefully ). - will post server details at a later date

    Its kinda like an RPG where players earn xp by playing the game like they normally would. For example gathering resources, crafting, killing animals and players will all give the player small amounts of xp. They can then spend this xp using the skill tree to increase certain skills

    I have only added 2 skills at the moment, Lumber Jack & Miner where increasing these skills will increase the amount of xp & resources gathered from trees and rocks.

    Other skills i plan on adding are things such as:

    Hunter - higher damage to animals from weapons, increased resources from animals
    Strength - Small amounts of max health increase ( like 10% / 20% max ) and increased stack sizes per player
    ( not sure if the stack sizes per player is possible, i know stack sizes can be set server wide )
    Crafting - reduce the time taken to craft items.

    IOf anyone has any ideas for skills please let me know.

    Players can also view stats specific to the server running the plugin such as amount of resources gathered, deaths & kills ect ect.

    I also plan on adding NPCs that wander around the world but havnt looked into the possibility of that yet.

    http://requiemrust.gamingdeluxe.net/images/screens/skills-1.jpg
