    Building Block - Fully prevents building in the building blocked zone. Fixes the "Devblog 158 of 4th may"

    This is a very simple plugin that prevents building in the building privilage for the players, that don't have access to the cupboard.

    Devblog 158 gave us an "awsome" feature - devs allow raiders to build twig constructions even if they don't have access to the tool cupboard. So - here is the fix for this.
    Now the raids would work as they use to.
    Config file:
    "Bypass block privilage" - allow user with this permission to bypass the block of the plugin, so they will be able to build twig...
