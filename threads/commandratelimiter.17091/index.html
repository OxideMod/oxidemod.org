  1. Calytic Community Admin Community Mod

    Calytic submitted a new resource:

    CommandRateLimiter - Limits the rate players may use (all) commands by kicking them

    Features
    • Limits the rate players may use all commands
    • Configurable cooldown (in millseconds, default 195ms)
    • Configurable kicking after certain amount of offenses (default 10)
    • Configurable messages
    • Configurable server logging
    • Configurable number of seconds before rate limiter is discarded per player (default 5s)
    Testers please note that this version of the plugin emits a warning about a conflict with rustcore. I have a version ready that does not do this.
