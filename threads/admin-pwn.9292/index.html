  1. DumbleDora

    DumbleDora submitted a new resource:

    Admin Pwn - Toggleable super damage/repair for admins

    Pwn gives admins access to /pwn and /checkpwn. By default the damage and repair boost is set to 10000, and this applies to every block or entity an admin damages/repairs.

    If you're repairing some things, you will still need the required items to repair (iron, stone, etc) but only 1 of them should be enough.

    As an admin, this is just something I made a while back when players needed me to destroy stations or crests they could no longer pickup; so I figured others may want it also. It even...
