  1. PaiN

    PaiN submitted a new resource:

    XPSync - Synchronize xp across servers using MySQL

    This plugin allows you to keep XP across many servers using MySQL.

    I just want to say that it was my first experience with MySQL and if you find something weird or an issue just make a post in the plugin's thread.

    MySQL Settings:

    - Create a new table named 'xpsync'
    - Structure =>
    id BIGINT(32) PRIMARY KEY
    xp FLOAT

    Config File:
    Code:
    {
  "Database": "databasename",
  "Host": "localhost",
  "Password": "",
  "Port": 3306,
  "User": "root"
}
