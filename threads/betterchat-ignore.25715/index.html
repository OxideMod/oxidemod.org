    BetterChat Ignore - Players can ignore chat messages from other players

    Tired of having to listen to toxic/abusive/annoying players in global chat?
    Now you can ignore them!

    Requires both BetterChat and Ignore API plugins to work

    Chat Commands:
    /ignore add playerName
    /ignore remove playerName
    /ignore list
