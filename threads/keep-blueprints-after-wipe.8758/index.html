  2. seanterris

    do you want to wipe the items too? if so, all you have to do is to go the server folder, then "persistance" and "players", and delete every folder EXCEPT those with IDs of your VIP players.
    AFAIK no official oxide plugin is allowed to interfere with rust's players/maps saves, so you would have to use something else if you don't to do this by hand every time you wipe.

    if you have FPT access to your sever google "how to regulary backup files from ftp" and you will find tons of solution. if you are renting the server ask your provider what would be the best way to backup specific files on regular basis, they might already have some built-in tools in their control panels that serve exactly this purpose.
