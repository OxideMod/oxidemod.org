    Ankawi submitted a new resource:

    KDR Scoreboard - Enables scoreboards that can show Kills, Deaths, or K/D Ratio.

    This plugin uses the Scoreboard API to add more available options to display.

    Installation:
    • Install Scoreboards API
    • Install KDR Scoreboard
    • Use the commands within the Scoreboard API (i.e: /scoreboard select | disable | list)
    Scoreboards Available:
    • K/D Ratio
    • Top Kills
    • Top Deaths
    Commands:
    • /kdr - Displays personal stats
