    Click to expand...
    Sounds like you're using an old Oxide build with an older Rust patched DLL then because Oxide has nothing to do with animals and doesn't touch any of the game mechanics in Rust. There were numerous issues with animals this last Rust update, which I'm pretty sure they addressed the known ones with today's Rust update. Oxide provides hooks for plugins to use, it doesn't actually change the game without plugins.

    Installing an old Oxide build would mean installing old Rust files on top of your server, because Oxide modifies 3 of these files for Rust. So if there were animal issues in this older Rust version (there was), then you'd be downgrading your server to that and re-introducing those issues.
