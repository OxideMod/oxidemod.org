    Not showing in the server list means you're either not using the latest build that has the latest patched DLL for Hurtworld, or the server isn't fully started. Oxide does not control if the server is listed or not, that is determined by your server's version of Hurtworld. The Linux builds work fine aside from the compiler, which will only work on x64 versions of Linux. We're adding support for x86 soon, but not sure when.
