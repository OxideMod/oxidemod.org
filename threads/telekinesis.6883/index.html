    Telekinesis - Move objects by thought!

    How to use?
    • Use the USE Button (Default: E) to grab the object
    • Use Mouse Wheel to move object forward or back
    • Object will move to your look position
    Commands
    • /tls <-- toggle/untoggle telekinesis mode
    • /tls speed [speed] <-- set move speed for grabbed object
    • /tls max/min [distance] <-- the max/min distance at which an object can be grabbed
