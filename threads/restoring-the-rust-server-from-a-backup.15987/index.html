  2. Wulf Community Admin

    If you haven't ran the server recently, those backups will most likely not be usable anymore as Rust forces wipes of the map and blueprints monthly. Plugins do not really change or get removed, so those are generally fine. If you'd like to try to restore from a backup though, just copy the contents of one of the save folders to your server root and overwrite it all.
