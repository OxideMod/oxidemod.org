    Dora submitted a new resource:

    Guess The Number - An event that requires player to guess the correct number

    Chat Commands:
    • /startNumber - Start the event. (1-100)
    • /startNumber <minNumber> <maxNumber> - Start the event with your desired random rolls.
    • /endNumber - Ending the event.
    • /number <number> - To reply your answer.
    Rewards:
    • 10% XP based on current unspent XP.
    Configurations:
    Code:
    {
  "maxNumber": 101, (min 1)
  "minNumber": 1, (min 1)
  "repeatInterval": 1800 (seconds)
}
