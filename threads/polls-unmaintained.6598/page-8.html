  8. Feramor

    Currently, since there is no support for textbox in CUI, you may create polls using /poll PollType Timer OtherArguments.For example if you want to create a poll to Kick a user /poll Kick 60 Feramor where 60 is timer to poll run and Feramor is the username, or to create a poll to call an Airdrop you can use /poll Airdrop 60 again where 60 is timer to poll run.
