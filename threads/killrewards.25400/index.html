    KillRewards - Receive items, weapons with ammo and full health when killing players.

    Receive a list of items and weapons with a custom amount of ammo when killing players. The killer's health is set to the value specified in the config file.

    The number value next to the weapon name in the list of weapons is the amount of ammo the weapons is spawned with.

    If you don't want the player to receive any new items but only full ammo in the players' current weapon, set the OnlyReloadCurrentWeapon config value to true.
