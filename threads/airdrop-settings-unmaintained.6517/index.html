  1. t0pdevice

    t0pdevice submitted a new resource:

    Airdrop Settings - airdrop limit parachute drop loot cargoplane supplydrop

    This plugin allows to change the airdrop settings:
    • Minimum number of players on server to run airdrop:
      Admin command: /airdrop_min <number>
      Config: MinimumPlayers - number
    • Frequency of the call airdrop
      Admin command: /airdrop_freq <minutes>
      Config: DropFrequency - minutes
    • Remove parachute after supplies landing
      Config: RemoveParachute - true/false
