    The ammoType setting uses the shortname of the item. The default is "ammo.rifle" Some other options might be "ammo.pistol" or "ammo.rifle.explosive".

    Keep in mind that even if this setting works, it won't change the firing animation, only the type of ammunition that the turret consumes and the sound it makes when firing.
