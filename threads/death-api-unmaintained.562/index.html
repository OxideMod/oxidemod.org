    How to update!
    1. First remove the old DeathHandler.lua file
    2. Go to the oxide\data folder and dealth the cfg_DeathHandler.txt
    3. Restart the server or as an admin type in console: oxide.reload DeathHandler
    4. Enjoy

    New Config Options
    Code:
    self.Config.zombie = true
    self.Config.player = true
    self.Config.bear = true
    self.Config.wolf = true
