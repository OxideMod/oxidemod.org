  17. NyloLive

    OnesnZeroes said: ↑

    Click to expand...
    This is how I've done it:

    1. Stop the Server
    2. Delete:
    rust/oxide/config -> ZLevelsRemastered.json
    rust/oxide/data -> ZLevelsRemastered.json
    rust/oxide/data -> ZLevelsCraftDetails.json
    rust/oxide/plugins -> ZLevelsRemastered.json

    3. Start the Server again and wait until it's completely started
    4. Stop the Server again
    5. Input the plugin in rust/oxide/plugins
    6. Restart the Server

    Hope this helps
