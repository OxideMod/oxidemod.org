  10. Maximoryl

    Nironn said: ↑
    hello, i do not remember how to set the maximum level to 20 and to multiply the x3 on it, can anyone help?
    Click to expand...
    To set max levels you'll change these variables in the config...
    "LevelCaps": {
    "A": 200,
    "C": -1, <----change to 20
    "M": 200,
    "S": 200,
    "WC": 200
    }

    If you are trying to set your night bonus for XP to x3 then take what you have under "PointsPerHit" and multiply those values by 3. Put those numbers under "PointsPerHitAtNight".

    If you are trying to set the resource multiplier its under "DefaultResourceMultiplier".

    If you are looking to change the 'per level' bnus for resources.. find that under "ResourcePerLevelMultiplier"
     
    Maximoryl, Oct 4, 2017
