    I would recommend using RustCui - Editor to get that specific location just change the cui editor to 1080p then take the min and max from z-levels and put them into the cui editor that way you get the right size for the panel then just drag it around the ditor to the position you want.

    (Note: the editor used to have an image that gave you an idea of the rust bg but looks like the image has been removed, however you can still work out a lot easier the needed min/max x y)
