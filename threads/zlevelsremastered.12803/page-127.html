  14. Togoshige

    VsUK said: ↑
    I noticed that the exp earned to level up like woodcutting & mining level up pretty quickly & don't appear to slow down the rate after each level.
    Where do you slow down the level increase per level up? So each level takes longer to level up than the previous?
    Click to expand...
    You can edit "PointsPerHit" or "ResourcePerLevelMultiplier" or both!
    Below is section of my settings from my Primitive server:

    Code:
      "Settings": {
    "PointsPerHit": {
      "A": 0,
      "M": 1,
      "S": 1,
      "WC": 1
    },
    "ResourcePerLevelMultiplier": {
      "A": 0.3,
      "M": 0.3,
      "S": 0.3,
      "WC": 0.3
    },
  }
