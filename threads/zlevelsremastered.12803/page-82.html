  6. Aye Ghost

    Maajo SK said: ↑
    This not gived me clue how to move ZLevel HUD
    Click to expand...
    I've seen the same question asked time and time again on this thread... "how do I move the GUI/HUD/Bars" etc..

    Play with these damn numbers here: server -> oxide -> config -> ZLevelsRemastered.json.
    Code:
      "CUI": {
    "Bounds": {
      "HeightLower": "0.020",
      "HeightUpper": "0.135",
      "WidthLeft": "0.70",
      "WidthRight": "0.840"
    },
    That's literally all you need. Play with those and figure it out. After you update the numbers go into the in-game console and type "oxide.reload ZLevelsRemastered" and BOOM it updates without having to reset the server.
     
    Aye Ghost, Feb 17, 2017

    Code:

      "A": 2.0,
      "M": 2.0,
      "S": 2.0,
      "WC": 2.0
    },
    2.0 = 20%
    0.2 = 2%
