    SouZa submitted a new resource:

    HW Clans - Create/join clans and write on clan chat

    HW Clans is a simple clan system to Hurtworld. It has some ideas already implemented on Rust:IO Clans for Rust from dcode.

    Players are able to:
    • Create new clans.
    • Join existing clans.
    • Promote members to moderators to manage the clan.
    • Write on clan chat with /c "message"
    • Toggle global/clan chat. (Don't need /c to chat on clan).
