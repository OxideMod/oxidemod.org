  9. SouZa

    [ U P D A T E ] Since BetterChat doesn't seem to be updating in a while, I upload here BetterChat.cs (1.1.3) and HWClans (1.3.1) with some modifications to allow:
    • [Clan Message] prefix to be displayed when talking in clan chat, because currently BetterChat overrides that. It is possible to change the prefix to whatever server owner wants, like: Clan | Clan Message | Clan Chat | ...
    Note 1: This prefix already shows if BetterChat is not installed. This hotfix is just for servers using both BetterChat and HWClans.
    Note 2: An example of the output can be seen in this screenshot: Screenshot (Colors and prefix can be changed by server owner).
    Note 3: In BetterChat Config file, on the field "Formatting" for every group: remove the space between: {Title}{this_space}{Name}. This will remove a space before PlayerName when chatting. So it stays like: {Title}{Name}. Also on the field "Title" add an extra space after the title, like: "[VIP]{space_here}", if you're using groups.
