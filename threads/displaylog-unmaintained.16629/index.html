    DisplayLog - Displays log contents in chat

    DisplayLog is a plugin which allows admins to view the contents of log files without having FTP access to the server.

    Use /dlog <log name> to open the reading mode for the specified log file. The Rust chat is fairly limited, so logs are seperated into multiple pages.
    When opening the reading mode, the first page of the log is read.
    Logs are read in reverse because most recent events are usually the ones you care about the most.
    To navigate to the next page, use /dlog next.

    Adding cmds\\.txt to the whitelist allows people with the displaylog.use permission to display the contents of the cmds.txt log.
    If you're logging /pm and /r using Spyon, then people with displaylog.use will be able to read cmds.txt using this plugin and read uses of /pm and /r as well.
    This is essentially nothing but a text file viewer for the Rust chat.
