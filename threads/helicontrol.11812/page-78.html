    Shady757 said: ↑
    Hi, firstly, a bit of clarification: Koenrad is the original developer, I took over the project a while ago.

    Now more importantly:
    The way loot is set up in this plugin is a bit outdated, not as modifiable or as easy it should be. In the overview of the plugin, you may have missed an example .json file, which has lots of loot tables already pre-defined. It can be found on the overview where it says: "HeliControlData (click this)". Once you download the file, you should be able to get a better example of what a large loot table looks like.

    In fact, I'd recommend just taking that large pre-made loot table and just modifying the items and amounts to whatever you like.
    That example json is a bit old and still has things like blueprint fragments in there, which you'd want to remove. There is nothing really more to the loot table except copy and pasting the stuff you want more of. Also, jsonlint still works, but seemingly only in https now. https://jsonlint.com

    Lastly, it may be better or easier to use another plugin that modifies loot tables, because they're designed exactly for that purpose. The feature for modifying helicopter's loot in this plugin isn't really fleshed out that much, and lacks a lot of accessibility and features that other plugins have.
