  15. Shady757

    Schaffe89 said: ↑
    The problem is they can call the heli to every target now? And the Helicontrol mod kill the serverperformance. Is that normal?
    Click to expand...
    HeliControl has little to no effect on server performance, unless you're calling a lot of Helicopters at once, which is normal and isn't the plugin's fault, Rust was never intended to have more tahn 1 helicopter active.

    If you want them to only be able to call to themself, you need to do:
    Code:
    oxide.grant user UserName helicontrol.callheliself
    Which is described in the overview, unless you're saying with this permission they can still call on everyone? You might be granting "helicontrol.callheli", which is basically full access to the command, which allows you to call as many as you want, to whoever/wherever you want. "helicontrol.callhelitarget" and "helicontrol.callheliself" are intended to be used with the cooldown and or limits, while "helicontrol.callheli" is not.
     
    Shady757, Oct 3, 2016
