  13. Shady757

    BeautyKilled said: ↑
    i called a bunch of heli to my position, on two different servers, and both times they did not spawn crates. this is first time ive done this in a long time so i dont know when this problem started. any advice?
    Click to expand...
    You may have granted yourself all of HeliControl's permissions, which would also grant you the permission "helicontrol.nodrop". You can either revoke this from your user or group, e.g:
    Code:
    oxide.revoke user UserName helicontrol.nodrop
oxide.revoke group GroupName helicontrol.nodrop
    Or you can use callheli through server rcon/console, which should ignore those permissions. If you don't have those permissions, still happens after revoking them, or doesn't even drop crates when the Helicopters are called through console (or come naturally), then there is a different issue, but I think this is probably it.
     
    Shady757, Sep 16, 2017
    #1633
