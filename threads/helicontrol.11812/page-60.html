    You mean using it in a modified version of HeliControl or in your own plugin that references it? If it's the latter, I don't think you'll be able to get the player's cooldown times (at least not very easily).

    However, if you're modifying HeliControl itself... You can use GetLowestCooldown(BasePlayer player) and GetHighestLimit(BasePlayer player), to get the player's lowest/highest permission, respectively. This means if they have 3 cooldown permissions, and one of them is 5 seconds, one is 10, and one is 15, it will pick 5. Same for limits but reversed. If you're wanting to do something else, you'll have to explain a bit more as I'm not sure what parts you need to understand.
