    You can quickly fix BetterLoot from overriding helis by opening the plugin in a text editor, and finding the line: "private Regex crateEx = new Regex" and replacing it with this line:

    Code:
     private Regex crateEx = new Regex(@"crate_normal"); //_normal prevents heli crates from being detected

    Unless you have the option to disable rockets enabled, it should be firing them. However, I think without using the console command provided by rust (heli.strafe) it will only strafe/shoot rockets if you're hiding inside a base.

    Also, inside of the "BetterLoot.cs" plugin file, you search for:
    Code:
    private Regex crateEx
    and replace the entire line with:

    Code:
     private Regex crateEx = new Regex(@"crate_normal"); //_normal prevents heli crates from being detected
