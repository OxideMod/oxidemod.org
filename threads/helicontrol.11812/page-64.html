    *Panda said: ↑
    Hello, can somebody help me, how i can set cooldown between spawning next heli? (etc. players spawn 1 heli and must wait 15 min to be able to spawn next heli). Can i set permission that default players can spawn heli to world, but vip players can spawn heli on yourself? Thanks for any help
    Click to expand...
    Grant a limit permission to a user or group, and then grant a cooldown permission that's set to 15 minutes in the config. With those settings, a regular player can call a Helicopter every 15 minutes until they hit their limit (if you've set a limit). If you want VIPs to be able to call it to themselves, grant them the permission "helicontrol.callheliself"
