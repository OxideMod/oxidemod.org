    The only way HeliControl may be stopping that is if you have "Disable Default Rust Spawns" set to true. Unloading the plugin doesn't actually reset this value, so if you wish to undo that, you'll have to remove HeliControl (or set the option to false), and then restart the server. I don't think it has any effect on CH47 spawns, but it's worth a try.
