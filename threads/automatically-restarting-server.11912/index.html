    i found out even easier way to do what u want to do...

    first you have to disable mentioned "Application has crashed dialog", doing change in the registry:

    Code:
    HKEY_CURRENT_USER\Software\Microsoft\Windows\Windows Error Reporting
    Set the DontShowUI entry value to 1 [MORE DETAILS]

    You should also disable Windows Error Reporting, as shown at the same link above to make sure there will be no delays before rust server restart...

    Then you start your rust servers using classic infinite .bat script, like:
    Code:
    :loop

C:\start_rust_server_01.cmd

goto loop
    this should be it...
