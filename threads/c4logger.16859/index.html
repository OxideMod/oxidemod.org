    C4Logger - Saves informations about the use of C4.

    C4Logger saves useful informations about the use of C4.

    Permissions
    c4logger.message
    Get a message everytime someone uses a C4.

    Config file
    Code:
    {

    "Date Format": "yyyy/MM/dd HH:mm:ss",
    "Ignore Self": false,
    "Max Age (Days)": 3.0,
    "Players Near Radius": 100.0
  }
}
    Language file
    Code:
    {
