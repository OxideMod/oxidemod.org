    You need to access your server files through FTP.Then go to oxide>experimental>server>"server name">plugins - this is where you put downloaded plugins from this website.You also have another directory next to plugins, called configs.That is where .json or config files will show up once you load the plugins.You can copy these files outside of your server directory on desktop or somewhere, edit it, then go back to server directory, delete the old .json and paste new one.That is for changing the configuration of plugins. DO NOT edit any .cs files in >plugins directory.

    As for permission system, I`ll be very short.Open console by F1 in game, type: ownerid <yoursteam64id> <yoursteamname>
    You may also need to do this: usergroup add <yoursteamname> admin (this is if you are adding someone or yourself to admin group, which has max auth level automatically)
    You will need to do this then: server.writecfg

    Then you apply permissions that you need, which are listed on the overview of this plugin or any other.Basically:

    grant [user/group] [default/admin] [permission name]

    So lets say you want to give your admins an ability to spawn items through console thing:

    grant group admin removeaaa.giveid

    I hope this gives you an idea.
