  2. Spicy

    Out of interest, is this for a hardcore-style server?

    In addition, this is how you could go about doing this:

    Detect when a user dies with a hook
    Get the user SteamID
    Assign them a permission such as "banned"
    Set a 1 hour timer
    Kick them
    When a player joins, reject the connection if they have this permission
    Remove the permission after the timer has ended
