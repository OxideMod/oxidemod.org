    Everything looks fine according to that, it just looks like you don't have any plugins installed where your oxide/plugins folder is. The default location is oxide/plugins from your server install location, but that can be changed with +oxide.directory (host may have this set.) If the plugins are in the right location and Oxide is installed, you'd see at least some indication that they are trying to be loaded.
