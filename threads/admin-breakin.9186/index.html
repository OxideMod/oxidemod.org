    Admin Breakin - Easy get into a players base as admin.

    As an admin you need to check someone's base sometime, but you could not get in without turning off security or teleporting bugging you in. So i decided to make a Admin Breaking plugin that helps you as admin to get into a base.

    Usage:
    Code:
    /breakin x [numberofblocks]
/breakin x -[numberofblocks]/breakin y [numberofblocks]
/breakin y -[numberofblocks]/breakin z [numberofblocks]
/breakin z -[numberofblocks]/breakin x -10
