    Bobakanoosh submitted a new resource:

    DiscordReport - Send reports from players in-game to a Discord server

    DiscordReport is a plugin that uses the Discord API to allow players to use an in-game command to report a player. It then sends information regarding the report to a given Discord Server.

    Required:
    • Discord.cs
    • DiscordReport.cs
    • Any Discord bot
    • A server or your computer to host the discord bot on (not resource-intensive at all)
    I've made a very simply Discord bot if you wish to use that. The GitHub for the bot...
