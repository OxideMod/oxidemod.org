    There are times where you will need to delete the old config file first. Not always. It is usually written in update notes. The dev will say to delete the old config..

    Otherwise, try keep an eye on each mods thread, for bugs and issues. Sometimes updating right away is not good. Old versions have less bugs a lot of the time. So don't always be keen to update right away.. The update could easily bug out..

    As of now. There are a few mods that will break of you update..
