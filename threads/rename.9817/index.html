    Wulf submitted a new resource:

    Name Change - Instantly rename any player and keep it that way

    Name Change allows players with permission to instantly rename other players, and keep them with that name. Names are persistent and stored by each player's Steam ID.

    Permissions
    This plugin uses Oxide's permission system. To assign a permission, use oxide.grant user <username|steamid> <permission>. To remove a permission, use oxide.revoke user <username|steamid> <permission>.
