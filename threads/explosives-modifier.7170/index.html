  1. Mughisi

    Mughisi submitted a new resource:

    Timed Explosives Modifier - Allows you to modify the damage a timed explosive deals.

    This plugin adds an in-game admin command (only for players set with ownerid) to modify the damage percentage of the timed explosives:
    /explosivedamage <value>

    Examples:
    Disable timed explosives damage:
    /explosivedamage 0

    Deal 50% damage with explosives:
    /explosivedamage 50
