  1. LaserHydra

    LaserHydra submitted a new resource:

    Stake Authorizer - Authorize yourself for every stake

    Stake Authorizer allows you to authorize yourself for every stake or for all stakes in a radius.

    Permissions:
    • stakeauthorizer.use for using the plugin

    Chat Commands:
    • /authorizeall <radius> Authorize for all stakes in a radius from you
    • /authorizeall Authorize for all stakes on the server

    Language file:
    Code:
    {
  "No Permission": "You don't have permission to use this command.",...
