  1. Colon Blow

    Colon Blow submitted a new resource:

    FauxAdmin - Allows Authorized players to be a Fake Admin, allowing admin noclip and Green chat name

    Gives Players the IsAdmin flag when waking up which allows players to only:

    1. Use Admin noclip function
    2. Have a green name in chat
    3. Plugins that use the IsAdmin authentication check will allow that function.

    Thru a lot of testing. As the IsAdmin flag is applied after they login, players do not get any access to server side admin functions. Players cannot spawn, give, ban, mute, see any server info with the find. Plus all other admin commands within rust beside the few...
