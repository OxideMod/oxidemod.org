    I have been using Fly for a long time. If you get stuck, cannot move, and can only look around, the quickest solution I found was to simply remove the Steam ID of the stuck Admin in the /persistence/player directory on your server. Restart the server and good to go. If you use Inventory guardian, it will restore whatever the Admin had on them at the time.

    Hope this helps
