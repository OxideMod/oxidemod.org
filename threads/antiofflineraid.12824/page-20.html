  17. NiueanTurtle

    Not sure if this will help anyone update the plugin, but changing the variable for:
    "afkMinutes": 10,
    Will switch the AntiOfflineRaid on after x minutes.

    So I've set the time on my server to 120mins:
    "afkMinutes": 120,
    so the protection stays OFF while the player is online (at least for 120mins). I know this is not its original function, but it seems to work. The plugin seems to think the player is AFK. and will protect the base after the x mins (even when the player is still online).
