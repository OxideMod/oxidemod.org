    you must set the permission per console
    examples:

    oxide.grant group player antiofflineraid.protect u can use the follow groups to set up admin, moderator or player this is for use the AOR

    oxide.grant group moderator antiofflineraid.check this allows the Group to use /ao playername

    you can change group to user then you can set an specified player to use AOR or the command /ao playername
