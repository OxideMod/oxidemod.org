  4. isuelt

    This plugin would allow an Admin to:

    1. Drop all current permissions and roles (owner, admin, moderator) and pickup the server permissions for standard players.
    2. Additionally, it would change the in-game player name of the admin to a desired name.
    3. Of course the admin in player mode can switch back to admin mode and regain all permissions and his/her admin name
    4. Reserves the Admins name when in player mode


    For the sake of transparency

    1. Notify server community in chat whenever a transition is made
    2. Log all transitions between (admin -> player mode) and ( player -> admin mode)
    3. Give ALL users the ability to see the log of transitions in the console screen using a chat command
