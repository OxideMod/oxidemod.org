  4. xXdarrkXx

    Trux try this .. but back up your current world.

    1. You will need to download the map to your computer.
    2. Then upload it via FTP or File manager to the root of your service
    3. Go to 'Command line Manager' -> 'Custom Command Lines'
    4. Create a new Command line
    5. Set the world name (exactly what you just uploaded - except don't put the .hwb on the end
    6. Save the command line and click 'select' on the right
    7. Restart the server

    hope that helps
