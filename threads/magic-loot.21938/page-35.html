  16. Kunazai

    Thank you to all who have been answering questions about the plugin, I very much appreciate the community support.

    Those of you who have issues with absurd item counts, the config option VannilaLootTablesOnly being set to false will use an extra loot file in the data folder of oxide. This loot file is generated based off your stack sizes on the server. I plan to deprecate this feature in my next major update as it usually just confuses users.

    MagicLoot update
    I am starting work on a complete overhaul to the plugin, adding several requested features, plus more. The goal of this overhaul is to make things easy to use and set up, and more transparency with what the plugin is doing to your loot.
     
    Kunazai, May 15, 2018
