    if you have given an oxide permission and you want to remove it you use the revoke command: revoke <group|user> <name|id> *|<permission>
    Example: revoke user Mughisi inventoryviewer.allowed

    If you are talking about Rust's ownerid or moderatorid I think you use the removeid command but not 100% sure about that one

  6. Mughisi

    Here are all the available commands regarding permissions:
    grant <group|user> <name|id> *|<permission>
    revoke <group|user> <name|id> *|<permission>
    group <add|set> <name> [title] [rank]
    group <remove> <name>
    group <parent> <name> <parentname>
    usergroup <add|remove> <username> <groupname>
    show <group|user|perm> <name>
    show <groups|perms>
     
    Mughisi, Sep 12, 2016
