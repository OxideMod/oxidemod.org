  6. Yatta

    Ah, it seems you're not linking the child element to its parent. You're adding both to container, via "Container.Add", but with no relation to each other.

    One use is :
    Code:
    container.Add(element, strNameOfParent, strNameOfThisElement)
    Afaik, the line Name = CuiHelper.GetGuid() gives an automatic string name to your element, but I prefer to manually set my elements names. This could cause conflict with other plugins, but as long as you use explicit names such as "myModName_background" you should have no problem.

    So, to add and name your 'home' element to the container, you do :
    Code:
    container.Add(myBackgroundElement, "Overlay", "myUIBackgroundName");
    ("Overlay" being the base UI element afaik)

    Then to a add a child to it :
    Code:
    container.Add(myChildElement, "myUIBackgroundName", "myChildElementName");
    You should have your child element in your home element now, with its position accordingly relative.

    (edit : improved anwser)
