  1. Steenamaroo

    Hi,
    Items that you want your players to be able to loot should be stored in the main container of the inventory.
    This allows you to use belt wipe/clothes wipe to prevent players getting the bot's high tier gear, if that's what you want.

    If that's what you've done, empty your inventory and redeem the kit to make sure it is as you expect.
    Often people forget to do the /kit items command when creating with Kits Plugin, resulting in an empty kit.

    If you were expecting your players to be able to loot the bots personal items (belt+clothes), make sure the wipe-on-death options for those contains are not enabled.

    If you're using 'keep default loadout' clothing and belt items in your kit, if there are any, will replace the bots default attire.
    If these containers are empty, the bot will have default weapons and attire which are never lootable.
     
    Steenamaroo, Apr 21, 2018
