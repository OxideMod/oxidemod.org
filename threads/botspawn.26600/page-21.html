  13. Steenamaroo

    Steenamaroo updated BotSpawn with a new update entry:

    1.1.1

    BotSpawn v1.1.1

    Custom Spawn points with chat commands.

    '/botspawn add customname'
    '/botspawn remove customname'
    '/botspawn list' (shows all custom location names)

    **Read**
    Custom spawn points are saved in /data/BotSpawn.json
    To edit the parameters you must unload Botspawn.cs first!

    Performance Workaround
    Option 'Ai_Falling_Disable' will disable bot AI when parachuting.
    This may not be ideal, but it helps performance immensely, if you need that.
