    Suicide and re-spawn are never active together. They're alternatives for different circumstances.

    Suicide timer is to prevent the long term build up of non-respawning bots which haven't been killed.
    That's bots from airdrop calls, toplayer chat command calls, or external hook call.

    I think, at some point, I should probably make the monument names match the map.
    At present they match up with the prefab names, which is why you see Warehouse instead of Mining Outpost, for example.
