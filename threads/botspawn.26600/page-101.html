    o.reload BotSpawn
    That will recreate it.
    Then open both up in a text editor and merge back in your changes.
    Then rerun the reload to pick up the changes.

  11. mspeedie

    the setting changed, you can still make supermen, check you have all the options correct in their definition, including the new settings:
    Code:
          "Aggression_Range": 200,
      "Bot_Accuracy": 2,
      "Bot_Damage": 0.9,
      "Bot_Firing_Range": 250,
      "BotHealth": 400,
    That is a decently tough bot (IMO). Make sure you set all those.

    And don't forget to set all the options, there are a bunch of new ones:
    Code:
      "Options": {
    "Allow_Rust_Loot": true,
    "Animal_Safe": true,
    "APC_Safe": true,
    "Cull_Default_Population": true,
    "Ignore_Animals": false,
    "Ignore_HumanNPC": true,
    "NPC_Retaliate": true,
