  11. Steenamaroo

    You need to create a kit using Kits for Rust | Oxide

    This allows the Bot to have clothing and a weapon.

    You'll want to do
    '/kit add madeupname' - to create a blank kit with that name
    '/kit authlevel 2' - so players can't redeem it
    '/kit items' To copy your current inventory to the kit

    Then you can put madeupname into the config of this plugin.
    The bots will now spawn with the inventory you had at the time of creation.

    Remember, your players can loot the corpses so cheap clothing is recommended.
    Primary weapon drop can be disabled in config.
