  11. Steenamaroo

    Mark Wahlberg said: ↑
    - nav_grid applied in console
    - nav_disable applied in console
    Click to expand...
    You need to check these settings took after a restart.
    Restart the server and type "nav_grid" into console - It should return true.
    then type "nav_disable" - It should return false.

    If either is wrong, your setting isn't taking and you should add the setting to the server startup file and restart, rather than adding by typed command.
    If that fails, your host may be overriding the value.
