    try this as example:

    "VIPHomesLimits": {
    "nteleportation.vip1": 5,
    "nteleportation.vip2": 10,
    },

    "VIPCooldowns": {
    "nteleportation.vip1": 5,
    "nteleportation.vip2": 10,
    },
    "VIPCountdowns": {
    "nteleportation.vip1": 5,
    "nteleportation.vip2": 10,
    },

    just add permission to the specific groups and edit the line what u want home cooldowns and countdown
