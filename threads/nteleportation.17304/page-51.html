  6. MongerVog

    In the config file change the following to true.
    Code:
    "TPR": {
    "AllowCraft": false,
    "BlockTPAOnCeiling": true,
    "Cooldown": 600,
    "Countdown": 15,
    "CupOwnerAllowOnBuildingBlocked": true,
    "DailyLimit": 5,
    "Pay": 0,
    "RequestDuration": 30,
    "UsableIntoBuildingBlocked": true,
    "UsableOutOfBuildingBlocked": true,
    "VIPCooldowns": {
      "nteleportation.vip": 5
    },

    Default is false so update it to true like I have above.
    Code:
        "UsableIntoBuildingBlocked": false,
    "UsableOutOfBuildingBlocked": false,
