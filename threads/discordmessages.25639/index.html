    Slut submitted a new resource:

    DiscordMessages - Report and Ban plugin with messages sent straight to discord!

    Introduction

    This plugin is for those who are unable to setup real discord bots or do not have the hardware for it. DiscordMessages makes use of the webhook function of Discord therefore you do not need any dependencies to get up and running!

    DiscordMessages includes it's own ban function, it not only sends the message to discord, it has a better functioning ban feature.


    Commands

    () = Required - <> = Optional
    • /report (player) (message) - This will send a report...
