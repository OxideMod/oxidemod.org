    RD156 submitted a new resource:

    Shop - Shop for buy and sell

    This plugin is a admin shop, player can buy and sell.
    Just a admin can chose where item is in sell and where is in buy.
    Just a admin can modify price of item.
    NPC can just sell or just buy one item or buy and sell one item.

    Specificity:
    Player can sell just the first line in her inventory. (not a navigation bar)
    Price is a Integer.

    Player command:
    /shop view (for view all item on shop)
    /shop view [item name or ID item] (for view one item)
    /shop p [number] (for view page [number] on shop)...
