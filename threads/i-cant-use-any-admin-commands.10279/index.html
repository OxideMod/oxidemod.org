  2. Wulf Community Admin

    Oxide doesn't control your ownerid status. See http://oxidemod.org/threads/admin-and-moderators-in-rust-experimental.6520/.

    Make sure you've saved your owners/moderators by using server.writecfg after you're done. Also make sure you've logged out and back in.
