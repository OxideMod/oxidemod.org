    Zone Command - Executes commands when player is entering a zone

    This plugin executes server commands when a player is entering a zone that was set via ZoneManager plugin.

    Commands:
    /zcmd add <zoneID> <command>     — Adds one command to a specific zone
    /zcmd remove <zoneID> — Removes zone with command
    /zcmd list — Shows list of all zones and their commands

    You are can also use short commands: add = a, remove = r, list = l.
