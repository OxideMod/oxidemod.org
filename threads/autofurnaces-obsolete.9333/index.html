  1. AnExiledGod

    AnExiledGod submitted a new resource:

    Auto Furnaces - Automatically start all furnaces after a server restart.

    All this plugin does is when the server is started, all furnaces will be automatically turned on and begin cooking. This is done to fix the issue where when an admin restarts the server, all furnaces have to be re-started. This was made because I wanted it for my own server and the user CHR requested it. http://oxidemod.org/threads/auto-furnances.8355/

    Feedback
    Any feedback is welcome, whether it be ideas and suggestions, bug reports, or just to tell me I coded it wrong....

    Added Check to see if furnace has cookable

    I've added a check that looks for bearmeat, metal ore, raw chicken, raw human mean, raw wolfmeat and sulfur ore in the furnace before starting it. If not found, the furnace will not be started as it doesn't have anything cookable in it.

    My players were complaining about me turning their furnaces with nothing but wood in it, on. So this is the solution haha.
