  17. FR-DarkRod

    Code:
    @echo off
@title Rust Safe Mode

:start
cd steam
steamcmd.exe +login anonymous +force_install_dir ../rustds +app_update 258550 validate +quit
cd ..

xcopy E:\rust-serveur\RustDedicated_Data E:\rust-serveur\rustds\RustDedicated_Data /Y /E /I

cd rustds
RustDedicated.exe -batchmode (and params ...)
cd ..

goto start
    It's work

    Reboot + Autoupdate <3
