  2. Wulf Community Admin

    Steam Group is an Oxide 1.18 plugin, not compatible with Oxide 2.0.

    The Chatmute plugins uses Oxide's permission system, not the Rust owner/moderator auth levels. See the Overview of the plugin for the permissions, and http://oxidemod.org/threads/using-oxides-permission-system.8296/.

    You can only execute console commands through RCON, not chat commands.
