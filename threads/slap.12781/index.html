    Slap - Allows players with permission to slap other players around

    Slap allows players with permission to slap other players around a bit. Also features random flinching and suitable sound effects for a premium slap experience.

    Permissions
    This plugin uses Oxide's permission system. To assign a permission, use grant user <username|steamid> <permission>. To remove a permission, use revoke user <username|steamid> <permission>.
