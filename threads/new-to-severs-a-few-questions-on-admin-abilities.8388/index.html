  3. BasKoning

    tip for (right)SHIFT L: try to fly below the ground to not upset your players

    Raid cam: (right)SHIFT P
    Note: Flying in raid-cam mode to the other side of the map does not work as expected: you will only see the mountains and the trees, no barrels, ore, houses or players. You can only see in raid cam what your character can see from where he is standing.

    There is also 'spectate' mode:
    F1 spectate F1 (you will die)
    (space to swap players)
    F1 respawn F1 to respawn alive again.

    ---

    If you cant use the ingame item list to give yourself items, you are not owner nor moderator.
    In the server console: type
    ownerid STEAM16NR "Optional Name" "Optional Reason"
    or
    moderatorid STEAM16NR "Optional Name" "Optional Reason"
    Then log in again with the client, your server console should say:
    Auth level 2 (for owner) or Auth level 1 (moderator) when you log in.
    If that is not the case, try to quit the server, start again, then log in. But that should not be necessary.

    ---

    To build buildings for events, just give yourself resources and build them as an admin.
    Or check the plugins section: I think there is one that can 'save' and 'load' buildings. http://oxidemod.org/plugins/copy-paste.716/
