  2. Henrikmeister

    When Rust updates it often comes with tweaks to the algorithm that generate the procedural map (ex. this time they tweaked it to include the bandit camp and swamp biome). When the algorithm changes the output of the seed changes too.

  4. Henrikmeister

    I might add that if you would like to keep the same map you can extract the .map-file -> Add it to dropbox -> use -levelurl="YourLinkToTheMapFile" as a startup argument. This way you will always get the same map but notice that you will not get any new biomes, compounds or radtowns to the map when they are released to the game.
    Code:
    -levelurl="YourLinkToTheMapFile"
