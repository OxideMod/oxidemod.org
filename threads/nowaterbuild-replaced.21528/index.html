    NoWaterBuild - Block players building in water

    NoWaterBuild will automatically destroy building blocks (When placed) that are in water, You can change if you want to block in the shallows or block in very deep water. Read FAQ to understand this a bit more.

    Permissions
    This plugin uses Oxide's permission system. To assign a permission, use grant user <username|steamid> <permission>. To remove a permission, use revoke user <username|steamid> <permission>.

    You need to change EnityDepth in your config. Example from FAQ:
    Q): What is Entity Depth in Config?
    A): This is to allow building in shallow waters or on beaches;
    Example:
    0.1 = 10% of the block is in water. Anything Higher than 10% and it'll be destroyed.
    1 = 100% of the block is in water (Under water).
    0.5 = The block is 50% above the water and 50% below the sea level.
