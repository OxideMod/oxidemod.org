    logically you would just set tree density to "0" and be done BUT

    if you do that you create a serverwide bug, causing trees not to be displayed correctly and when displayed they are almost unchoppable, or disappear on chopping.

    To fix this, delete all the "DeterminedPoints" coordinates and set the tree density to 0.00000000001 (or something similar small)
    looks like this in the end:

    Code:
    Spawning.CrownLand.Tree4.Density                        = '1E-06'   # 0.0075
Spawning.CrownLand.Tree4.DeterminedPoints               {   # (-183.9213,195.7811,-1041.38);(-214.6017,239.4198,-1111.36);(-200.4,245.0205,-1119.29);(-365.8259,179.276,-1179.991);(-230.6135,255.2666,-1140.201);(-347.3234,167.2904,-1144.922);(-286.9247,235.7691,-1093.446);(-190.1967,190.3629,-1015.386);(-356.454,165.8171,-1213.769);(-301.7709,248.7005,-1148.83);(-289.676,242.977,-1121.676);(-245.2613,251.3778,-1116.321);(-226.8348,252.401,-1129.155);(-202.8476,224.879,-1092.667);(-257.216,244.7658,-1108.851)
}
Spawning.CrownLand.Tree5.Density                        = '1E-06'   # 0.0075
Spawning.CrownLand.Tree5.DeterminedPoints               {   # (-217.6218,246.4322,-1117.828);(-356.1497,172.2966,-1158.413);(-287.4132,256.356,-1147.541);(-183.2655,185.8758,-996.7165);(-271.657,236.7741,-1111.756);(-251.8578,215.8387,-1090.434);(-294.4022,247.6516,-1135.007);(-199.8987,249.8414,-1134.008);(-236.1255,245.5311,-1113.63);(-294.3929,238.3591,-1107.549);(-192.0009,235.6692,-1103.041);(-199.8242,201.4255,-1065.43);(-344.6197,164.1824,-1217.535)
}
    (the value 1E-06 is a server created value, after the server restart)


    when you do this, all self grown trees will appear corretly, you can chop them just normal, and no randomly spawning trees will ever appear again !
    Of course your island will look pretty empty then, but forestry is a job you know
