  1. Dubz

    Dubz submitted a new resource:

    Day/Night Gather - Set different gather rates for day and night.

    A basic plugin that changes gather rates for day or night.

    Code was modified from k1lly0u's NightLantern plugin.

    Config File Explained:
    Code:
    {
  "DayGatherRateDispenser": 1.0, <--- Day Rate for Tree/Node/Animals
  "DayGatherRatePickup": 1.0, <--- Day rate for Picked up items Cloth/Stone/Wood ect.
  "DayGatherRateQuarry": 1.0, <--- Day rate for Quarry
  "DayMessage": "Day: {GatherRate}x Gather Rate.",...
