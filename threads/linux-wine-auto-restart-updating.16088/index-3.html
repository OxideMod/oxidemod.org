  2. sqroot

    start.bat
    Code:
    @echo off
:loop
start /wait RustDedicated.exe <params>
goto loop
    update.bat
    Code:
    @<steamcmd path> +login anonymous +force_install_dir <server dir path> +app_update 258550 validate +quit
    update_oxide.bat
    Code:
    <wget path> "https://github.com/OxideMod/Snapshots/raw/master/Oxide-Rust.zip" --no-check-certificate
<7zip path> x "Oxide-Rust.zip" -aoa
del "Oxide-Rust.zip"
    update_all.bat
    Code:
    call update.bat
call update_oxide.bat
pause
    Or the equivalents for bash on Linux.
    Replace <variables> with your desired values and create these batch files in the server directory.
