    VendorRotateProtection - Prevents anyone except the owner from rotating a vending machine.

    Overview
    Prevents anyone except the entity owner from being able to rotate a vending machine, even with building privileges.

    Details
    Users will receive the following message upon unsuccessful rotate attempts: "You may only rotate vending machines that you placed."
