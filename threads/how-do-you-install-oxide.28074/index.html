  3. Wulf Community Admin

    Make sure you copy the exact structure from the zip file, don't move files around, which is what looks like what happened in your screenshot.

    Download Oxide zip > stop server > extract Oxide zip > copy/move and overwrite all > done.

    Keep in mind that running SteamCMD with validate will remove the modded game files and Oxide will not start. Only run SteamCMD when intending to update the game re-install Oxide.
     
    Wulf, Nov 10, 2017
