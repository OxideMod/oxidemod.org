    BradleyControl - A plugin that controls Bradley properties.

    BradleyControl is a plugin that allows customization to the Bradley APC, or the complete disabling of there spawns. This plugin also adds in a simple chat command for admins to respawn the Bradley (/bradley respawn), and for now sorta disables the Bradleys attacks on vanished players.

    This plugin is very much WIP and will have more customization in the future, for now there are basic properties to change.
