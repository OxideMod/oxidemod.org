    GunGame - GunGame event arena game

    The rules of GunGame are simple. Each kill you get will advance you 1 rank, each rank comes with a new weapon. The weapons get harder the higher you go. The first person to get a kill on rank 15 wins. Kills with a machete will lower the enemy's rank. Reward points are issued to the winner (see event manager reward system)

    Setup for GunGame
    1. Create your arena
    2. Create a Zone around your Arena (Zone Manager)
    3. Create a spawn file (Spawns)
    4. Start a game (Event Manager)...
