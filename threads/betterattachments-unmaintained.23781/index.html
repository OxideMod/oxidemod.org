    BetterAttachments - Allow control of attachment attributes

    BetterAttachments is a plugin to override default attachment attributes and customize penalties/bonuses for each attachment.

    Configuration:
    Configuration elements are derived from the built-in Rust attachment attributes, and stored in a map based on the attachment short prefab name:
    flashlight.entity, pistol_flashlight.entity, holosight.entity, lasersight.entity, muzzleboost.entity, muzzlebrake.entity, silencer.entity,...
