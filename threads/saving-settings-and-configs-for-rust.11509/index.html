  2. Wulf Community Admin

    Oxide is a mod, it needs to be installed AFTER every update you do. For this reason, you should only update when an update is available for Rust or if there are issues with the current version of Oxide you are using.

    For the settings, you need to use "server.writecfg" after making any changes via Rust commands.
