    How to get the list yourself using code
    I couldn't really find a better solution so I just went with this, it's ugly but it does the job.
    Code:
    [ChatCommand("items")]
        void ItemsCommand(PlayerIdentity identity, NetworkMessageInfo info, string command, string[] args)
        {
            for(int i = 0; i < 300; i++)
            {
                try
                {
                    var item = GlobalItemManager.Instance.GetItem(i);
                    Puts(i + " : " + item.GetNameKey());
                } catch {
                  
                }
            }
        }
    This will check for an item id from 0 - 300. Like I said it's not great, might find something better later on. There are only items in the game up to 271 so far.
    To grab the list just check your oxide logs, grab the output and then search and replace to remove anything before the ID.
