    Wulf submitted a new resource:

    Country Block - Allows or blocks players from specific countries via a whitelist or blacklist of country codes


    Country Block is a plugin that either allows or blocks players from specific countries via a whitelist or blacklist of country codes. This is useful if you want to keep your playerbase somewhat localized.

    Configuration
    You can configure the chat name, messages, and other settings in the countryblock.json file under the oxide/config directory.
