  8. [X] Meep

    > oxide.group
    Code:
    Syntax: oxide.group <add|set> <name> [title] [rank]
Syntax: oxide.group <remove|show> <name>
Syntax: oxide.group <parent> <name> <parentName>
    Use this only if they are going to be able to do something on the server that other players cannot. Like use plugins for crosshairs or something.
    >oxide.grant
    Code:
    Syntax: oxide.grant <group|user> <name|id> <permission>
    > oxide.usergroup
    Code:
    Syntax: oxide.usergroup <add|remove> <username> <groupname>
    An example of Better Chat name config with donator status

    Code:
    "Donator":{
   "NameColor":"red",
   "Permission":"color_vip",
   "Prefix":"[Donator]",
   "PrefixColor":"blue",
   "TextColor":"red",
   "Rank":5
}
    I cant make it easier than this for you. If you simply follow Hydras instructions on the addon you would have it working for yourself in minutes.
