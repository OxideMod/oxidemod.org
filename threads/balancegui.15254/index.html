    lethal_d0se submitted a new resource:

    Economics GUI Balance - Displays the player balance on the GUI

    This plugin is designed to work with Economics, it shows the survivor their current balance in the lower-left corner of the interface.

    How does it work?
    The server checks for balance changes, and when a balance differs from that of the previous, ie a transaction has been made, the interface is updated.

    How do I install this?
    Place the script in your Oxide plugins directory, then restart your server for it to take effect. There is currently no config file to alter the...
