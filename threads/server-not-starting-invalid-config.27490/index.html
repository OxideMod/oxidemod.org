    For your server listing, all Oxide does is add the "modded" and "oxide" tags which place it under the Modded tab in Rust, unless you have it set to not do that under the oxide.config.json. Oxide doesn't control if/where your server is listed though. If your server isn't actually starting, you'd need to check your logs for errors, but I'd suggest trying to direct connect to it first.
