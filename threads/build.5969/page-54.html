    You need to acquire a current prefab list. A user named "I Am Groot" posted one a few posts back. You then look on the prefab list for the numbered item you want to place. In this case you want the recycler, which is #105. So what you do is type /plant 105 and you will see the build tool open up on top of your screen. Generally, the front of the item will be away from you, so it's as if you are standing behind the item, keep that in mind when you're placing it. Also, remember that you have to be careful with this plugin. Make sure you consciously click your right mouse button one time and do not hold the mouse button down, if you hold the mouse button down it will place multiple instances of the item and you will have to remove them all. It can be a pain. Depending on what it is you're placing, multiple instances can cripple your server.

    What I find to be really helpful is to use a crosshair plugin to better assist with alignment since you don't have an image to use that will show you exactly where it will be. You might want to practice somewhere else on the map to help you get familiar with how items are placed.

    What if you place it incorrectly? I always use the /erase command as a last resort. It will literally remove anything in the line of sight, potentially things you don't want removed. You can even remove yourself if you click on your foot. Try using the R-Remove tool and type /remove admin. It is much safer if you're working in a confined area with buildings or other deployables.

    Hope this helps. This is an awesome tool and it has been invaluable in the customization of my server, but it can give you some headaches if you aren't familiar with certain nuances that come along with it.
