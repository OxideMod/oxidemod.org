  15. Werkrat

    If you hold right-click with the plugin still active, it'll mass spawn. Just right-click once to spawn 1 animal. If it's still mass spawning, edit this line in the config (I changed mine to 5 seconds, that's what I'm showing you):

    "Pressed time before multiple spawns (seconds)": 5.0

    Edit the delay, save, then: oxide.reload Build

    Now, for example, type in chat: /animal bear
    right-click to spawn a bear
    type "/animal" in chat to disable the plugin from spawning more
     
    Werkrat, Mar 6, 2016
