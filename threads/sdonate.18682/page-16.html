  4. nimdasys

    I checked my apache logs on the server hosting my SDonate site and saw that none of the requests were coming through from my other machine. The issue was my new servers hostname is exactly the same as the domain name its making requests to. So everytime it would send a request it was resolving to localhost.. lol oops. I fixed it by changing the URL in the .json file to http://ipaddress/donate instead of the domain name.
