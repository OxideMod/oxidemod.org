  18. Whispers 88

    in Sdonate when you make a package you can add a console command. To add a user ID to a group use the following "oxide.usergroup add {{VAR=STREAMID}} groupname" so obviously all you have to do is change the group name to the oxide group you want to add them too. Then just adjust t permissions you want that specific group to have using RCON.
