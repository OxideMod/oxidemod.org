  1. Reneb

    Reneb submitted a new resource:

    Building Owners - Set Buildings Owners

    This plugin will automatically set owners of houses when someone builds a house part.

    You have two options:
    1 = will set one owner for the entire buildings.
    2 = will set one owner per building parts.

    Commands:
    - /changeowner PLAYER => will change the owner of the building where you are.
