  5. uHostPro

    Create a file in the root directory called "autoexec.cfg" there should be an example one already called "autoexec_default.cfg", inside the file per line write:
    addadmin <Steam64ID>
    addadmin <Steam64ID>
    addadmin <Steam64ID>

    and so on, for as many admins as you'd like, to have the server execute the command on launch, alternatively if you've made yourself an admin already, open the F1 console and write the same command addadmin <Steam64ID> to make that user admin.

    To find your Steam64ID or that of your friends, go to this site and type in their Steam Name; Steam ID Finder / Simple ID Converter
