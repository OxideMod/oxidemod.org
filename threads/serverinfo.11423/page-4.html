  19. baton256

    H3XServers said: ↑
    I'm not sure how to use the permission, I've set 3 different tabs to the permissions 'owner' 'admin' 'moderator' but the tabs don't show at all
    Click to expand...
    Afaik oxide creates 3 default groups: admin, moderator, player. No owner group. If you set yourself as owner it doesn't add you to any of these groups.
    The plugin uses oxide groups, not rust authentication levels. Check your files: oxide/data/oxide.groups.json and oxide/data/oxide.users.json

    Also this article may be helpful: Using Oxide's permission system | Oxide
