  20. adhdork

    adhdork said: ↑
    Having an issue where changes made to the config file aren't reflected in-game. Editing ServerInfo.json - is there a different config?
    Click to expand...
    Solution: I needed to edit server_info_text.json instead of ServerInfo.json. I deleted server_info_text.json before because the description said that ServerInfo.json was the updated config file, so I recreated server_info_text.json and pasted my updates there.
