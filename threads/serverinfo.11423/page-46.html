    Here's what worked for me:

    > Copy the text in ServerInfo.json and make the changes you'd like.
    > Paste into your text editor and save as "server_info_text_.json" (If you're using Notepad, make sure the extension is .json..not .txt)
    > Move that file to your .../config folder
    > Delete ServerInfo.json
    > Reload ServerInfo

    It should recreate ServerInfo.json with the text from server_info_text.json

    When you need to make a change:
    > Make the change(s) in the server_info_text.json located in your .../config folder and save
    > Delete ServerInfo.json in .../config folder
    > Reload ServerInfo
