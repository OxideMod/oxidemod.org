    baton256 submitted a new resource:

    ServerInfo - UI customizable server info with multiple tabs

    This plugin allows you to create customizable UI server information with multiple tabs.
    /info - only chat command plugin provides - it shows info.
    You may configure plugin to show info on server join.

    Configuration is pretty straightforward: Tabs, text, show on join.

    Example config:
    Code:
    {
  "Tabs": [
    {
      "Name": "Info",
      "TextLines": [
        "type <color=red>/info</color> to show this GUI",
         "You can remove tabs by deleting tab from config"
      ]
    },...
