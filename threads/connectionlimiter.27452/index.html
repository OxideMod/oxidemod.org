    ConnectionLimiter - Help server admins block spam re-connecting.

    ConnectionLimiter help server admins block spam re-connecting. When player connect/disconnect, he will be added to Dictionary<> list. If player reconnect in (default: 30sec) seconds, he will be kicked. Kick message can be modified from oxide/lang/en/ConnectionLimiter.json and reconnect cooldown time oxide/config/ConnectionLimiter.json
