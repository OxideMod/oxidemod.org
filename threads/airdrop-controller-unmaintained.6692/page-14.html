    The entire plugin IS working. Make sure your Oxide is updated to the latest version, along with Rust.
    If you're having an issue with the ingame console, try using Rusty as a remote console. It's what I use, and I have NO problems with my server. *
    * http://oxidemod.org/resources/rusty-server-rcon-administration-tool.53/
    Also, when calling Mass Airdrops, do not drop too many. It has the tendency to lag the server and players. Keep it around 5-10.
    If you want automatic Mass Airdrops, use Rusty rcon, and set a timed command to trigger once every X minutes. After making this timed command, execute it once, and it will follow them timer after that.
