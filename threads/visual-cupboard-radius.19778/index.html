  1. Colon Blow

    Colon Blow submitted a new resource:

    Visual Cupboard Radius - Allows Authorized Players to Visually See Building Privildge Radius of Tool Cupboards

    This plugin makes a complete sphere, ACTUAL cupboard radius falls off below cupboard.. FYI.

    When authorized player uses /showsphere chat command. Player will then see visual spheres for the building privilege radius on all nearby tool cupboards within range.

    Visual Spheres will only show for duration set in config file, then disappear.

    Optional - visual spheres can be turned on when placing a tool cupboard. this is OFF by default in config file....
