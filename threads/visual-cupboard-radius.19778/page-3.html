  16. Colon Blow

    mafiosilla said: ↑
    Sorry for the dumb question, but... how i can authorize me?
    Click to expand...
    You would use something like this in your server console:

    Code:
    oxide.grant user "mafiosilla" visualcupboard.allowed
    And this if you also want to be admin:

    Code:
    oxide.grant user "mafiosilla" visualcupboard.admin
     
    Colon Blow, Aug 27, 2016
