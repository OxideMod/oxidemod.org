  1. Calytic Community Admin Community Mod

    Calytic submitted a new resource:

    Bank - Safe player storage

    Gives players a safe box to bank their items.

    Permissions
    • bank.use
      Allows players to use their bank
      Ex. grant group default bank.use
    Commands
    • /bank
      Player command to view their bank
    • /viewbank PlayerName
      Admin command to view any bank
    Default Configuration
    Code:
    {
  "Settings": {
    "boxes": {
      "bank.big": "assets/prefabs/deployable/large wood storage/box.wooden.large.prefab",
      "bank.default":...
