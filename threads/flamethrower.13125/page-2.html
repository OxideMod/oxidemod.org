  9. Colon Blow

    ok, updated overview to help.

    Code:
    oxide.grant <group|user> <name|id> canuseflamethrower

Example: To allow the group player to use it. (all players)
oxide.grant group player canuseflamethrower

Example 2 : To just allow a person to use it:
oxide.grant user "colon blow" canuseflamethrower
     
    Colon Blow, Dec 20, 2015
