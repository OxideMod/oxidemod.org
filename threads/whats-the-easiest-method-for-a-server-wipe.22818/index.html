  2. Wulf Community Admin

    Why exactly are you copying files or wiping files at all? Any forced wipes happen automatically, but they are just the .sav files with Rust. All other Oxide-related files and such are never wiped unless you or your host manually removes them.

    If you're wanting to start fresh with the map, just delete the .sav file. If you want to wipe plugin data, delete the oxide/data folder. If you want to wipe sign data and other images from plugins, delete the storage folder.
