    SmallShelves - Allow players to place smaller shelves

    This plugin allows players to place smaller version of the Shelves item

    Permission:
    • smallshelves.use - Players with this permission will be able to use the command listed below
    Commands:
    • /smallshelves - This command will toggle between placement modes. When activated, placing a normal shelf item will instead place a small one
