  2. Wulf Community Admin

    The "oxide" folder is generated automatically after you have successfully installed and started your server with the modded Assembly-CSharp.dll that is included with Oxide. If that that and potentially other files were not installed correctly/fully, Oxide will not be started.

    If your server is starting, you can verify that Oxide is installed by using the oxide.version console command. If the command is unknown, then Oxide is not installed correctly.
