    1. Introduction

    Hello everyone!

    After a quite good start of my first project MyRustStats.com I would like to present you one of my newest projects: MyRustAdmin.com
    Due to the fact that I need to administrate several servers at once from time to time, I was looking for a tool that creates the possibility to administrate all of those servers at once. I didn't want to take the detour by "opening the tool 10 times and select a different server everytime". At the same time that tool should run on mobiles and tables, because I'm on the road quite often while I don't have access to a PC or laptop.

    Long story short: I didn't find a tool that fits my needs.

    Therefore I decided to create such a tool on my own which opens all the possibilities I mentioned above - and I would like to present it to you.

    Just in case you directly want to take a look at it, here are the links:
    Website:     MyRustAdmin.com - The most advanced Rust Admin Tool
    Tool: MyRustAdmin.com

    But of course I would like to invite you to read on.


    2. System Requirements

    Server:
    Just start your server with the parameter "+rcon.web 1". By adding this your server starts using the WebRCON protocol which makes it possible for my tool (and therefore your browser) to directly communicate with your server.

    Client:
    You only need a browser with active JavaScript. It doesn't depend if you are using your mobile, tablet or PC: This tool just runs on every device.

    Well, that's it: No plugins or binaries are required and of course: Everything is completely free!


    3. Features
    • Administrate your server right out of your browser
    • No proxy: Your browser directly communicates with the server
    • Cross-Device Ready: It works on your mobile, tablet and pc
    • No download required
    • UTF-8 support: You know those funny characters you see when someone writes special characters in to the chat? It belongs to the past
    • Multi-Server support: Monitor multiple servers at once (1)
    • Detailed server overview and statistics (1, 2, 3, 4, 5)
    • Direct access to your console (1)
    • Access the chat including several comfort functions like Kick, Ban and Mute (1)
    • Detailed player overview including playerstatistics and of course: comfort functions like Kick, Ban, Mute, Give etc. (1, 2, 3, 4)
    • Comfortable overview of all actual bans (1)

    4. Future

    Depending on the acceptance and the use by admins we will of course extend the features:
    Automated tasks, autokick, item packets for the give command are addable without real problems: Basically the sky's the limit.

    Furthermore I would kindly ask you to leave some feedback and wishes. That's more or less the only way to extend the functionality so the tool fits all of your needs.

    Last but not least we are already planning a release as a standalone desktop, andorid, windows phone and iOS app: In case the feedback is positive we are targeting a release on those platforms in the next months.


    5. Conclusion

    I think you see that the tool in its current state doesn't need to hide from other rust administration tools. We already got a huge functionality which we are going to extend continuously - but of course our main focus is the fact, that this tool runs in mostly all available browsers and therefore it's possible to use it without any limitations on different platforms.

    As I said already I would kindly ask you to leave some feedback. In case of questions or issues I'm available at anytime
