  6. Wulf Community Admin

    Wulf updated Natives with a new update entry:

    Added back exclusion for 127.0.0.1 for local

    If you are hosting the server on the same computer you are playing on, the Rust server will think your IP address is 127.0.0.1. Obviously since 127.0.0.1 will not resolve to a country, the checking has to be bypassed for you.
