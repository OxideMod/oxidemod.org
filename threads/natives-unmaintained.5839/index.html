    Wulf submitted a new resource:

    Natives - Allows only players from the server's country to join


    Natives is a basic plugin that allows only players from the server's country to join.

    Configuration
    You can configure the single message in the natives.json file under the oxide/config directory.

    Default Configuration
    Code:
    {
  "Messages": {
    "Rejected": "Sorry, this server allows only players from its native country!"
  }
}
