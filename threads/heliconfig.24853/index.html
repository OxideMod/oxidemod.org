  1. Kappasaurus

    Kappasaurus submitted a new resource:

    HeliConfig - Disable, enable, and control the amount of time between helicopter spawns.

    Features
    • Disable the helicopter (based on HeliBlock)
    • Control the time between helicopter spawns.
    Configuration
    Code:
    {
  "Helicopter enabled": true,
  "Time between spawns (hours)": 1
}
