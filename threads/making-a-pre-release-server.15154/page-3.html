  11. Colon Blow

    There is no prompt about the updates, just only as you said. the Date will change on the Server update tab like in your picture.

    The Oxide2, Server Update, and Rust:IO buttons will stay. Even after hitting install for each.

    What I do with the Gameserver server, is

    1. Stop server on my main info page. wait for confirmation. Then run Server Update.
    The console will then not allow you to access anything until is done. (hit refresh, this one takes a while sometimes)

    2. Once completed, Stop server again. Then run the Oxide Update. same as before, wait till you can access page.(hit refresh, usually within a minute)

    3. then stop server again. and a run Rust:IO. and wait for it to be able to access page. .(hit refresh, usually within a minute)

    You can now either stop server again. Until you a ready to start it. Or let it restart itself. It always restarts after you install any update.

    You can always repeat the process at any time, the buttons are always there. On occasion, I have had to update oxide and rust:io a time or two after a server update to get it to "like" them. lol

    And its possible to have more Server updates after the main one. It happens a lot with bigger updates. They could release fixes for Rust server in hours or next day. just whenever. Then you would need to install all three again each time. Just keep looking at the date.

    Rust client does not always show it needs a update even when the server might. and visa versa.

    And, as for will the map generate. Yes.. it can. If the update changes the map or if its one of the monthly "wipe" updates. then it will over write you map with a fresh one.
