  2. EnigmaticDragon

    You can use the weapon prefab name. The following code disables damage from rockets and C4:
    Code:
    bool? OnEntityTakeDamage(BaseCombatEntity entity, HitInfo info)
{
    switch (info.WeaponPrefab?.ShortPrefabName)
    {
        case "rocket_basic":
        case "explosive.timed.deployed":
            return true;
    }
    return null;
}
