    Whitelist - Restricts server access to whitelisted players and Steam group members

    Whitelist restricts access to your server, only allowing players who have the 'whitelist.allowed' permission from Oxide.

    Permissions
    This plugin uses Oxide's permission system. To assign a permission, use grant user <username|steamid> <permission>. To remove a permission, use revoke user <username|steamid> <permission>.
    • whitelist.allowed (allows player to connect to the server)
      Ex. grant user Wulf whitelist.allowed...

    You have a few options:
    1. Invite them to your Steam group and add that group name to the config.

    2. Users in the "admin" group are automatically excluded, this means "owners" from Rust. You do not need to grant the admin group the permission if you use the default "AdminExcluded" setting in the config.

    3. Grant the "whitelist.allowed" permission to any other group or individual user by Steam ID if you want them to have access.
