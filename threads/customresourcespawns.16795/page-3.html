  16. DarkyGoD

    1 - Group add admin
    2 - usergroup add USERNAME admin
    3 - grant group admin customresourcespawns.admin
    4 - Add Items : /crs add ID
    5 - Remove Items : /crs remove ( the one you're looking at )
    6 - wipe just seems to remove the first items then second etc.. won't wipe all.
    Usefull ID :
    1 = Corn
    2 = Hemp
    3 to 8 = Mushroom
    9 = Pumpkin
    56 = Metal Ore
    57 = Stone Ore
    58 = Sulfur Ore
