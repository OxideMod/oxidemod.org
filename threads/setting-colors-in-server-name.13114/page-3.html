  7. Needlez

    OK so first off all you need to be Admin {obviously}
    the steps
    Note when changing a server there is No restart required

    1)Go into your server in hit f1 '
    2)Type rcon servername <color=#00FFFF>PUT YOUR SERVER HERE</color>

    To change the colour copy the colour code into <color=HERE>Servername</color>
    example rcon servername <color=#00FFFF>Yolo x15 /Kits/TP/PvP/UBERLoot</color>

    Color code chart NOTE NOT ALL COLOUR CODES WORK
    Red #FF0000
    White #FFFFFF
    Cyan #00FFFF
    Silver #C0C0C0
    Blue #0000FF
    Gray or Grey #808080
    DarkBlue #0000A0
    Black #000000
    LightBlue #ADD8E6
    Orange #FFA500
    Purple #800080
    Brown #A52A2A
    Yellow #FFFF00
    Maroon #800000
    Lime #00FF00
    Green #008000
    Magenta #FF00FF
    Olive #808000

    For preset names copy these
    1)Blue
    2)Yellow
    3)Orange
    4)Lime
    rcon servername <color=#00FFFF>SERVER NAME HERE</color>
    rcon servername <color=#FFFF00>SERVER NAME HERE</color>
    rcon servername <color=#FFA500>SERVER NAME HERE</color>
    rcon servername <color=#008000>SERVER NAME HERE</color>

  11. Noviets

    Anubisu said: ↑
    My server name gets cut-off when I use too many color codes, how do i fix it? Here's the line. after the CL of "CLANS" the rest gets cut off in the server list.
    servername <color=#165ad1>World</color> <color=#fd1c15>Hurt</color> [2X,CLANS,TP,SHOP]
    Click to expand...
    It's a max character length limit. Even though the color codes are not displayed they are still used in the server name (which is why your colours work). Either reduce the length of the name, use use simple colours. For example. Instead of <color=#FF0000> you could use <color=red>.
