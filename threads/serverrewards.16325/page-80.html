  6. Guise Knightcott

    Hi, the Skin IDs in the Oxide API list are wrong for newer skins, they show workshop ID which is no longer how they are called in. The new Skin IDs are longer and bear no resemblance to the workshop number. So, if you know the newer ones, they'll still work (but it's a long, painful and boring process to get them all).
