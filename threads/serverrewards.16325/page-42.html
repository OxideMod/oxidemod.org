    So I have my images in my Icons folder, type the image/icon names in the URL argument in the data .json file.

    EDIT**

    Nevermind figured it out. If you run into the same issue first, edit your .json file in the data folder.
    Then in console run: oxide.reload ServerRewards
    Then run the command: loadlocalimages
    That resolved it for me.
