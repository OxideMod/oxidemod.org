    The referral system should only be used once. Otherwise your just giving out points for referrals to already existing players.

    I'm not sure everyone knows how it works so here is a example.
    PlayerA invites PlayerB to the server. PlayerB can then type /refer PlayerA which will give them both points. PlayerB can no longer use /refer because they have already been referred to the server. If PlayerA or PlayerB want more points they need to invite more players, the new players can then use /refer, and so on and so on.

I don't forsee PlayerX going 'I didn't invite him, so I won't accept and claim those free points'
