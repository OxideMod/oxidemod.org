    Fixed for latest Rust changes

    The command used to issue server commands changed a bit for Lua, so I've switch to the rust.RunServerCommand library function instead of global.Run.Server:Normal. I also moved the commands on startup back to the OnServerInitialized hook as issuing them earlier may have been causing problems for some.
