    m-Teleportation, Prod and RemoverTool still depend on BuildingOwners.
    For m-Teleportation there isn't really an alternative yet (although it wouldn't be hard to do), so you'd be able to set homes on foundations of others.
    For Prod there is Built for Rust | Oxide as alternative without BuildingOwners (soon EntityOwner will be able to support the same as well, although it isn't done yet).
    For RemoverTool you've got R-Remover Tool | Page 70 | Oxide as an alternative that doesn't depend on BuildingOwners (The setting UseBuildingOwners instead uses Rust's builtin OwnerIDs).
