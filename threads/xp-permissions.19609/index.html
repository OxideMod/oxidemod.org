  1. PaiN

    PaiN submitted a new resource:

    XP Permissions - Grants player permission on their level up

    This plugin grants permissions once a player levels up, where permissions and levels can be set in the config.

    Default Configuration(oxide/config/XPPermissions.json)
    Code:
    {
  "LVL_PERMISSION": {
    "10": "otherplugin.permission2",
    "5": "kit.permission1"
  }
}
    Lang File(oxide/lang/XPPermissions.en.json)
    Code:
    {
  "LVL_PERMISSION_GRANTED": "You have been granted the permission '{perm}' for reaching level {lvl}.",
  "CONSOLE_LVL_PERM_GRANTED": "'{perm}'...
