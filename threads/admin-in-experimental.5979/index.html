    You need to set an initial user (your steamid) via the commands, and then you can set more users. You use the same command for setting the initial owner as you do for others afterward, but you have be physically in front of the server console or use Rusty and RCON. The only way you can use Rusty though, is if you have the +rcon.password and +rcon.port arguments being passed to your server via the command-line parameters. For users that are hosting locally, there is a thread for this and it's fairly clear, for users hosting remotely with no console access, steps are a bit more complicated and often overlooked.
