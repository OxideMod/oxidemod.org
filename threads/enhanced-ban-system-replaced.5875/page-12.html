  8. theconezone

    Wulf said: ↑
    You should be able to ban by SteamID or IP while they are offline.

    From the Overview:
    Click to expand...
    Banning offline users by username does work, banning offline players by steam ID or IP does not. It gives you an "unknown player" error message.

    It still bans the players by steamID and username in the data file though.

    Below is an example with censored steam IDs, usernames and IPs:

    reallyniceadmin ran chat command: /ban 012.345.678.901 "attempted DDOS"
    Player not found
    reallyniceadmin ran chat command: /ban 12345678901234567 "attempted DDOS"
    Player not found
    reallyniceadmin ran chat command: /ban reallymeanguy "attempted DDOS"
    SERVER: reallymeanguy has been permanently banned.
    Click to expand...
    the final command inserts the following into ebsbanlist.json:

    {
    "12345678901234567": {
    "expiration": 0,
    "IP": "0.0.0.0",
    "name": "reallymeanguy",
    "reason": "attempted DDOS",
    "steamID": "12345678901234567"
    }
    }
    Click to expand...
    which bans him by name and steam ID, but not by IP. Unfortunate, but I think it will be sufficient.

    EDIT: I just looked over the code, you can manually add the naughty user's IP to the JSON data and it will block them until someone hopefully looks into this.
