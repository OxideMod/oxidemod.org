    Frenk92 submitted a new resource:

    Vote For Money - Players who vote are rewarded with Economics

    Welcome to my first plugin. This allows you to reward the players who vote with Economics or RP (Server Rewards). For now only works on Rust-Servers.net (because only use this on my server), but if you want, I'll try to add other sites.
    Please report any bugs. Thank you.

    Commands:
    /vote - checks if you voted.

    Default Config
    Code:
    {
  "Id": "",  //Server ID
  "Key": "",  //Server API Key
  "Money": "250",  //Money Reward
  "UseRP": false,  //Use or not RP (Server Rewards)...
