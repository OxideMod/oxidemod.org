    Repair Tool - Repair any entity that has health

    A simple plugin that allows you to repair any entity with health. There are 3 options to choose from;

    - Repair is a tool similar to the remove tool. You can set a amount of time the tool is active. It has a UI countdown and also displays the name of the object you are looking at
    - Radius will repair all objects within the set radius
    - All will repair every object on the map
