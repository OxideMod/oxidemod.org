    That notification that pops up there uses a command. You can use a hook such as OnPlayerCommand() and block it.

    Code:
    private object OnPlayerCommand(ConsoleSystem.Arg arg)
        {
            if (arg.GetString(0).ToLower().StartsWith("note.inv"))
                return true;
            return null;
        }
