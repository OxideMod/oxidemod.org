    Name Grabber - A command to grab name of any item, deployable or entity

    Well i did this a few weeks ago and i find it extremely useful to get names of new items or deployables etc, so i decided to share

    The only command is in console: grabname type name
    Types avaliable: item, deployable and ingame
    Ingame: will loop in every entity deployed in your server and check if the name contains what you want.
