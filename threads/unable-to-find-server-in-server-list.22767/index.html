    The "Amsterdam II" thing is when you directly connect to it via your F1 console, you won't have the actual header, name, or description showing when you hit ESC or disconnect from it, as those are only shown when you connect via the server list, not F1. It's not from it not being set.

    Updating the .bat file does not wipe your server, but your map may change if you change the +server.seed, +server.worldsize, or +server.salt which would also create a new .sav under the "server" folder to match those.
