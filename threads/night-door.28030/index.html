    Slydelix submitted a new resource:

    Night Door - Allows opening of selected door(s) during set time period

    Night Door allows you setup doors that can be only opened in time period specified in the config file

    Chat Commands:
    /nd add - adds the door you are looking at to list of doors that can be open only at certain time period (you must be looking at door for this to work)

    /nd remove - makes the door normal again (you must be looking at door for this to work)
