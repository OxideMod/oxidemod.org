  1. Slydelix

    Slydelix submitted a new resource:

    WipeProtection - Blocks raiding after wipe

    WipeProtection prevents raiding after wipe with option to refund explosives at raid attempt.

    Console commands:
    wipeprotection.manual - use this command if you don't get notified by plugin that it has detected a wipe

    wipeprotection.stop - makes all raiding possible

    Config:
    Code:
    "Refund explosives": true,
"Wipe protection time (hours)": 24
