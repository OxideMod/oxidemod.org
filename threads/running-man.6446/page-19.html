    Hello,
    To set economics, you have to put "money" instead of "karma" in this code
    Code:
        "Karma": {
      "RewardItems": {
        "Karma": {
          "MinValue": 0,
          "MaxValue": 1
        }
      }
    },
    like that
    Code:
        "Money": {
      "RewardItems": {
        "money": {
          "MinValue": 1000,
          "MaxValue": 10000
        }
      }
    },
    You may read the FAQ
