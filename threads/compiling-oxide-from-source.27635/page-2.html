  7. Frenk92

    I found a solution. Run a build batch and select only release configuration (select all needed dependencies and Rust). If you get an error of "UnauthorizedAccess", you have to edit a line of "Rust.csproj":
    Code:
    <Exec Command="powershell -ExecutionPolicy Unrestricted $(SolutionDir)Steam.ps1 -project $(AssemblyName) -dotnet $(TargetFramework) -appid $(SteamAppId) -branch $(SteamBranch) -managed $(ManagedDir)" />
    in
    Code:
    <Exec Command="powershell -ExecutionPolicy Bypass $(SolutionDir)Steam.ps1 -project $(AssemblyName) -dotnet $(TargetFramework) -appid $(SteamAppId) -branch $(SteamBranch) -managed $(ManagedDir)" />
    After you have builded the project, reload it to see all the dependencies. If you get some new errors, let me know.
     
    Frenk92, Oct 21, 2017
