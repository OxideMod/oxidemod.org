  1. Ankawi

    Ankawi submitted a new resource:

    Stash Blocker - Disables players from placing stashes near foundations

    This plugin allows you to configure a radius around a foundation that disables the placement of stashes in them.

    Config:
    Code:
    {
  "FoundationRadius": 5.0 (The radius at which players cannot place stashes near foundations)
}
    Localization:
    Code:
    {
  "CannotPlaceStash": "You cannot place stashes near foundations"
}
