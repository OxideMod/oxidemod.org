    HardcoreDeath - Unlearn a random blueprint on death

    HardcoreDeath make players unlearn a random blueprint on death.

    Configuration
    You can configure the settings in the HardcoreDeath.json file under the oxide/config directory.
    Code:
    {
  "Prefix": "<color=#fdde23>[HARDCORE DEATH]</color> ",
  "ProtectAdmin": false
}
    • Prefix: Prefix style for messages.
    • ProtectAdmin: Protect admins from unlearn blueprints.
