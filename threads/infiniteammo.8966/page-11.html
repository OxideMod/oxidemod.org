    Code:
            private void OnWeaponFired(BaseProjectile projectile, BasePlayer player)
        {
            if (!HasInfiniteAmmo(player)) return;
            projectile.GetItem().condition = projectile.GetItem().info.condition.max;
            if (projectile.primaryMagazine.contents > 0) return;
            projectile.primaryMagazine.contents = projectile.primaryMagazine.capacity;
            projectile.SendNetworkUpdateImmediate();
        }
    Thank you!
    Click to expand...
    You'd have to remove projectile.primaryMagazine.contents = projectile.primaryMagazine.capacity; to start with, then you'd have to add code to check when the player is attempting to reload a weapon. The problem is that if a player has no ammo, the reload hook is never called. So you'd have to also give the player at least 1 ammo for them to reload with and then add more if needed.
