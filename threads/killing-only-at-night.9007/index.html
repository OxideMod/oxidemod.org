  1. CrZy

    CrZy submitted a new resource:

    Killing Only At NIght - Players can only kill each other at night

    This plugin allows players to only be killed at night time. This plugin announces a message when night starts and another message when it ends. These message can be set in the config file. There is also a command to check if it is night or not, in case you join at night and are unsure. The command is '/checknight'.

    Default config:
    Code:
    {
  "Messages": {
    "NightEnd": "Killing time is over!",
    "NightStart": "It is now Killing Time!"
  }
}
