  7. Wulf Community Admin

    Enileca said: ↑
    Ok, so I'm not sure WHERE to put the signartist.cd. I tried in console and in game, both with errors. Can you give me an example?
    Click to expand...
    It's a permission which you'd grant using the console command provided by Oxide.

    URL example for the default user group:
    Code:
    grant group player signartist.url
    CD example for a single player:
    Code:
    grant user Enileca signartist.cd
