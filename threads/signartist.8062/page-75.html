  19. NiueanTurtle

    Go to the Console Command [Press F1]
    Type:

    oxide.grant user "YourCharacterName" "PermissionName"
    For example: oxide.grant user "Galantis" signartist.url


    Here are the 4 possible permissions you may want to give yourself access to:
    • signartist.url -- To use /sil <URL>
    • signartist.owner -- To ignore owner check
    • signartist.cd -- To ignore cooldown check
    • signartist.raw -- To ignore jpg compression
     
    NiueanTurtle, Dec 6, 2016
