    The cancustomimage permission u have used isnt from this plugin, so it cant work^^
    Here the permissions to use with funktions for chat:
    sil_url to give users access to use /sil url on the server
    sil_owner lets the player ignore the ownership of any sign (you can change pictures from signs you dont own)
    sil_cd let you ignore the cooldown until next picture can added to loading queue

    So try using oxide.grant group player sil_url to get it worked
    [DOUBLEPOST=1439546145][/DOUBLEPOST]You even have to get this permission to all other groups if they should able to use the /sil command
