    After downloading the image it is resized, and the filesize is checked at that point as well so it is possible that the resized file is slightly over 1Mb. You can either try increasing the limit to 1.5/2Mb or you could enable the jpeg enforcement and set the quality setting to produce jpeg files which will be considerably smaller than the png files.
