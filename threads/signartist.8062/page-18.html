  13. Brengun

    Hi,
    Had a breakthrough with the in-game chat issue where you type http:// and get kicked from game.
    We did a lot of testing and proved this to be a Unity game engine issue with some game server provider set-ups. (I am thinking a firewall rule, but not proven yet.)

    We still aren't 100% sure what is causing the issue, but we narrowed it down to the Unity game engine and with FPS servers we found only the DDOS protected servers seem to have the issue.
    This problem also happens on a fresh vanilla Rust server.
    We also proved Reign of Kings has the same issue with http:// in chat stopping the client communicating as well.

    Hope this helps others to at least know Rust, Oxide and the plugins you use are not the cause.
    FPS are looking into what can be done to rectify it.
    Note this is a wide spread issue and not just limited to FPS servers.
    From my testing it was over 75% of the 30 random servers I tested this on had this issue.
     
    Brengun, Aug 5, 2015
