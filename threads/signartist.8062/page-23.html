  5. Brengun

    Based on my testing and the fact it still works on some servers I would be asking your server provider for assistance.
    The plugin is NOT the cause of the unresponsive and being kicked

    We ended up going back to a fresh vanilla server to test for where the issue resides.
    I also tested the chat command on many other random servers both vanilla and modded where I got kicked from about 80% of the 30 I tested.

    Backup your server
    remove everything including Oxide.
    Try the same command to prove it is NOT a plugin issue.

    You could instead ask your server provider to give you a vanilla server for a few days to test this theory.
    You will find the vanilla server with NO plugins and NO Oxide will do exactly the same thing when typing the same command in chat.

    This will prove the plugin is NOT the cause.
    [DOUBLEPOST=1440813257][/DOUBLEPOST]A way to know the issue exists after typing the command in chat is to see if you can do a new chat of anything. If the chat doesn't respond you know you will be kicked for unresponsive eventually. This just saves waiting.
