    EasyChatCommands - Easily Add Chat Info Commands

    Edit the configuration file to add your commands!

    For example, you could add /ts to give players your teamspeak server, /rules to tell players the rules, or ever put in a help command! The possibilities are endless!™

    Config file format is:

    {
    "commandName":"commandText", <-- Notice how there are comma here
    "commandName":"commandText", <-- and here
    "commandName":"commandText" <--- But no comma after the last item
    }

    You use \n to create a newline.
