  1. k1lly0u

    k1lly0u submitted a new resource:

    Automatic Client Commands - Automatically run a configurable list of client commands when a player joins the server

    Add client commands to a list and have them run automatically when a player joins your server.

    Console Commands
    addcommand "command here" - Adds a new command
    removecommand "command here" - Removes a command

    ex. addcommand "graphics.branding false"

    ** Note the default config contains the above command!
