    XP Death Reducer - Players lose a percentage, or configurable amount of XP upon death

    As requested by @Brko™

    Deduct XP from players when they die. Either use a percentage of their total amount of XP or set a static amount of loss, with the option to give that amount to the attacker

    Messages can be changed through lang

    Config
    Code:
    {
  "Deductions": {
    "UsePercentageDeduction": true, // Toggles between a percentage or static amount
    "XPDeductionPercentage": 5.0, // Percentage of total XP lost
    "XPDeductionStatic": 50.0 // Static...
