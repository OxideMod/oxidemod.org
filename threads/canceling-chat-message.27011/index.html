  2. Wulf Community Admin

    You can cancel chat by returning the OnPlayerChat or OnUserChat hooks, both work in RustPlugin and CovalencePlugin. If using BetterChat though, you'd need to see its Overview on how to work with it for compatibility, which I'd recommend using its support thread for help with it.
