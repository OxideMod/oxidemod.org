    Try doing the command this frequently manually and make sure it's even possible to do. Some commands the game design only lets you do once until the process is complete... some other require a larger spread between uses. Commands without any of these limits only need a 1 second interval... but never zero (as you already seem to know). If it works via manual command then you may want to try the ingametime-timer instead, and scheduling the commands at the desired intervals in there. You could try that in realtime-timer as well but obviously that will be a lengthier configuration.
