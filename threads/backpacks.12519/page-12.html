    oxide.grant group abcd backpacks.use <----- for granting group abcd
    oxide.grant group default backpacks.use <------ for granting everyone Backpack (default=Everyone).

    You do not need to restart server, just remove the Backpack.json file from oxide/config folder and plugin from oxide/plugins folder. then reinstall plugin I posted. Let me know how it works out!
