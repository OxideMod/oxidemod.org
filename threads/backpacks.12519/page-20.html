    You can assign permissions per player or per group. assigning to groups is the easiest way, like wazzzup said.
    Or take the time to individually assign and unassign each player a permission.

    default, moderator, admin are the base game groups that are default when you start the server. (as you should already know this)

    adding extra groups allows you to assign the "added" groups permission then adding/removing players to those groups which will not affect any of the default game groups.

    the way I have mine setup allows me to set up a group with permissions, example

    default: base mods that have permissons,
    Moderator: admin mods + auth lvl 1
    admin: admin mods + auth lvl 2 which is for me the owner.

    vip: the mods permissions you don't want players to have access to if they aren't in the group.

    I have 3 lvls of vip, which the group "default" and "moderator" cant access unless they purchase.

    VIP 1: basic vip permissions
    VIP 2: better vip permissions
    VIP 3: ultimate vip permission

    Good luck.
