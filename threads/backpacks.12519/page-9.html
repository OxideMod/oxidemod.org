    Click to expand...
    To add backpack to the permission system, you need to have groups that people are using.

    To enable the backpack plugin, (and expecting that you have installed oxide and putt the Backpacks.cs file into the plugin folder) you need first of all to add a group that you can link the backpack plugin to.

    Use the following steps: (type in dedicated server concole)


    grant group player backpack.use

    The following command will add the backpack.use plugin to use by all players.
