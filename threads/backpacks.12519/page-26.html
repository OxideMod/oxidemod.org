    In case it's not clear, Rust now blocks the server from binding keys on the client (that is, connecting a key to command). When the plug-in tries to run a command to bind the "B" key to the backpack.open command, the client blocks it. This may have been done to prevent Rust Servers from making changes on the player's client machines.
