  12. mspeedie

    Make sure you have the most recent Oxide version. A lot of plugins will not function when you have an out of date oxide. I check once a day for updates to Oxide and a few times a day for the first few days if the patch is big (like the last two have been).
    make sure you have done these grants:
    oxide.grant group admin backpacks.use
    oxide.grant group admin backpacks.use.large
    oxide.grant group admin backpacks.admin

    that will allow admin to use backpacks, give them the large one and allow them to issue admin commands.

    For other folks use default to grant access and pick a size small, medium or large. I think small maps to medium in the current code looking at line 57.
    oxide.grant group default backpacks.use
    oxide.grant group default backpacks.use.large

    Some servers only allow VIP access to backpacks, I like servers that give it to default, and the larger the better. ;-)
     
    mspeedie, Nov 1, 2017
