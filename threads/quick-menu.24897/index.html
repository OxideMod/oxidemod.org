    Xianith submitted a new resource:

    Quick Menu - A easily accessible menu that contains three configurable buttons.

    A easily accessible menu that contains three configurable buttons.



    Currently only supports red, green and blue for color input!

    The command config will be sent as chat so make sure it's a chat command and not a console command.

    Example config:
    Code:
    {
  "Bottom Button": {
    "Color": "red",
    "Command": "/remove",
    "Text": "Remover"
  },
  "Middle Button": {
    "Color": "blue",
    "Command": "/tpc",
    "Text": "Teleport Cancel"...
