  2. Wulf Community Admin

    The first step would be to rule out possible culprits. You can completely rule out Oxide being a cause by removing all plugins. If the "lag" continues after doing that, I'd try a server restart and let it run for awhile to see if it comes back. If it seems to be fine, then I'd start adding plugins back one-by-one. If the lag does not go away with all plugins remove, then it's either an issue with your host or the game itself.
