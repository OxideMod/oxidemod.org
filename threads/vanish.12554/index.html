    Wulf submitted a new resource:

    Vanish - Allows players with permission to become invisible

    Vanish allows players with permission to become completely invisible. Other players will not be able to see, hear, or touch you! By default, admin will be able to see each other, but you can change that in the config file.

    Permissions
    This plugin uses Oxide's permission system. To assign a permission, use grant user <username|steamid> <permission>. To remove a permission, use revoke user <username|steamid> <permission>.
