    Click to expand...
    What???????
    To grant permissions you need the following:

    A. be admin (Auth 2)

    B. in Rcon or f1 console type o.grant group admin vanish.use

    C. If you don't want to damage anything whilst in vanish then grant the following:
    1. o.grant group admin vanish.damage.buildings
    2. o.grant group admin vanish.damage.animals
    3. o.grant group admin vanish.damage.players

    D. if you want to be invunerable(not take damage) grant :
    1. O.grant group admin vanish.abilities.invulnerable

    E. Once you have setup your permissions then in text chat type /vanish

    F. if you want to bind a key to vanish then in F1 console type:
    bind x vanish
    Change x for what ever key you want to use

    hope this helps

    Zodiak
