    There may be a permission not quoted in the plugin description: vanish.abilities.weapons

    To allow the user to pull their weapon when invisible: o.revoke user [playername] vanish.abilities.weapons
    To disallow the weapon when invisible: o.grant user [playername] vanish.abilities.weapons
