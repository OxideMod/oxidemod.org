  1. FuJiCuRa

    Fujikura submitted a new resource:

    SaveMyMap - A save location bridge to support a better map save revision

    This plugin is a simple bridge, which splits the map save process in two locations.

    It does save the default file in your instance folder, and does save into a custom folder in your oxide directory. Those saved files get an UnixTimeStamp as prefix, to save them all in a folder sorted from oldest to newest file.

    Those files could then be named like that: 1472842414-craggyisland.1024.1725106428.134.sav

    It does by default-config use an saveinterval of 15 minutes = 900...
