  6. Domestos

    Its returning a userdata so you need to enumerate through it.
    Code:
    local steamID = rust.UserIDFromPlayer(player)
local groups = permission.GetUserGroups(steamID)
local enum = groups:GetEnumerator()
while enum:MoveNext() do
    print(enum.Current)
end
