  18. CaptainNebulous

    Try removing the other styling, like size and all that. Really the only formatting that works right now in hurtworld is like the following example

    "<color=colorname or hexcode> Your Message Here </color>"

    Size and font-styles either don't work at all or they work but cause tons of chat lag and errors, from my personal experience.

    If you just use the color name, you do not use a hex (#) at the beginning. If you use a hex code, then you use #. I recommend the following site to get your hex codes

    Color Hex Color Codes

    Also try not to go too crazy as the more color codes you have in your chat output, the more laggy chat can get. For example I recently had to redo my adverts in a way that they only use one color. Before I'd have two colors in each advert and it caused chat lag. Although right now chat lag seems to be somewhat of an issue no matter what, unless you don't run oxide mods (think it's on Hurtworld's side though because there used to be no chat lag until one of the summer patches)

    Also even just ONE broken code will mess up your entire chat for a time. It is a bit difficult to tell which is actually broken by just chat alone unless you saw precisely which message popped up at the moment chat broke. Best to just go into the file yourself and search for where you screwed up.
