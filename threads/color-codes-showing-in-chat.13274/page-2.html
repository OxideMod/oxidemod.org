  19. Wulf Community Admin

    Armourhotdog said: ↑
    still happens, all the time, hurtworld devs think it is caused by oxide.
    Click to expand...
    It isn't directly caused by Oxide, but it happens only with Oxide as it allows sending color codes in the chat messages, vanilla Hurtworld doesn't aside from the very first part in the name prefix.

    Pretty sure it's caused by the client's chat splitting up messages, which breaks the HTML formatting in Unity when it wraps the text to a new line. Oxide can't control how the Hurtworld client handles color codes, so I don't think it's something we can easily fix without blocking colors entirely from chat messages over X length or manually sending an entirely new message for each line of text instead of allowing the client to wrap it.

    To sum it up, Oxide doesn't do anything other than sending the message, the handling is all done by the game and Unity.
     
    Wulf, Jan 17, 2016
