    Make a owner group in the config so it will auto add the Owner tag to that group
    Code:
    "Owner": {
      "NameColor": "#5af",
      "Permission": "owner",
      "Prefix": "[Owner]",
      "PrefixColor": "#FF7F50",
      "PrefixPosition": "left",
      "PriorityRank": 6,
      "ShowPrefix": true,
      "TextColor": "#ffffff"
    },
