    Chat Handler - Many features to help moderate the chat

    Description
    This plugin helps you moderate your chat in many ways.

    Features
    Mute/Unmute
    Chat Commands
    /mute <name/steamID> to permanently mute someone
    /mute <name/steam64id> <time[m|h]> to mute someone for a specific time e.g. "/mute Domestos 2h" to mute for 2 hours or "/mute Domestos 10m" to mute for 10 minutes
    every muted player who writes in chat while muted gets a notice with the time left on his mute
    /unmute <name/steam64id> to unmute someone
