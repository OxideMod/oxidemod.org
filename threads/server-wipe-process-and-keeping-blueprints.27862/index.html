    1. If it's a force wip like yesterday it will wipe. You can't back it up.
    2. I'm not sure about that but the blueprints are kept in a folder called Users. Just try to copy and paste that over.
    3. Each First Thursday of every month there is a force wipe along with the rust update. You can't backup your save. It will be wiped no matter what.
    4. If you want to wipe your server just delete the .sav file and if you want to wipe blueprints delete the users folder. When wiping your server it's important to remember that some plugins have data files where they store stuff like N-Teleportation. Go to the oxide/data dir and delete some of the data file of plugins that store info
    Here's a really great plugin to manage blueprints: Blueprint Manager for Rust | Oxide
