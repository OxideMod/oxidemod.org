  1. DumbleDora

    DumbleDora submitted a new resource:

    Mute - gives adminsa /mute and /unmute for players in chat

    The mute plugin gives admins access to /mute and /unmute commands to mute players from the normal chat ingame. Players can still use / commands - so be warned some of these for other mods like the whisper mod will still work to produce chat.
    The muted players will persist server restarts and shutdowns if the server is shut down correctly.
    There is also a /muted command to see who is muted, however this currently just shows the players' id rather than name

    The plugin allows all those with Oxide 'Admin' permission to use it yes.
    Originally it had a config file where users could specify whoever they want to use it - however plugins that don't require config files are generally more well regarded as there users don't have to go and edit them or the script.
    To be honest, server's should have properly set up Oxide permissions folders as other plugins will most likely follow this route, so it may be a good thing to do now.
