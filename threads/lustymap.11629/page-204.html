    ok...think I may have figured it out...well, I tried something and it's working for me now...when i extracted the map icons and put them into the "/server/oxide/data/" directory, i thought everything was getting extracted...but it turns out there are 197 map icons that are in the zip file and only 196 were getting copied into the server (/server/oxide/data/)...there is a "thumbs.db" database file that wasn't making it's way over...i had to physically drag and drop it from WinRAR into my FTP. Bottom line, you need to have 197 files in the LustyMap/Icon folder...i only had 196 until i manually moved the "thumbs.db" file into the server. Now the map works like a champ...it's 100% accurate, and I can bring up the large screen "m" map, without getting kicked, which was one of my issues earlier today. Anyway, worth a shot, hope it works for ya'll as well.
