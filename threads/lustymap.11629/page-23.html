    We used to host with Streamline, I highly recommend you connect with a proper FTP tool rather than use their web interface. It'll still be weird connection-wise but you can queue up what you want.

    As for what to upload, you don't need to upload the MapSplitter.exe. That is only for you if you want to use the complex mode of the minimap.

    Start by replacing the map.jpg with your map file and making sure that it's called map.jpg. If you want to use complex mode this is when you run MapSplitter.exe, after it's finished remove it from the folder. Then upload the entire directory LustyMap to the oxide/data folder on the FTP. Then reload the plugin.
