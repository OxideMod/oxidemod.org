  10. Tyrmas

    Last update fixed all the issues I had in my server - thank you for that.

    PHANTOM 6 said: ↑
    Is this a bug or am i just not doing it right?
    I disable vending machines on map but players still reporting vending machines show up when they build a papermap and press g
    Click to expand...
    You disabled the vending machines in the LustyMap config? Can you see the vending machines in LustyMap? Configuring LustyMap doesn't affect paper map so it's not a bug.

    EDIT:
    Had to check this out. Vending machines have broadcasting on by default. You need to disable it one by one. Go behind a vending machine, hold E and choose "Disable Broadcasting". This removes the icon from the paper map.

    I noticed LustyMap doesn't automatically remove icons if you disable broadcasting from a vending machine on the fly (or then it has huge delay?) but if you reload the plugin, icons disappear too. Downside is that when someone puts down a VM it begins broadcasting by default so when they disable it, you have to reload the plugin.
