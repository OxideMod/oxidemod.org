    FunkyArt for me it worked like this:
    Delete old plugin, config file and in oxide/data/LustyMap leave only the folders containing the icons delete the rest(json config files).
    After this copy the plugin to your folder. Now you will get an error that you need an API key. Go get your API key from BeancanIO and set up your LustyMap config file. Once this is done reload the plugin. At first you may get an error(i got one) if so reload the plugin again and it should work.
