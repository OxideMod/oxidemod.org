  6. GoDANKorGoHOME

    I can confirm. Map auto generation when you use the API key is off. The monuments are in the correct place but the topography is completely different. That being said if you generate the exact seed on beancan.io it DOES generate the proper map. Other than the issue with the topography with the API key and being unable to add custom markers ATM I'm digging it. Keep up the good work!

    Edit: After having found the topography to be off, all it took was unloading LustyMap, deleting the CustomData and ImageData in the data/LustyMap directory. I proceeded to generate the same seed on beancan.io saved it as a jpg. From there place the jpg into your data/LustyMap directory and edit the config. You will want to put the file name. Ex. "CustomMap_Filename": "Map.jpg" and turn the "CustomMap_Use" to true. After doing those steps I now have the proper map showing for myself and my players. I also removed my API key since I was using a custom image.

    Edit Edit: If your custom map has a space it WILL NOT load!

    It seems you have old LustyMap files in your data directory. When I updated to the newest LustyMap I deleted everything and started fresh. The new LustyMap will only need the Icons and Custom folder in the data directory which you download on the overview. What I would do is unload LustyMap if its loaded and delete everything in the data/LustyMap EXCEPT the Icons and Custom. Delete the config and go to the lang directory and delete the LustyMap file there just to be safe. After that make sure you have the latest LustyMap download and load it up. That should create a new config. In the config put the name of the of the map you are wanting to use Ex. "CustomMap_Filename": "Map.jpg" and turn the "CustomMap_Use" to true. Save it, delete the data/LustyMap/Imagedata and CustomData if its there and reload LustyMap. Hopefully this will help ya bud.

    Edit: I don't use complex mode so I am not 100% sure if the 6x6 12x12 32x32 64x64 are used for that. My data/LustyMap directory ONLY has the Icons and Custom folder as well as my custom map and the ImageData/CustomData file.
