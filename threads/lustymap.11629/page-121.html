  13. sami37

    Solution to all admin with black map LustyMap | Page 119 | Oxide
    Take this version from wulf, update your server to 2.0.2781+
    Edit LustyMap.cs and replace "CommunityEntity.ServerInstance.net.ID" by your map seed like

    Code:
    uint textureID = FileStorage.server.Store(stream, FileStorage.Type.png, 156184);
    Once plugin uploaded, tip in console resetmap

    I am not using local map anymore, so i haven't tested it with local storage.
