    1.1.26

    Added option to change the Keybind

    There is currently a bug with the rust client (not LustyMap!) that triggers keybinds when typing into chat / console etc...

    Added an option so you can change the Keybind used to toggle the map.

    I recommend using F2 until the bug is fixed
    Code:
    /map keybind F2
