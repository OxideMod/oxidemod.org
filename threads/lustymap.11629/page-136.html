  18. luckyguardian

    Got it working using these steps. It used a local file instead of beancan.io

    I used wulf's LustyMap.cs from page 119.
    Deleted the json files and the original .cs file.
    Generated a custom map on beancan.io using my seed number and map size.
    Placed the .jpg file of my map in oxide/data/LustyMap
    Opened LustyMap.cs in my text editor.
    Changed CustsomMap_Filename to "my .jpg filename"
    Made sure APIKey was = ""
    Changed CustomMap_Use to true
    Changed UseComplexMap to false
    On Rusty I used oxide.unload LustyMap
    Then used oxide.reload LustyMap
