  3. Phoenix Renegade

    It really is not hard to get this working.

    1) Download the plugin and put it onto your server in the plugins folder
    2) Go to Beancan.io, make an account and register your server. Use something like Rusty or the in-game console to get the info you need to register your server like the ip/port etc. if needed.
    3) Once registered, copy your api key that is displayed once you post the server. (Note: It does not matter if it says the server is going to be checked by a moderator or whatever. Your key will still work.)
    4) Open LustyMap.json with something like Notepad++ and find the "APIKey" = "" line. Put your code inbetween the two "" (Example: "APIKey" = "YOUR CODE HERE"
    5) Save the file and upload it to your server in whichever way your host allows.
    6) load/reload LustyMap with something like Rusty or the in-game console.
    7) If you were in game while these changes were being made, close Rust and reopen and log back onto your server.
    8) IF you had LustyMap before this update, to remove all traces of LustyMap files from before the update. So, delete the LustyMap data folder and the LustyMap.json file. Load and reload the plugin to recreate these files to make sure they are up to date and current.
    9) If any changes are made while you are logged in to your Rust Server, RELOG IN. <-- IMPORTANT! DO NO SKIP THIS STEP OR YOU WILL THINK IT DID NOT WORK!

    These steps have worked for me and many others. Just go slow, READ each step, do the step then move on. Go step by step and be certain of what you are doing. Don't be impatient and rush and half-ass an attempt to get it working.
     
    Phoenix Renegade, Dec 12, 2016
