  5. GSTYLEZ

    Hey guys. Here's how I get my maps.
    I use either playrusthq or rust:io to generate a map.
    After I choose a map. I find my playrust.io map either by searching or typing in your ip like what's linked above.

    I then usually generate a map on playrusthq using the SAME seed and size. The image should be exactly the same as playrust but playrusthq shows the cave locations.

    If you left click on the map, it will navigate you to another page with a high-res version of the image without caves, monuments etc. Right click. Save image as map. (navigate to oxide/data folder under the LustyMap folder and replace or save it there under map).

    Optional: You can then navigate to that folder on your computer and just double click the splitter.exe. It will run but not give you any confirmation and just close. Rest assured, the program automatically split up your map.

    Go in game. Either type this in console without quotes: "reload LustyMap" or just type "/map images" in game chat.

    Note: You have to have Rust:IO installed I believe for this to work.

    Your map should refresh with the new image.
    Good luck
