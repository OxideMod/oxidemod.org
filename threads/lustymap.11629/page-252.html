    Since a lot of people are using 2.0.93 I added the rocket icon to the version attached. This is without imagelibrary, if you replace it, use "resetmap" cmd and Wait until it's done.

    v2.0.93 gave an "RPC_Error" if your map image's resolution was too large.
    I think Beancan.io maps are saving at 1440x1440 now so automated maps should work.
    If you're using a custom map, Make sure you check the image size- Right click the image, /properties/details- to view the size. Edit the image with any app (if needed) to be closer to 1024x1024. 1440x1440 should still be fine, any larger will crash the server with the rpc error.
