    Should I edit the LustyMap.cs to remove the API key if I'm using the custom map?
    Click to expand...
    Go to Beancan.io and save your map image to your desktop.
    You'll need to crop the image to 1440x1440 using any form of photo edit tool. (I use Gimp2.8, it's like a free version of PhotoShop)
    If you're not familiar with Gimp, after installing it, Click File>Open> and find the map image you saved to desktop.
    It will open and you will see the layer to the right that contains the image.
    Click on the layer, then click "Image" at the top (top of app says: File, Edit, Select, and Image. Find the option to "Scale Image" and set it to 1440x1440.
    Now go to "File" and chose "Export As" and you can name it "Map" and make sure it exports to desktop as a .png
    Now place the image in the Oxide/data/lustymap folder. Make sure you have your config set to work with a custom map. Type "resetmap" in the console. It should now work

    The reason for editing the map is that Beancan.io default map size is too large. Using Gimp2.8, You not only make the map resolution smaller, but compress the image file as well.
