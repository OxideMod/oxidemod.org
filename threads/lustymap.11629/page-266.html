  6. YoDawg

    UBI it took us forever to get this to work using my own map not beancan

    Here are the steps i took..
    1. Download icon pack and extract to /Oxide/Data/LustyMaps/icons folder and if its not present then create it yourself.
    2. Verify your LustyMap Config is correct. Use mine as an example if you are using your own map and not beancan
    3. Then enjoy doing this a few times
    4. oxide.reload LustyMap "check server"
    5. oxide.unload ImageLibrary "check server"
    6. oxide.reload ImageLibrary "check server"
    7. oxide.reload LustyMap "check server"

    this should cause a tick flood but it got mine to work
