    "duped" icons are from using an image that already has icons, then using lustymap to place icons. if your image has the icons already (which yours does) you would turn "ShowMonuments" to "false" in the config, or use an image that doesn't have icons already.

    *edit I see you wiped the map, but now you know why it happened I guess

    *edit #2
    lustymap sets the marker size on the map and the position on the map, if it added duped icons they would be in the same place and the same size.
    if it was a plugin issue it would be from removing the wrong folder, the data is stored in the data folder. oxide/data/imagelibrary/image_data
