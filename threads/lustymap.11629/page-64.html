  20. Mughisi

    It might be possible if the hook is moved a bit to be triggered earlier but I am not sure if that would be a solution. Disconnect can also be triggered after the client crashes and at that point there won't even be a client to run the command on when the player disconnects, same goes for when a player just closes the game while on a server. Another option would be by using the hook that captures the input of players but that might have a big impact on performance on more populated server.

    It is also just possible to ignore the error on vanilla server, it might be annoying that it shows up multiple times but it doesn't really cause any harm.
