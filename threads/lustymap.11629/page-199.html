    As a temp fix, use a custom map image. Download them from Rust Live Map & Map Generator by typing the map seed into the website.

    1) Download the image from Rust:IO and name it map.png
    2) Upload the image into data>lustymap
    3) Go into config>lustymap and change "CustomMap_Use": to true
    4) Add "map.png", to CustomMap_Filename
