  8. GonzoTheThird

    I have the same problem with the drops not showing after the last update.

    I also have problems with some players having a black map, but since ever....
    The black map can usually be fixed by
    1) running resetmap in the console
    2) reloading the plugin o.reload LustyMap
    3) player reconnect

    If that does not fix it or you get an error during resetmap try
    1) run refreshallimages in the console to refresh the image library
    2) wait, wait, wait
    3) eventually run resetmap and o.reload LustyMap
    4) player reconnect
     
    GonzoTheThird, Feb 3, 2018
