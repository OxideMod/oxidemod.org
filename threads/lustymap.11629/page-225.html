    There's tips in the Overview tab to get your server on Beancan.io
    While waiting for approval or as an alternative, You can use a Custom map.
    Things to note:
    -If live changes to the config don't update automatically, you need to reload the plugin.
    - If you add or change a map image while the server is running, "resetmap" cmd will write the new imagedata to the imagelibrary.
    -If you follow this video step for step "oxide.reload ImageLibrary" should load the map for the first time, it may take a minute to load.
    -If you use the cmd "refreshallimages" (lustymap use doesn't require this cmd) Images will say "No Image Found" to fix that type "resetmap" and the images will reload.
    That should cover the Basics.
