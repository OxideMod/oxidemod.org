    Only those that aren't using Oxide's Rust CUI helper. The fix is to either update to what Rust changed, or use Oxide's Rust CUI helper.

    If using Rust's CUI directly:
    Code:
    var obj = new Facepunch.ObjectList?(new Facepunch.ObjectList(<THAT_4TH(LAST)_ARG_HERE>));
    and then replace 4th (last) arg in the `ClientRPCEx` method to `obj`
