    "speed" in the config refers to their normal walking speed. hostile animals only seem to use boost speed when they are running away (less than 10% hp). for hostile animals i would make the speed high enough for them to catch people but not too high. if the speed is like 1.0 then they wont likely catch anybody. for passive animals, they seem to use the boost when they see players.. so you can make this setting much lower so it seems more natural. for example, set horse speed to 1.0 and their run speed to 2.0. another thing with horses, the default HP for them is 500 so you'll want to edit that too. here's my config:
