  3. Wulf Community Admin

    The noclip/fly features are provided by Rust and are entirely client-side. They require auth level 2 with Rust, not any permissions from Oxide. To see how to setup owners with Rust, see Admin and moderators in Rust | Oxide.

    I'd also recommend looking at this thread: No clip / fly not working as admin | Oxide.

    As far as kicking goes, you can already do that with Rust's auth levels, else you can use a plugin to allow it per user or group with Oxide.
