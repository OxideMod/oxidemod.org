  6. PaiN

    I will explain you here (READ IT ALL)!!
    A. Some plugins are using Oxide Permission System and some other are yousing Rust's Authlevels (1, 2)

    B. About the GodMode plugin it uses Oxide Permission System so you go to it's Overview( http://oxidemod.org/plugins/godmode.673/ ) and as you can see it says "Required Permissions For players to be able to use the commands that this plugin offers the permission canusegodmode is required." so what you need to do is:

    B1. Download the plugin.

    B2. Go to your console (RUSTY)

    C. Lets now make a group named owner and assign the permission to it!

    C1. oxide.group add owner -- Yay you created your own group named owner!

    C2. oxide.grant group owner canusegodmode -- Now you gave the group "owner" the permission to be able to fly! (You can assign more permission from other plugins too later)

    C3. oxide.usergroup add "Grim Seeker" "owner" -- Grants you the group with the permissions that you have set

    C4. ALTERNATIVE WAY You can also assign the permission to a user.. if you dont want to make a group.. oxide.grant user "Grim Seeker" "canusegodmode"
