  2. Wulf Community Admin

    Rust has a built-in auto-save, which can configure via cfg/server.cfg. There are also auto-save plugins and command plugins for Oxide, if you're using that. If your map was "wiped", then it's likely it was either corrupt, you/server didn't save it before restarting, or your host wiped it from an update/upgrade you may have done.
