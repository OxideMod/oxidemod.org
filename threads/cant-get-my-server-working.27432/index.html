  5. Fredern

    Hi Magma, I'm using gameservers aswell. To change your server name you should go to gameservers.com and then head into "config" folder. There it will say "+server.hostname" "GameServers.com", change the "GameServers.com" to what ever you like. To install mods you head into "mods" at the same place you find the config folder. Install Oxide 2 and it should install by itself. To add mods you have to use FTP, a plugin for your FireFox browser, when you got that sorted out you have to go to "info" folder and right-click the FTP URL and select "open with FireFTP" type in your password, after going through the first folder you will find a folder called "ruste" go into that and if oxide is installed correctly it will be a oxide folder there. Go into the oxide -> plugins and add all plugins you want.

    About the server not being public is also an issue that I have at the moment, it just doesn't show up on the server list, don't know what's causing it. Contacted GameServers about it, I hope they'll respond fast.

    Edit: Hope this helps, if something is unclear just ask and I'll try to explain it better.
