    Reneb submitted a new resource:

    DeadPlayerList - addition table for rust. (useful for plugins that require it)

    usefull only for plugins that require it.

    Rust has 2 table:
    1 for active players (currently playing)
    1 for sleeping players (disconnected or sleeping)
    but when you get killed + disconnected you disappear from the list.
    so this list will provide another table from which plugins can call to get those players informations.

    how to use: (thanks Domestos)
    Code:
    local deadPlayerList = false
function PLUGIN:Init()
local pluginList = plugins.GetAll()
