  8. Cheeze

    I have a plugin of my own, I say my own when really it's a mashup of code from a couple of other people's plugins. Obviously I can't release it as it's not really my own work as such. I'll explain what it does so that you could consider adding it to an existing plugin.

    I call it event-system and how it works is when you've built an arena for events, you make sure you have a holding area for players to wait. You use Reneb's zones plugin to make sure certain things in the holding area are disabled like pvp. Once ready my mash up plugin comes into play. You set a spawn point using a command. Then players who want to take part can warp to that spawn point but when they arrive their inventory is automatically stripped and their health healed. This puts everyone at the same level. Then we give out a kit we've set up for the event.
