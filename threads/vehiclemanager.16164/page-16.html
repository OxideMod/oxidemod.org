  16. TruxDeadnitro

    TheOrphanCrippler said: ↑
    So, there is still no way to claim more than 1 vehicle (and keep them)?
    Click to expand...
    There is when you've installed this plugin inside of your server,
    then goto the json file from the vehicle manager which can be find on the section oxide/config/VehicleManager.json
    and set the ammount of the vehicle's that you want to claim default from the plugin is 1,
    so remove the 1 and set it like this 20 or higher,
    save it and type inside of the console/server oxide.reload VehicleManager
    When it is done with reloading the plugin you're able to claim more then one vehicle,
    But beaware, on every server restart you've to claim your vehicle's again,
    with the command /car claim
     
    TruxDeadnitro, Feb 26, 2018
