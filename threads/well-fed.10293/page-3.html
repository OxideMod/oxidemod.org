  18. DonMario

    Thanks
    And yes, it does work, if you read all the comments you'd know.

    If you want admins to be full fed on spawn type in console
    "oxide.grant group admin wellfed.onspawn"
    If you want admins to be full fed on login type in console
    "oxide.grant group admin wellfed.onlogin"
    If you want to apply both, go ahead and type both those commands in console, just without the " "

    If you want all players to be full fed on spawn type in console
    "oxide.grant group default wellfed.onspawn"
    If you want all players to be full fed on login type in console
    "oxide.grant group default wellfed.onlogin"
    If you want to apply both, go ahead and type both those commands in console, just without the " "

    Please like if this helped
