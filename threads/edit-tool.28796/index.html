  1. Jake Rich

    Jake Rich submitted a new resource:

    Edit Tool - Provides a GMod type tool to edit your environment. Additonal plugins can add functionality.

    Edit Tool lets you change your environment as you please!

    Mainly for use by admins, but also support permissions so players can be allowed to use it, and prevent players from picking up the tool and abusing it.

    Permissions:

    edittool.use:     Required permission for non-admins to use the tool.
    edittool.transform: Required permission for non-admins to move around and rotate entities.
    edittool.clone: Required permission for non-admins to copy and paste entities.
