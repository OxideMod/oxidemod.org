    The syntax is
    oxide.grant user <UsernameHere> <PermissionHere>

    So if I wanted to grant myself permission for calling all kinds of airdrops, I would write
    oxide.grant user LaserHydra easyairdrop.call
    oxide.grant user LaserHydra easyairdrop.call.player
    oxide.grant user LaserHydra easyairdrop.call.position
    oxide.grant user LaserHydra easyairdrop.call.mass

    You can also grant it to a group, for example the admin group:
    oxide.grant group admin easyairdrop.call
