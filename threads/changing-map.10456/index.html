    Code:
    @RustDedicated -batchmode +server.port 28015 +rcon.port 28016 +rcon.ip 0.0.0.0 +rcon.password "my password" +server.seed 123456
    or

    Create a "cfg" folder in your server's root, where RustDedicated.exe is, and then create a server.cfg file in that cfg folder. Add the "server.seed 123456" line there, and any other commands/settings you'd like with one per line.
