  6. Mezakin

    Hey. Info for people struggling.

    This is how I got things working.

    1. Install ImageLib
    2. Install AbsolutMarket
    3. Run Game
    4. Unload AbsolutMarket (oxide.unload) command
    5. Load AbsoluteMarket (oxide.load) command
    6. "getimages" command
    7. Done

    A problem I've found though and I can't see someone posting it so I apologise if it has;

    If you accept a trade and you have the item in your inventory then all good. However, If you have the item in your hotbar, then you get both Items and the person who made the trade request gets nothing. Tried and tested a few times with various items (Apples, Ammo and Signs)

    Also I dont know if intended but you cant accept a trade if the requested item is not in your hotbar or inventory.

    Great plugin though. Really love it!
