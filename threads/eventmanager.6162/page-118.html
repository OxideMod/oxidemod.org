    very easily. first generate your arenas then create your zones.
    after that make all your spawn points for all your arenas and label them whatever you want for each game.
    make your kits and do the same.
    after that then go into config files for each game and set them up correctly .
    put in whatever you labelled for spawnA&B for TDM.
    do the same for Gungame.deathmatch.chopperSurvival.
    then also label the correct kits.
    having everything labelled correctly then you will not see the errors. because the messages you are seeing are actually for when the plugin loads for the first time and it is looking for Default Parameters not the ones you specified.
    most people wont edit the config and they will just make each event in the GUI (/event) but you will still see these errors because its the plugin loading for the first time and looking for default settings.
