  12. CaseMan

    CaseMan updated Gather Control with a new update entry:

    1.4.1

    1. Added permissions for using chat and console commands.
    Without permissions, commands are only available to the administrator.
    Code:
    gathercontrol.AllowChatCommand
gathercontrol.AllowConsoleCommand

    2. Added support for static quarries and oil pumps.
    The gather multiplier is specified in the config file and does not depend on the player's multiplier.
    Only the static multiplier!!! It is not dependent on the permissions.
    Code:
    "DayRateMultStaticQuarry": 1.0,...
