  1. Ankawi

    Ankawi submitted a new resource:

    Player Rankings - Automatically gives players ranks based on playtime

    PlayerRankings is a plugin that automatically gives players ranks based on playtime on a server.

    Config Info:
    • For playtime, it counts the hours, meaning that a playtime of 1.0 means 1 hour.
    • Simply edit the permissions, playtime, and rank that you want, and also edit the permissions only in BetterChat's configs.
    Configs:
    Code:
    {
  "Casual": {
  "Permission": "color_casual",
  "Playtime": "5.0"
  },
  "Noob": {
  "Permission": "color_noob",...

    DikkatDeli said: ↑
    should i use this command for plugin to work
    oxide.grant group player color_noob ?
    Click to expand...
    You don't need to use anything besides ConnectionDB and BetterChat, the plugin automatically sends permissions to the players. There are 3 groups, Noob, Regular, Pro. With the latest update from LaserHydra, those groups were created, and already have the permissions, so what the plugin does it sends that specific permission in that group and automatically gives them the colors, titles, etc that you set in BetterChat's configs.

    It's also oxide.grant user "color_noob", but the plugin does that for you.
