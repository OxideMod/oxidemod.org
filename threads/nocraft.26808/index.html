  1. Ryan

    Ryan submitted a new resource:

    NoCraft - Blacklist craft items or disable crafting alltogether...

    Prevents people crafting blacklisted items or gives you the option to disable crafting on your server altogether.

    This plugin uses Oxide's permission system. To assign a permission, use grant user <name or steam id> <permission>. To remove a permission, use revoke user <name or steam id> <permission>.
    • nocraft.bypass (Bypasses craft config options)
      Ex. grant user Ryan nocraft.bypass
      Ex. revoke user Ryan...
