    ☭ evil eyes ͡๏̯͡๏ said: ↑
    Hello everybody! I'm new to here, but thanks to your guides I've launched my ever first VDS server
    But there is still one question: does anybody know optimal hardware configuration for the 60 slots server?
    Click to expand...
    If you run a constant 50-60 player server that is always full and you don't restart it, it will start to eat up ram. Almost 6GB ram needed. This also is if you do not wipe the server.. If you have a clean fresh server install it wont eat up that much. I suggest a dedicated server with 8GB ram MINIMUM if you are wanting to start hosting bigger servers.

    Something like a Dual E5420 with 8GB or 16GB ram for the server will run the game pretty decent. Anything newer in the CPU department such as a xeon / opteron with the the same clock speed or higher will be better. The dual E5420 is a cpu from 2007..


    If you want OPTIMAL something like a Intel Xeon E3-1231 v3 with 16 GB ram with a Solid State Drive will do great..
