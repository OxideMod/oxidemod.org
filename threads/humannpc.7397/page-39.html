    [DOUBLEPOST=1442872211][/DOUBLEPOST]
    saintnoodle_ said: ↑
    Have you tried addressing the issues directly? My players keep killing the NPCs for the sake of it and they have invulnerability set to true and attack damage set high.
    Click to expand...
    Easiest way to fix it would be to open the plugin, search for the line: void OnEntityTakeDamage(BaseCombatEntity entity, HitInfo hitinfo)

    Then add this under the Invulnerability call: hitinfo.damageTypes.ScaleAll(0f);

    Would look like this:
    Code:
                        if (entity.GetComponent<HumanPlayer>().invulnerability)
                    {
                    hitinfo.damageTypes = emptyDamage;
                    hitinfo.HitMaterial = 0;
                    hitinfo.damageTypes.ScaleAll(0f);
                    return;
                    }
    That will filter and reduce all damage to None

    If it doesn't work, try rebooting your server and see if it does work. Unfortunately, my plugin used to break when a npc dies. Now it shouldn't break if you put them invulnerable
