    first: create a kit
    /kit add Kit_Npc
    get items that you want into your inventary (clothing, weapons, etc...)
    /kit authlevel 2 items hide true (very important to save the kit, and hidden when the players write /kit list)

    then create a new NPC: (this is an example)

    /npc_add
    /npc name Fookey
    /npc enable true
    /npc radius 20
    /npc invulnerable false
    /npc hostile true
    /npc lootable true
    /npc kit Kit_Npc
    /npc respawn true 60
    /npc speed 5
    /npc_end

    when you write /npc_list, you find a list of all NPC

    if you want to clone a NPC you need to use the ID
    for example if your NPC is 288625703 you need to write

    /npc_add 288625703
    you can stay far from the NPC 288625703
    where you are, here respown a new NPC with the same characteristics of the NPC 288625703

    got some help but this seems to stop him from attacking fire pits and cactus when taking damage.

    add this in
    if (!(hitinfo.Initiator is BasePlayer ) || !(hitinfo.Initiator is BaseEntity)) return;

    Code:
            void OnEntityTakeDamage(BaseCombatEntity entity, HitInfo hitinfo)
        {
            var humanPlayer = entity.GetComponent<HumanPlayer>();
            if (!(hitinfo.Initiator is BasePlayer ) || !(hitinfo.Initiator is BaseEntity)) return;
            if (humanPlayer != null)
            {
                if (hitinfo.Initiator is BaseCombatEntity && !(hitinfo.Initiator is Barricade) && humanPlayer.info.defend) humanPlayer.StartAttackingEntity((BaseCombatEntity)hitinfo.Initiator);
                if (humanPlayer.info.message_hurt != null && humanPlayer.info.message_hurt.Count != 0)
                {
                    if (hitinfo.InitiatorPlayer != null)
                        SendMessage(humanPlayer, hitinfo.InitiatorPlayer, GetRandomMessage(humanPlayer.info.message_hurt));
                }
                Interface.Oxide.CallHook("OnHitNPC", entity.GetComponent<BaseCombatEntity>(), hitinfo);
                if (humanPlayer.info.invulnerability)
                {
                    hitinfo.damageTypes = new DamageTypeList();
                    hitinfo.DoHitEffects = false;
                    hitinfo.HitMaterial = 0;
                }
                else
                    humanPlayer.protection.Scale(hitinfo.damageTypes);
            }
        }
