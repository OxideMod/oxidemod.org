    FeedMyBeetus said: ↑

    You should check the GUIShop thread for that information. You need to set the NPC_ID found in /npc_list as the Shop ID similar to the example below. Once that is setup then you simply walk up to the NPC and push E or w/e you mapped your interact key as.

    Code:
    "5498734": {
"buy": [
"Assault Rifle",
"Bolt Action Rifle"
],
"description": "You currently have {0} coins to spend in this weapons shop",
"name": "Weaponsmith Shop",
"sell": [
"Assault Rifle",
"Bolt Action Rifle"
]
}
