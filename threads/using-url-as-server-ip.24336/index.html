  8. NT39

    Ok I figured out my problem. Apparently it's not possible to connect through a URL if it redirects the port. You cannot do client.connect rust.com even if rust.com redirects to WAN: port. You have to have rust.com redirect to WAN and then in the console add the port. Such as client.connect rust.com: port.

    You can however connect directly to the RustIO using the rust.com that has the built-in port redirect. Type in your URL into the search bar and it goes directly to your server. Granted you have RustIO installed.
