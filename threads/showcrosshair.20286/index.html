    Marat 2 submitted a new resource:

    ShowCrosshair - Shows a crosshair on the screen

    The plugin shows a crosshair on the screen consisting of icons which can be changed as desired by editing the URL in the config file. This plugin was made for my server and not passed tionless testing.

    Default chat command and console command is crosshair, but you can change it in the config file.

    Chat Commands:
    /crosshair     - enable/disable the crosshair
    /showmenu - opens crosshair selection menu ( degault keyBind f5 )

    Console command:
    crosshair     -...
