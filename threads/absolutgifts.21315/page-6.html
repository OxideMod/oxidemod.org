    So this is how it works.. I double checked the code..

    When a player first joins your server it creates a profile in AbsolutGifts for them. At that moment it sets their "reset" time to whatever is in your config. Once that time is up it will once again set their "reset" time to whatever is in the config.

    Only thing I can guess is the players initialized when the time in the config was still set at default 1 day... Also make sure you are reloading the plugin after making config changes.

    Beyond that I have no idea because I am not seeing anything in the code that says .. after 1 day reset players.. it is instead all based on the config days.
