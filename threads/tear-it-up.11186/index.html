  10. Deicide666ra

    Just a fair warning here... http://oxidemod.org/plugins/moegicbox.1296/ had this feature as a beta test on my server for a couple hours...... I removed it because it was extremely prone to abuse. Players would make bullets, convert them to blueprints (they are so cheap!) and then just recycle the blueprints for quasi-infinite fragments.

    I guess one could address this (like, have stackable items give back amount / stackable.... but still, there are a lot of extremely cheap items out there that don't stack which could be abused in the same way (lantern just to name one, lots of clothing items, etc).
     
    Deicide666ra, Aug 22, 2015
