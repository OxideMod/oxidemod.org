  7. TheRealDannyyy

    Hello there!
    Thanks for this amazing plugin, it works just as expected.

    I'd like to suggest some minor features that you might want to add in the future:
    "Pre-Warning" messages when radiation is about get turned ON or OFF.
    So basically the user has to set a value in minutes when the "pre-warning" messages will be triggered.
    For example the value is set at "5", that means that the "pre-warning" message will be shown 5 minutes before radiation gets turned ON/OFF.

    It would look like this or something equal, really depends on how you would code it:
    pre.radiationon.warning: 2
    pre.radiationoff.warning: 4

    I thought a feature like this would be nice since the most of the players go inside an irradiated area while it is off and mostly the radiation tends to start again in the middle of looting/exploring. So a warning message would make it possible for players to prepare themselves and for example go down from the dome in time without any risk of getting damage by radiation.

    Regards,
    TheRealDannyyy

    Code:
                    {
                    ConVar.Server.radiation = false;
                    foreach (var zone in RadiationZones)
                        zone.Deactivate();
                    if (configData.Options.Broadcast_Timers)                   
                        MessageAllPlayers(lang.GetMessage("RadsOffMsg", this), offtime);
                   
                    OffTimer = offtime * 60;
                    timer.Repeat(1, OffTimer, () =>
                    {
                        OffTimer--;
                        if (OffTimer == 0)
                        {
                            foreach (var zone in RadiationZones)
                                zone.Reactivate();
                            ConVar.Server.radiation = true;
                            if (configData.Options.Broadcast_Timers)                           
                                MessageAllPlayers(lang.GetMessage("RadsOnMsg", this), ontime);
                           
                            StartRadTimers();
                        }
