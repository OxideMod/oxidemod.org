  3. Mughisi

    Lutini said: ↑
    There is an opportunity to do so when you write "quit" in the console, then worked to save game?
    Click to expand...
    You can achieve this by adding this adding this to the end of the plugin:
    Code:
    function PLUGIN:OnRunCommand( arg )
-- Check if the command is being send from an in-game player.
if ( not arg ) then return end
if ( not arg.cmd ) then return end
if ( not arg.cmd.name ) then return end
-- Check if the quit command was used.
if arg.cmd.name == "quit" then
print( self:SaveAll() )
print("Shutting down...")
timer.Once(5, function() global.ConsoleGlobal.quit( arg ) end )
return false
end
end
    This will basically intercept the built-in quit command, cancel it and save the world. And after 5 seconds it will terminate the server like the command quit should.
