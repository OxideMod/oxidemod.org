    Teleport Home - Provides the ability for multiple home teleport locations

    Allows players to set home locations for teleporting. The number of locations is defined within the config file;

    default config;

    Code:
    {
  "chatname": "SERVER",
  "homeamount": 2
}
    More work to come but this is a starter too allow the setting of home locations.

    Coming soon - cooldown timers, damage detection, teleport delays and more
