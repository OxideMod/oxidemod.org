  18. chadomat

    I'm not familiar with this plugin, but according to the overview there are two options.

    Chatcommand:
    /playercounter left/middle/right - Repositions the player counter to the left, middle or right part of the screen. The position is unique per player and persists through restarts/reloads

    Config option:
    defaultPosition = "left" - The default position of the player counter. right/left/middle. This option is not retroactive, meaning that it will only apply to new users, not the ones currently on the server. If you want this to apply to the current users as well. 1. unload the plugin. 2. change the default position in the config. 3. delete the plugins data file. 4. load the plugin

    Do you have tried this?
