    Hey Reneb, Not sure if its just me being me haha.
    But when I start the server it overwrites the config file back to default
    [DOUBLEPOST=1422926701][/DOUBLEPOST]@xillium59

    Put the plugin into your oxide/plugin folder.
    Go to oxide/config and open "ItemsManager.json"

    Do a search for wood or stone etc and you will see under it a line that says Stackable. The number after this is the stack limit.
    So for wood it looks like this:
    Code:
      "514": {
      "displayname": "Wood",
      "shortname": "wood",
      "category": "Resources",
      "description": "Wood. Made from trees.",
      "itemid": "514",
      "worldprefab": "items/generic_world",
      "icon_main": "1386",
      "stackable": "1000",
      "canBeHeld": "False"
    }
    I have my stacks set to 10000 for now. If your still not sure I can upload my config or make changes to suit you.
