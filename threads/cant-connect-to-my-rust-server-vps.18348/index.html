  13. Mr. Tea

    Percy黎鈴 said: ↑
    I need turn off Firewall to open all port so Rust server can run ?
    Click to expand...
    You do not need to turn off the firewall since that would make your VPS insecure, you simply need to open 28015 UDP port on the firewall.

    Go to start menu and type 'Windows Firewall with Advance Security" and open it. It should bring up a window which will allow you to to open ports. Click on 'Inbound Rules' on the left and then click 'New Rule' on the right hand side.
    Select 'Port' and click Next.
    Then select 'UDP' and in the text box type '28015' and click Next.
    Click Next again and finally give your new rule a relevant name such as 'Rust Server'

    Then do exactly the same thing for Outbound Rules EXCEPT when you get to the 'Action' section make sure to select 'Allow the connection' since by default it is set to 'Block the connection'.

    That is how you open your port to allow Rust to communicate through your firewall. Hope this helps you bud.
