    Teleporter - Basic player-to-player and player-to-location teleportation

    Teleporter is a micro-plugin that I put together per request. It currently features player-to-player and player-to-location teleportation for admin only. I'll be expanding on it as I go along to add permissions, messages, a configuration, etc.

    Chat Commands
    • /tp <player> <target>
      Teleports player to another player.
    • /tpl <x> <y> <z>
      Teleports player to the specified coordinates
