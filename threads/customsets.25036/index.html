    CustomSets - A Set Creation and Management System with Built in Purchasing

    SPECIAL THANKS TO SKINGAMES FOR THE UI WORK!

    CustomSets is a labor of love for me. I always envisioned a Kits-style plugin with built in purchasing (to eliminate kits in ServerRewards or GUIShop or any of the other numerous workarounds) and skin support that was 100% UI based and that is exactly what this is.

    This plugin allows admins to create Sets which include items for the belt, wear, and main containers. Items for the belt are saved in their exact order and placed in that exact spot...
