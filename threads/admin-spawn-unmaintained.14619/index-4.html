    insane8472 submitted a new resource:

    Remove Admin Warning - Remove message shown when administrators spawn items.

    Simple plugin to remove the warning message when admins spawn items. Just install and let it do its thing. As of right now, this is just a simple plugin I put together for my server.

    Only note is that when spawning a blueprint, it will create the blueprint for the item in your inventory instead of learning it. Learn it yourself or give it away.

    Suggestions always welcome. Be on the lookout for LaserHydra's version!
