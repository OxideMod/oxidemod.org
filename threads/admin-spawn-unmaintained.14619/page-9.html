  1. insane8472

    insane8472 updated Admin Spawn with a new update entry:

    1.0.9

    • Added setting Log. When true, all spawns will be logged to "oxide/logs/AdminSpawn.txt".
    • Blacklist setting is now false by default.
    • Added permission "adminspawn.hide". Players or groups with this permission will not trigger a chat message or message for players with permission "adminspawn.warn". Spawns are still logged.
    • Messages will now show item long names instead of short names.
