    Okay, I'm going to try and address all of the issues above.

    1. "I'm finding when I type </sl globalmode all>. It does nothing. All that I get is this message {Changed (0) Lights To All Mode}. I want all lights to respond to players and Heli."
    • This feature is working properly. If it says that zero lights were changed, then that means it didn't find any lights in the data that you owned.

    2. "When you say; It reverts to a security light. What EXACTLY are you saying????"
    • I assume you are talking about when you use the /sl remove command. This means that the light will behave the exact way that the base games search light behaves.
    3. "Does it follow the owner for security at night. Or do they just come on in the direction they are pointed. Can you be more Precise in your interpretation and meaning of "Security Light"?"
    • Please review the General Information section in the overview. (Security Lights for Rust | Oxide)
    4. "I've read that this does not work and that you have to do each individual one. ("Auto Convert": false,). Is it true this does not automatically make them all searchlights/security. And follow the owner?? Do we have to do each and everyone? One at a time?? Anyways it's not working for me."
    • The "Auto Convert" feature is working properly. I have not heard of anyone having issues with the auto convert feature, so I am not sure where you've read this. I see that you wrote ""Auto Convert": false" if this is how the setting is set in your config, that is your issue. Set it to true.
    • As far as following the owner, the security lights only follow the owner in lightshow mode.
    • You should only need to convert individual security lights if the "Auto Convert" setting is set to false, otherwise, they should be automatically converted to security lights.
    5. "Is the the /sl globalmode all that is wrong? What am I doing wrong? Please help !!!!"
    • Like I said in number one, this is working properly. How are you attempting to use this command, and how many security lights do you own?
