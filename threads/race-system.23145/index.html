    D-Kay submitted a new resource:

    Race System - Add races to your server

    This plugin will add races to the server.
    Players can choose one of the races and when they die they can choose if they want to change their race.

    Permissions:
    RaceSystem.Show - Allows the use of the /rsplayer command.
    RaceSystem.Modify - Allows the use of the /rsreset, /rsrace, /rsappearance and /rstime commands.

    Commands:
    /rshelp     - Shows a list of the available commands.
    /rschange - Activates the race change dialogue.
