  7. D-Kay

    The permission part of the race system is the permission group needed to be able to select that race, not the permissions it will give to the player. Everything else is possible.

    Example with 5 races:
    You create 2 races with no level requirement and 3 with a level requirement of 20.
    You add 5 kits and or each kit add a race requirement for either of the races that does not have a kit yet
