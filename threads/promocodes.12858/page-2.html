  15. LaserHydra

    LaserHydra updated Promocodes with a new update entry:

    2.0.0

    Please delete your configfile before updating!

    * rewritten
    - removed permission granting
    + added command running

    Commands:
    • /redeem <code>

    Configfile:
    Code:
    {
"Promocodes": [
    {
      "availableCodes": [
        // 10 random codes
      ],
      "command": "oxide.usergroup add {steamid} vip"
    }
  ]
}
    in "command" you can use following tags:
    • {steamid} represents the players steamID
    • {name} represents the players name
