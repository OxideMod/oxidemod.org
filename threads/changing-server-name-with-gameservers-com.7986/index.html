  2. Wulf Community Admin

    You would need to change it via your host's control panel, or the cfg/server.cfg file if they set it with that instead. Rust has a bug where it can only be set in one or the other, not both, as the cfg/server.cfg file will not be properly used if the same settings are set via the command-line.
     
    Wulf, Apr 26, 2015
