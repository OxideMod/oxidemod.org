  6. Ankawi

    @DerRene The plugin Name Manager allows you put place a maximum name length required, so if you place it at 20 and say you give yourself permission to enter by bypassing the checks, no one else will be able to join your server unless you grant them the permissions as well. So it can also be used as a whitelist, but in a unorthodox way. You can also use the lang for custom messages for when people get kicked (the reason). If there was a way to input the password, one way could be by having it required with the player name, and if his name contains the password, then he is allowed connection, which is possible as well.
