    Via @bawNg

    Unparking cores has absolutely nothing to do with what cores a process uses. The Unity 5 version which Rust is currently using can use up to 2 full physical CPU cores on a heavily loaded server.

    Other Unity-based games such as Hurtworld, which use an older Unity 5 version, can only one 1 core.
