  2. Boa Constrictor

    Don't know of a manager plugin yet, but there is a command available and 3 variables. Discovered via find xmas @ console. xmas.refill spawns the gift packages and stocking refills. You might be able to use another loot manager plugin to make the presents/stockings or other xmas items spawns in crates/barrels. The xmas items short names and such are listed here.

    Code:
    > find xmas
Variables:
xmas.enabled             no description (True)
xmas.spawnrange          no description (50)
xmas.giftsperplayer      no description (2)

Commands:
xmas.refill( void ) no description
     
    Boa Constrictor, Jan 6, 2016
