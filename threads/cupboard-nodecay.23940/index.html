  1. Dubz

    Dubz submitted a new resource:

    Cupboard NoDecay - Prevents decay on items under a cupboard radius.

    Prevents decay on any entity that is within a tool cupboard radius. Also does a check to see if the owner of the tool cupboard is still authed, if they are not then entities will decay, can be enabled/disabled via config file.
