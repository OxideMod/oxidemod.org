    Code:
    #!/bin/sh
clear
while :
do
    echo "Starting server...\n"
    exec ./RustDedicated -batchmode -nographics \
    -server.ip 0.0.0.0 \
    -server.port 28015 \
    -rcon.ip 0.0.0.0 \
    -rcon.port 28016 \
    -rcon.password "changeme" \
    -server.maxplayers 10 \
    -server.hostname "//" \
    -server.identity "my_server_identity" \
    -server.level "Procedural Map" \
    -server.seed 12345 \
    -server.worldsize 4000 \
    -server.saveinterval 300 \
    -server.globalchat true \
    -server.description "//" \
    -server.headerimage "//" \
    -server.url "//"
    echo.
    echo "\nRestarting server...\n"
    echo.
    goto start
done

  5. Xsander

    if you use LGSM server script and Timed Execute, just edit your crontab with "crontab -e"
    For example: i'm using restarts at 00:00, 08:00 and 16:00
    Timed execute config:
    "07:59:30": "global.restart 60",
    "15:59:30": "global.restart 60",
    "23:59:30": "global.restart 60",
    so i'm adding to crontab this:
    01 00 * * * /home/path_to_rust_server/rustserver restart #this makes restart at 00:01 after server shutdown
    01 08 * * * /home/path_to_rust_server/rustserver restart #this makes restart at 08:01 after server shutdown
    01 16 * * * /home/path_to_rust_server/rustserver restart #this makes restart at 16:01 after server shutdown


    this means server shutdowns at 08:00:30 and restarts in 08:01:00 , same for 16:00 and 00:00

    By the way, LGSM script have very good feature - monitor. it monitors your server and let him up if it's crashed.
    You can use it by crontab very similar:
    */5 * * * * /home/path_to_rust_server/rustserver monitor > /dev/null 2>&1

    this will monitor your server every 5 minutes without logs and mail to your user
