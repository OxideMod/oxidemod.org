    Name Colors is not compatible with RanksAndTitles, you cannot use both, my plugin builds its own chats and handles it itself. Name colors also catches the player chat and adds a color to the name, so since chat is caught twice, both plugins will send their versions of the chat. (which is why you get duplicate chats)

    EDIT: however this plugin supports changing name colors (applies to all users) and staff colors (applies to all staff) along with changing chat color.

    the first ranks should be able to be deleted using the command but might not be right now I have to look into it. you can simply edit these ranks though inside the config file in order to change them though, just go to your server files, the config folder inside oxide, and it will be in there labeled RanksAndTitles.json, look for main and the ranks are right there to be edited or new ones added.

    as for /rt create this command is still a bit fragile but yes you should be able to use a negative number for karma needed however do not use a negative number for karma given. this is very important! if karma given is negative the plugin will not perform the math correctly or cause errors.
    This is another feature that is going to be getting improvements hopefully soon in the next large update.
