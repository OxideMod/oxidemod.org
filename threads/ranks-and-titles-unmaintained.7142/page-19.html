    on the players who have the "Civilian" title type /rt remove playername this will re process them through the ranks function with an empty title instead of a set title (It's probably set in data which is why it wouldn't change). It's possible these titles slipped through the cracks on player login. And once the player has a title if it does not match any of the current titles it is considered a "custom set" title by the plugin and it will not touch it.

    so once again use /rt remove playername on the players with the civilian names. Or (If this is a recently new install for your server) you can make sure your config is setup how you want it and do a /rt clear to wipe the entire data file and have everyone start from scratch again However this will DELETE all current data for ranks. everyones kills, deaths, kdrs, etc.. will be reset back to 0, and data will need to be rebuilt as players play the game. (It should handle this automatically)
