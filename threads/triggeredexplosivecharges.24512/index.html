  19. EnigmaticDragon

    Wombat said: ↑
    "TRIGGERED C4 | Disable beeping sound after duration (minimum: 0; infinite: -1) [seconds]": 1
    it wont disable the sound on anything under 10 seconds have i done it right
    Click to expand...
    Never change the part on the left side of the colon ("TRIGGERED C4 | Disable beeping sound after duration (minimum: 0; infinite: -1) [seconds]"), it's just the description, only change the value on the right.
    If you set the value between 0-4 it will automatically take the default value (10), because 5 is the minimum. -1 will let it beep infinitely. All values equal or greater 5 will let it beep that many seconds.
     
    EnigmaticDragon, Mar 21, 2017
