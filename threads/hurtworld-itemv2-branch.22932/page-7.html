    Probably not until the official Oxide for V2 is released.... which might not be until V2 itself is released officially. Although personally I think it would be a good idea for Oxide for Hurtworld V2 to be released before it's complete... it is understandable why it might take until the actual release.... as keeping Oxide updated for one complete version and one test version might become a bit of a pain. We can hope though

    But yeah so far there has been no reason to expect that the oxide platform and mods would be updated for V2 when it's still in dev test phase. Some admins, mostly mod creators themselves - have managed to go into the core oxide files and update everything them. This is tedious, however - as they need to update them manually with each V2 patch, which is every 1 to 2 weeks. This is pretty much why I have not even considered switching my server over to V2 yet. Once the official modding platform is created and updated for V2 it will make that decisions much easier.

    For now I'd recommend waiting on it unless you already know how to update Oxide files yourself. But also @Wulf it really would be helpful to have an official Oxide build for V2 within a month or so. I think in another 3 or 4 weeks there will be many more admins wanting to convert. Seems the real meat of V2 is close to being put into the dev test.
