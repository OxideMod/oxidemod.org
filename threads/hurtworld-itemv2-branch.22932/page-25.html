  17. CaptainNebulous

    Not sure if this will help, but I'll try.

    If you are running V2 it has a memory leak currenty, in which nothing short of more frequent restarts will help. With a memory leak, the more memory you add the more memory gets eaten up. Although honestly Arma 3 is the only game I've ever seen it make a difference between 6GB and 8GB of ram on a server anyway... and that game is much more memory intensive. Actually that's the only game I've noticed more than 6gb be useful at all.... but it's also the most performance intensive game I have any experience at all running a server on. I'd also recommend your VPS have at least 750kb/second upload speed per player, but at least 300 kb/sec per player is probably required for minimal performance. Also the minimum required specs listed for a hurtworld V1 server state that you need a 4 core processor of at least 2 ghz. So I don't think 2 V cores will be enough in the long run for either version.
