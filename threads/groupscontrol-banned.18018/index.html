    GroupsControl - Provides groups control commands and API.

    With the help of a script you can put the player in a group for a specific time.
    For example, /gc add vip jimless 1800
    Place the player jimless, in the vip group for half an hour. The time is specified in seconds.

    For more help use syntax: /gc add or /gc remove or /gc show.
    /gc add {group} {name|steamid} [{seconds}]
    /gc remove {group} {name|steamid}
    /gc show group {group} or /gc show user {name|steamid}

    API
    Code:
    bool GCAddUserGroup(string steamId, string group,...
