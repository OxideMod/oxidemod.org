    VVoid submitted a new resource:

    Limited Ladders - Disallows placement of ladders where building is blocked.

    Simply removes bypass of building permission from ladders and they cannot be placed when building is blocked.

    Configuration is not required, settings being applied upon server loading.

  11. VVoid

    VVoid updated Limited Ladders with a new update entry:

    Ability to restrict ladders only on constructions

    • Added ResourceId.
    • Added config to restrict ladders only on constructions (thx 2 maxon 4 idea).
      Ladder which are built on the construction where building is blocked will be immediately destroyed and will return to the owner.
      So it's possible to put ladders only on the rocks, on the ice etc when aren't permitted to build.
      This setting is not tested enough and also off by default.
