    Rust has no native chat commands, they are all from Rust. The "god true/false" setting from Rust is only if you are set with ownerid on your Rust server, and it only works through your F1 console.

    Also, if plugin commands are not working, that's generally a sign that it's either not installed in the right location or the plugin is not loaded. I'd suggest checking your oxide/logs to verify or for errors.
