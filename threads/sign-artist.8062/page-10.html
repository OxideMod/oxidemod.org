  19. NobodyFTW

    Permissions:
    • sil_url -- To use /sil <URL>
    • sil_storage -- To use /sil s <num>
    • sil_owner -- To ignore owner check
    • sil_cd -- To ignore cooldown check
    Sign Artist plugin overview said:
    You can grant a user permission by using:
    oxide.grant user <username> <permission>

    To create a group:
    oxide.group add <groupname>

    To assign permission to a group:
    oxide.grant group <groupname> <permission>

    To add users to a group:
    oxide.usergroup add <username> <groupname>


    If you want to grant permission to all users, then you can use this console command:
    oxide.grant group player sil_url
    To all admins:
    oxide.grant group admin sil_url
    Click to expand...
    You must set yourself these permission by following the examples.
