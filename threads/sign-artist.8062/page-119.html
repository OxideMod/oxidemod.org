    Click to expand...
    use RustAdmin RCON Tool to give the permission to the groups
    oxide.grant group default signartist.url
    oxide.grant group default signartist.text
    oxide.grant group moderator signartist.url
    oxide.grant group admin signartist.url
    oxide.grant group admin signartist.raw
    oxide.grant group admin signartist.raw
    oxide.grant group admin signartist.restoreall
    oxide.grant group admin signartist.ignorecd
    oxide.grant group admin signartist.ignoreowner
    oxide.grant group admin signartist.restore

    and if you still get that message if you test all then you are not an admin
