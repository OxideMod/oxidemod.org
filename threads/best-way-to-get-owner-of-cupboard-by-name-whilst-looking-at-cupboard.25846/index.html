  2. serezhadelaet

    Hello.
    For example, you can use Covalence.
    Code:
    var name = covalence.Players.FindPlayer(steamID)?.Name;
    But if you have big covalence data (a lot of people played on server) you will get some delay and freeze server.
    Best way: every cupboard spawn must be logged into data file, for example:
    cupboard_entity_ID { player.userID : player.displayName }
