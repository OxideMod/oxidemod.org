    2.0.1

    You can now have rcon commands ran when a player uses a CustomChatCommand - you will need to add the "RconCmd" field to your config (or re-generate it), for example:

    Code:
        {
      "Command": "noclip",
      "Messages": [
        "NoClip toggled."
      ],
      "Permission": "customchatcommands.admin",
      "ConsoleCmd": [
        "noclip"
      ],
      "UserID": 0,
      "Broadcast": false,
      "RconCmd": [
        "say {player.name} / {player.id} has just run the /noclip command!"...
