    Nope. If they disconnect I am resetting the flag for the user within the OnPlayerDisconnect function

    if (self.DisplayedWelcome[tmpuser] ~= nil) then
    self.DisplayedWelcome[tmpuser] = nil
    end

    This way if they die during the session they are not going to get the welcome everytime they wake up and if they disconnect, they will get it. No need to restart the server. Thats how its working on my test server anyway
