    Fixed it by updating Wine! Apparently, Ubuntu 12.04 stopped getting package updates at Wine 1.7.18 and the DLL needed by the compiler didn't start getting packaged with Wine until 1.7.30. Added an unstable PPA and updated to 1.8. All is well.

    Code:
    sudo add-apt-repository ppa:ricotz/unstable
sudo apt-get install wine
