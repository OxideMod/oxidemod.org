  2. Wulf Community Admin

    Oxide's commands nave no relation to RCON. The command changes are only related to the commands used to manage plugins and their permissions.

    The first thing you'd want to check is to see how your RCON is setup, if you are using Oxide's RCON protocol (same as Rust's legacy protocol, ie. rcon.web 0), and what your RCON client supports. None of this has changed though, so you may want to check your setup.
