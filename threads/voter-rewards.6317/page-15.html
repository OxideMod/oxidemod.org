    Ok, after reading through all the previous posts I have fixed it based on a post by @recon . He got round the period issue being dropped to a seperate line by using a hash symbol to add the underscore. Problem is not all the names are interchangeable with '_' underscores and '.' full points.

    I changed this line in his code.
    Code:
    SendMessage(player, msg:gsub("{REWARD}", table.concat(names, ", ")):gsub("#", "."), "[ID " .. tostring(i) .. "]" )
    Using the list here (still current)
    Docs/item_list.md at master · OxideMod/Docs · GitHub

    I now have this working. My config looks like this.
    Code:
      "Settings": {
    "DataFile": "VoterData",
    "Rewards": [
      {
        "price": 1,
        "reward": {
          "supply#signal": 1
        }
      },
      {
        "price": 2,
        "reward": {
          "explosive#timed": 3
        }
      },
      {
        "price": 4,
        "reward": {
          "autoturret": 1
        }
      }
    ],
    And my server set up looks like this.
    Code:
