  1. recon

    I did a quick and dirty fix for myself. Thought I would share it with you guys.
    So instead of using dots (.) in item shorts names, you have to use hashtags (#).

    Example:
    Code:
    "Settings": {
    "DataFile": "VoterData",
    "Rewards": [
      {
        "price": 1,
        "reward": {
          "ammo#pistol#hv": 60,
          "smg#thompson": 1
        }
      },
      {
        "price": 2,
        "reward": {
          "blueprint_fragment": 600
        }
      },
      {
        "price": 6,
        "reward": {
          "supply#signal": 1
        }
      }
    ],
    Use my version and this list for the SHORTNAMES and just replace dots with hashtags Click me
