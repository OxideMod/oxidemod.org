    Jump Block Stacking Blocker - Stops players from building those block towers by just jumping and building underneath

    Stops players from building those towers made of blocks by just jumping and building underneath themselves.
    The plugin will basically stop everyone from building while jumping, shouldn't be possible anyway
    And under certain circumstances it is possible to put a buildingblock on top of yourself so they could still do this so the plugin also stops players from building too close near themselves.

  4. cenk

    It was really necessary thanks to Mughisi for fix game bugs.

    without this plugin;
    1. players can raid other houses easily;
    - They can build foundation
    - then jump on it and build wooden block
    - and then jump and build one more block on it
    - and after 10-20try you will be top of the block tower
    - then build floor for run and jump to the other houses
    (tool cupboard have limited distance for avoid building - if you make tower enough long you can bypass cupboard for build floor on block tower for close to the house and run/jump to the roof)

    2. players can build un-raidable houses
    - make first floor
    - dont make stair inside of house
    - just use blocks for reach to the 2nd floor then destroy it
    - put your tool cupboard on 2nd floor
    - Since the other players do not able to build in your house if they destroy the wall or door they can access to the first floor only and since there is no stair they can not reach to the upper floors
    - owner of house can reach upper stairs by sleeping bag or build block and jump on it
    - actually it will not %100 prevent unraidable house thing becasue, now instead of use 1x1 blocks they will make stair system or 1x2 block stairs for further destroy them to prevent access to the 2nd floor but anyway it will just make the things hardener.
    (If the Mughisi create one more plugin for non-removable non-destroyable block + stairs then this problem will be fixed too)

    by the way Mughisi; is this same version you publish in forum or modified version?
