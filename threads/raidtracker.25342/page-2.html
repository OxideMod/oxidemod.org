  14. nivex

    nivex updated Raid Tracker with a new update entry:

    0.1.11

    The following functionality applies to newly logged explosions starting with this version (0.1.11)

    Added functionality for players to utilize the plugin. Players can only see explosions done to entities which they own or did own. They cannot see their own explosions.

    This functionality is DISABLED by default and requires special permissions to use.

    To grant a permission to a player:
    grant user nivex raidtracker.use

    To grant a permission to all players:
    grant group default raidtracker.use...
