    The problem is that plugins are generally moderated to make sure there isn't malicious code or backdoors in them before approval, which requires them to go through the Plugins section. Posting random plugins around all over the forum causes loads of extra work and headaches for the moderator team. A plugin can be rested privately or released in an alpha state, but it should still be posted under the Plugins section if it is intended to be shared through our site. It's not about it being supported or not, it's about it being checked and posted in a central location, and not scattered around.
