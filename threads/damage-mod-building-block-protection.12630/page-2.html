    1. I've always understood that "true" is protection. If you set it to false you are unprotected.
    Look at it like a question.... "I want triangle foundation protected? true/false = yes/no."

    2.No. You can build a castle unprotected and then change your cfg settings to True and it will be protected.....AFTER you restart the server.
    You must reboot your server after making changes to the CFG file. At least this has been my experience.

    3. I believe the protection is "server wide" not player to player-permission based; if I understand the question correctly. Per the way you set up the cfg file, building blocks are either protected or they are not. Admins can still destroy any players blocks with their hammer but other then that I have to use r-remove plugin so players can destroy their own bases. Tool Cupboards still protect other players from removing others building blocks in this manner.

    Note: Unless its a conflict in the plugins I use, I can not remove a water catcher at all even when using /remove admin. So I have to leave this item as false so that I can delete them with "C4 explosive" if someone needs it moved or removed.

    I hope this helps answer your question in some way; good luck.

    Doors are not protected by this mod and they are still subject to take damage. Only Door Ways (wall around the door) are protected with this.
    I've listed below what the differences between DM Building Blocks and DM Deployables are below.

    So unless you have doors protected with DM Deployables they can still be destroyed by c4, rockets, etc. and the walls in your maze will still be safe.

    Of course this limits the way players can raid a base, forces them to destroy doors only but if you take the protection off the bases you also take it off the maze. Maybe someone else knows of another way that I don't. I run a PVE server with a lot of crazy animals eating players so I haven't messed with a config for a pvp server using this mod.

    This plug in, Building Blocks, is only for protecting:
    Doorway
    Floor
    Floor Frame
    Floor Triangle
    Foundation
    Foundation Steps
    Foundation Triangle
    Low Wall
    Pillar
    Roof
    Stairs L Shaped
    Stairs U Shaped
    Wall
    Wall Frame
    Window Wall

    And Damage Mod Deployables protects:
    Auto Turret
    Barricade
    Base Oven
    Boxes
    Chain Link
    Double Doors
    External Gates
    External Walls
    Floor Grill
    Floor Ladder
    Mining Quarry
    Prison Cell
    Repair Bench
    Research Table
    Shelves
    Shop Front
    Shutters
    Signage
    Single Doors
    Sleeping Bag
    Tool Cupboard
    Water Catcher
    Windows
