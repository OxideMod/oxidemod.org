    Hey....
    There is an easyer way to get all permissions with out have to type all permissions by them self.

    1. group add owner (This will create a group called "owner")
    2. usergroup add "Your name" owner (This will add user to group owner)
    3. grant group owner * (This * give you all permissions that exist on the server, remember if you add a new plugin make sure to type grant group owner * every time you add an plugin that needs permission.)

    I hope this help you!

    [IMPORTANT MUST READ]
    The * give acces to F1 spawn menu and acces to edit all settings on the server!
    Tip: Dont give anyone else than you self the *

    There is an "admin" group automatically created by Oxide, and you are automatically added to it if you have auth level 2 with Rust. There isn't a need to create another group when it handles it automatically.

    I would also discourage blindly granting ALL permissions, as sometimes permissions in plugins do negative things that you do not want to have most of the time.

    Oxide's permissions also do not have anything to do with the F1 item menu, that's all Rust. The only time permissions would control that is if you are using a plugin such as Admin Spawn that adds permissions for it.
