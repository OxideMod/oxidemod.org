    Protocol: 2036.153.1

    EDIT 2 : OK, I found the issue. Oxide doesn't load plugins from the same place as before. Before the update, it was looking in the "\\rustds\server\server_name\oxide\plugins" folder, now it is in "\\rustds\oxide\plugins\". Changing this setting suddenly is not nice but at least I found out.
