  2. Wulf Community Admin

    Your server should never be updated by updating unless either you or your host wipes it, or a Rust forced wipe happens. Simply updating does not cause a wipe. Oxide is a mod, so whenever you run "Server Update", it will uninstall Oxide because it is validating and restoring the server files to originals. Every time you run an update, you need to install Oxide again. Installing Oxide will also not wipe your map.
     
    Wulf, Sep 22, 2016
