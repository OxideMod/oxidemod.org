    For example, if I was to go with this plugin this would be my config.
    Code:
    {
  "EnabledRealTime-Timer": false,
  "EnableTimerOnce": false,
  "EnableTimerRepeat": true,
  "RealTime-Timer": {
    "16:00:00": "say 'The gate for the event is open!'",
    "16:30:00": "say 'The gate for the event just closed'",
    "17:00:00": "say 'Restart in 1 HOUR'",
    "18:00:00": "say 'The server is restarting NOW.'"
  },
  "TimerOnce": {
    "reset.oncetimer": 181,
    "say 'Dont forget to like our fanpage!'": 60,
    "say 'Follow us on Twitter!'": 120,
    "say 'You can donate via PayPal!'": 180
  },
  "TimerRepeat": {
    "gather.rate dispenser Wood 2": 1200,
    "gather.rate dispenser Wood 1": 2400
  }
}
    Now with that config as you can see by the "TimerRepeat" if they would run one after the other then that would work but they run together.
