  1. Jake Rich

    Jake Rich submitted a new resource:

    No Despawning - Items will stay around for a long time, without impacting server performance.

    WARNING:

    Do NOT use this on high rate servers. Certain high value items will persist for days on end, and they will accumulate quickly if they are too common.

    Contains improvements that make dropped items use less resources, while allowing them to persist around much longer.

    Certain items will last longer than others: burlap clothing will last only a few minutes, all the way up to an M249 lasting a whole week!

    A config file contains all the default despawn times.
