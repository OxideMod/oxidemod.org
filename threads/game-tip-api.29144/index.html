    Game Tip API - API for displaying queued gametips to players.

    Game Tip API adds a queue system for displaying game tips to players.

    Hooks:

    Queue a game tip for a specific player.
    Code:
    void ShowGameTip(BasePlayer player, string message, float duration = 5f, bool mandatory = false)
    Queue a game tip for all active players.
    Code:
    void BroadcastGameTip(string message, float duration = 5f, bool mandatory = false)
    Chat Commands:

    /hidetips – Allows a player to hide non-mandatory tips
    /showtips...
