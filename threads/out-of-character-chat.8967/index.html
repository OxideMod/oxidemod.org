  1. D-Kay

    DumbleDora submitted a new resource:

    Easy OOC Chat - players can use /o to add an OOC tag to their message - for RP servers!

    OOC is a simple mod. It adds a command for all users on a server '/o'.
    if a player types '/o their message' an (OOC) tag is automatically added in a slightly different shade of color so everyone knows it is OOC (out of character) chat.

    There is no configuration file needed, simply drop the plugin into your plugins folder of Oxide.
