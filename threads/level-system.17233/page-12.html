    Click to expand...
    The patch is ready although it will be moved to the CraftOwnership plugin as the root of the cause is there.

    As you might know, smelters produce charcoal even if there is no fuel burning. Once it gets to 1.000.000.000 charcoal it will activate the hook for Craft Finished and the last person to interact with that smelter will get XP calculated for each individual charcoal item. This takes so long to process that the server stops responding and crashes.

    I will have the patch up somewhere tomorrow.
