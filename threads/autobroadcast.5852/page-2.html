    Click to expand...
    No, you need to open a different file, you're not supposed to make any changes to the plugin files (lua, py, cs or js). If a plugin uses a datafile or a config file these will be automatically created when the plugin runs for the first time. Datafiles will be created in the folder <drive>:\path\to\rustserver\server_identity\oxide\data and config files will be saved in ......\oxide\config.
    It's in that config folder a file named auto-broadcast.json will appear and it is that file that you need to make the changes in.
