  1. Wulf Community Admin

    Taffy submitted a new resource:

    Auto Broadcast - Send global broadcast messages over a timed interval

    Automates the sending of global broadcasts messages based on a timed interval.

    Configuration file defines the chatname, broadcast interval (in seconds) and the messages to send. One message is sent at each interval, the message number is incremented following the broadcast. For example, after 600 seconds (10 minutes) the Message1 string in the config is broadcast, after a further 600 seconds Message2 string is broadcast as so on.
