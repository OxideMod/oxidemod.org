    sami37 submitted a new resource:

    BlockBugPrevent - Prevent foundation block build on another foundation

    I recently saw foundation building block bug on my server, and because I made foundation invulnerable using InvFoundation for Rust. It just detects if there is a foundation under the position you try to build on and removes new entity if it match foundation.

    With this new plugin, your player will not be able anymore to build an invulnerable house.
