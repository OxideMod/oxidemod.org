    Moderators and owners/admins can do the exact same things in terms of Rust's own commands, but what they can do via plugins varies based on each plugin and how they have auth levels or permissions setup.

    Pretty much every Rust command/setting available can be used by both.

    Click to expand...
    You won't be able to use "/give" unless you install a plugin that does so. See (Give for Rust Experimental | Oxide)

    To give yourself stuff use either F1 > Item list or use Rusty RCON client. There's also a give command in console but I dont know it out of my head.

    Edit: Console command is "inventory.giveto "player" "item" "amount"
    Example: inventory.giveto "BeefyBeast" "stones" "1000"
