  2. Mughisi

    FindByID won't work for disconnected players as it basically just searches BasePlayer.activePlayerList for a match, you should however be able to find a disconnected player, if his sleeper hasn't been killed yet, in the sleeping players list: BasePlayer.sleepingPlayerList, this should also give you a BasePlayer. Once the sleeping player has been killed he's basically removed from the server and you aren't able to grab the BasePlayer anymore.

  12. baton256

    Oxide has covalence library you could use to retrieve info on all players ever logged in rust.
    Code:
    Interface.Oxide.GetLibrary<Covalence>().Players.GetPlayer("playerSteamId");
    Here is returned object IPlayer interface with all the methods and props you can use.
    Code:
    public interface IPlayer
  {
    string Nickname { get; }

    string UniqueID { get; }

    ILivePlayer ConnectedPlayer { get; }

    bool IsBanned { get; }

    TimeSpan BanTimeRemaining { get; }

    bool HasPermission(string perm);

    void GrantPermission(string perm);

    void RevokePermission(string perm);

    bool BelongsToGroup(string groupName);

    void AddToGroup(string groupName);

    void RemoveFromGroup(string groupName);

    void Ban(string reason, TimeSpan duration);

    void Unban();
  }
