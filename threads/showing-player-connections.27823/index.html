  4. JoeSheep

    GUI Announcements for Rust | Oxide

    Config section:
    Code:
    "Public Join/Leave Announcements":{
   "Announce Only Players With Permission":true,//Enables or disables only announce the joining and leaving of players with the permission GUIAnnouncements.AnnounceJoinLeave
   "Join Banner Color":"Grey",
   "Join Enabled":false,//Enables or disables join announcements
   "Join Text":"{rank} {playername} has joined.",
   "Join Text Color":"White",
   "Leave Banner Color":"Grey",
   "Leave Enabled":false,//Enables or disables leave announcements
   "Leave Text":"{rank} {playername} has left.",
   "Leave Text Color":"White"
},
