    The Oxide permissions system doesn't have anything to do with the ownerid command from Rust. The noclip functionality comes from Rust as well, and only works if you are set properly with ownerid. Make sure you've logged out after using the ownerid command and that you actually saved it to the users.cfg file either manually while the server is offline or using the server.writecfg command.
