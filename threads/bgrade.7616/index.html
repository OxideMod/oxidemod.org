  1. AlexALX_[rus-ua]

    AlexALX_[rus-ua] submitted a new resource:

    Automatic Build Grades - Auto update grade on build to what you need (for admins).

    This plugin allow for admin automatically update building parts to any grade just after you build it (using building plan).

    Command usage:
    /bgrade - help in-game, with display current mode.
    /bgrade 1 - auto update to wood
    /bgrade 2 - auto update to stone
    /bgrade 3 - auto update to metal
    /bgrade 4 - auto update to armored
    /bgrade 0 - disable auto update

    ps this is my first public released rust plugin, and sorry for my english.
