  16. AlexALX_[rus-ua]

    nickname. said: ↑
    Really how work ru?
    Click to expand...
    There is 3 variants:
    1. Change server default language to "ru" using server command "lang ru", with this all plugins what have russian translation will have it, but english one need be translated manually in new "ru" files (what will be automatically created).
    2. Change lang file name "ru" to "en" with replace old one.
    3. Change language per player when enter to server - using "/lang ru" command in chat. This way is proper if you want create multi-language server with more that one lang, so everyone can choose own lang per player.

    Also before do any of this actions make sure you did backup of lang files, because if do something wrong - translations may be overridden and reset to default.
     
    AlexALX_[rus-ua], Jun 1, 2016
