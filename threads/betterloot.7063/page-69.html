    2.6.4

    • Hotfix for Items with NO Rarity and as reason of that not considered by the plugin
    • This is resolved by a list in the plugin main config file, and prefilled with most used items with prefilled raritys
    • The plugin uses the following numbers:

      0 = "common"
      1 = "uncommon"
      2 = "rare"
      3 = "very rare"
    New part it the config:
    Code:
      "RarityIndex": {
    "ItemOverrides": {

      "box.wooden": 0,
      "crude.oil": 1,
      "fat.animal": 0,...
