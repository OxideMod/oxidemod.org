    just edit the files in \data\BetterLoot to what you want and then reload plugin: oxide.reload BetterLoot
    edit these files below
    HeliCrate.json << Default loot tables for helicopter spawns
    LootTable.json << Default servers loot table with MAX Stacksizes for items << you need to change this if needed
    NamesList.json << All BP's (exluded defaults) AND all itemnames AND all items with their current max stacking
    SupplyDrop.json << Default loot tables for SupplyDrop spawns
