  9. lordsirslumps

    Thank you all for the replies. I was able to figure out how to add items to the Better loot plugin. In the ruste/budles/items folder there is a list of all the available items. From there you can copy and past the specific item such as the shotgun.double (double barrel shotgun) to the loottable.json file. After you paste the items in the loottable.json file reload BetterLoot. From there the new items will start to spawn in. If you cannot copy or paste the item just type it in manually. For example in the loottable.json file I added the "shotgun.double": 1, and the "explosive.satchel": 1, to the file manually and after I reloaded BetterLoot both started to spawn in crates and barrels.
    Hope this helps,
    Lordsirslumps
