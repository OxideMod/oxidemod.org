    Disable BL for barrels:

    Code:
    {
  "Barrel": {
    "barrelTypes": "loot-barrel|loot_barrel",
    "enableBarrels": true,  /// << FALSE
    ...
  },
    For crates exist the following ones:
    crate_normal
    crate_normal_2
    crate_normal_2_food
    crate_normal_2_medical
    ( trash-pile-1 )

    To exclude medical...
    Code:
    "Crate": {
    "crateTypes": "crate_normal|crate_normal_2|crate_normal_2_food",
