  14. Evilyn Corporated

    Supahoe4754 said: ↑
    Is there a way to increase the respawn rate of barrels and crates?
    Click to expand...
    Unless something has changed (!!!) since 3rd November 2016 update (Devblog 134), you can increase loot spawns by amending your server.cfg and changing the value for spawn.player_base from 100 to say 10 or lower. The second variable to amend is spawn.player_scale which by default is set to 2.

    According to DevBlog 134 - "With these default settings, with up to 100 players, nothing changes, however when players increase beyond 100, things start respawning quicker so that when it reaches 200 players, the respawn time is halved. Fingers crossed!"

    So in theory (we do this and it seems to be good) setting the spawn.player_base to a lower number will speed up loot respawns, sometimes annoyingly so if there are a lot of players on and you're an admin and trying to clear an area.

    We use this and it seems to work nicely.

    \cfg\server.cfg

    Code:
    spawn.player_base 20
spawn.player_scale 2
