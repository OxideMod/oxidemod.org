  10. FuJiCuRa

    Jenna said: ↑
    Quick question - If I blacklist an item, will that remove all existing instances of it? And/or will it prevent people who'd already obtained the item from using it going forward?
    Click to expand...
    Blacklist only will prevent it from spawning in any included container type...this overrides the setting of any loot table where it is included.
    BlackList WITH enforceBlacklist set to TRUE will remove any of those items once they are moved to ANY container.
     
    FuJiCuRa, Jul 5, 2016
