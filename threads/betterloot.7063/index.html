    dcode submitted a new resource:

    BetterLoot - A complete re-implementation of the drop system - by playrust.io.

    BetterLoot is a complete re-implementation of the drop / loot system made by playrust.io.

    Instead of using the game's default loot tables, BetterLoot overrides any barrels and crates a player loots with something (hopefully) better.

    While this usually yields a better playing experience, there is nothing super special about it: BetterLoot uses the game's default item rarities and does not touch airdrops.

  5. dcode

    This should not interfere with any other plugins as it just uses what's inside of the item definitions, regardless if these have been modified or not. It for example adapts to changed stack sizes. E.g. if you have a stack size configured of 10000, it will drop up to 2000 per stack instead of 200 (with a maxStackModifier of 0.2)
