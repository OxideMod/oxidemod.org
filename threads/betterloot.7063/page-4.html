    That happens if you increase the stack sizes too much. As is pointed out in the description, the maximum dropped stack of a stackable item is STACKSIZE (just what you set through ItemConfig for example) * MAX_STACK_MODIFIER (default: 0.2). If you do not increase Explosives STACKSIZE, you are just fine with 100*0.2 = 20.
    [DOUBLEPOST=1426191074][/DOUBLEPOST]

    This doesn't matter when you are not using the default stack sizes. The default config of BetterLoot just has maxStackModifier = 0.2, which is applied to the maximum stack size of your items, which is modified with another plugin (!). Simply do not set insane stack sizes for items like C4 or Explosives.
