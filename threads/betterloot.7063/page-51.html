    The Game has 2 trashtypes:

    Crate: Assets/bundled/Prefabs/autospawn/resource/loot/Trash-Pile-1.prefab
    Barrel: Assets/bundled/Prefabs/radtown/loot_trash.prefab

    Those two are by default included to the config in this lines by thier shortened name:

    Code:
    "barrelTypes": "loot-barrel|loot_barrel|loot_trash",
"crateTypes": "crate_normal|trash-pile",
    To exclude for example every trash type from being affected, it should look like that:
    Code:
    "barrelTypes": "loot-barrel|loot_barrel",
"crateTypes": "crate_normal",
    With this, they will not be touched by BetterLoot and are Vanilla like with food

    If you change those seetings while th game was running, then those barrels or crate, which were filled by BetterLoot, need first to be lootet or deleted

    All those types can be easy deleted by your console with no need for an extra plugin.
    Simply use the names from above and the 'del' command:
    Code:
    del Assets/bundled/Prefabs/autospawn/resource/loot/Trash-Pile-1.prefab
del Assets/bundled/Prefabs/radtown/loot_trash.prefab
    this will delete all trahpiles and all loottrash spawns, abnd the server would respawn them alone, and with default content.
