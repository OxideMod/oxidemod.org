    Code:
    #!/bin/bash
# LGSM fix_rust.sh function
# Author: Daniel Gibbs
# Website: https://gameservermanagers.com
# Description: Resolves startup issue with Rust

local commandname="FIX"
local commandaction="Fix"

# Fixes: [Raknet] Server Shutting Down (Shutting Down)
export LD_LIBRARY_PATH="${systemdir}/RustDedicated_Data/Plugins/x86_64"
    So simply create a RustDedicatedStarter.sh script file in your game server root folder, and put this in it:
    Code:
    #!/bin/bash
export LD_LIBRARY_PATH="./RustDedicated_Data/Plugins/x86_64"
./RustDedicated $@
    Make this script file executable
    Code:
    chmod +x RustDedicatedStarter.sh
    You can call this file with the URL arguments, they will be passed on, or you can remove the "$@" and just paste your start command line options.
