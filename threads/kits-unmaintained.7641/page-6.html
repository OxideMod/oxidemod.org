    You can make the permission whatever you want but a few things you need to do
    Make a new group for the new permission the permission and group name must be the same
    If you want to call the permission best player then the permission would be Kits.bestplayer and the group name would be bestplayer
    Then add the member to the new group and also add the permission to that user Kits.bestplayer
