    M@CH!N3 submitted a new resource:

    Realistic Fall Damage - Makes falling death height more realistic

    Finally a mod that will make falling more realistic and keep players from falling from heaven and not die using massive towers.

    I actually paid to have this mod created and rewrote it to be a lot cleaner and have console commands and chat commands for easier editing. I will not disclose who made the basic part of it.

    Config File:
    Code:
    {
  "MaxFallHeight": "15"  - Sets the max fall height in meters
