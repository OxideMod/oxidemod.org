  1. Wulf Community Admin

    Wulf submitted a new resource:

    NoSleepers - Disables sleepers by killing players on quit

    NoSleepers disables sleepers by killing players on quit. You can also remove the corpses and existing sleepers by changing the config.

    Configuration
    You can configure the settings and messages in the NoSleepers.json file under the server/<identity>/oxide/config directory.

    Default Configuration
    Code:
    {
  "KillExisting": true,
  "RemoveCorpses": false
}
