    VendingManager - Utility for managing vending machines

    VendingManager is a plugin which provides some tools for admins to help improve Vending Machine management. This includes the ability to save, load, and clear sales order templates (which are saved to the config file), as well as enable lock attachment again.

    Chat commands (effect is applied to vending machine being looked at):
    /vm_save [name] <confirm>: Save the sales orders of the vending machine as template named [name]. If a template already exists with...

  8. ignignokt84

    ignignokt84 updated VendingManager with a new update entry:

    0.0.5

    - Added health option to set HP of Vending Machines (you may need to add it manually to your json: "health": 500.0 ). Default HP is set to 500 - this is hard-coded in, so Vending Machines are set to 50o HP max when this plugin is unloaded. Additionally, the current health factor is applied so overall health percentage will not be affected by updating the HP.
