    If you want a simpler solution this should work,
    Code:
            void OnEntityBuilt(Planner plan, GameObject go)
        {
            BaseEntity entity = go.ToBaseEntity();

            if (entity.name.Contains("snap"))
                    entity.Kill();
         }
    I haven't tested it, but I took the code from my ArenaWalls plugin.
