  2. Wulf Community Admin

    If you are set as "owner" with Rust, you're automatically getting all of Rust's native commands, and you'll automatically be added to the "admin" group with Oxide's permissions. If you want to add ALL permissions to that group, you can use "grant group admin *", but I wouldn't recommend it as you'd likely get quite a few permissions that are not intended to be used like that.
     
    Wulf, May 6, 2016
