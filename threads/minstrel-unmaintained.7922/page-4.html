    Well done with this, pretty simple to use. Had some difficulty at first since I had to figure out how to add the song file name to the Minstrel.json file. For those that have difficulty, just post your song names in the Minstrel.json file as listed below:

    "tunes": [
    "soiaf",
    "memory"
    ]

    Instructions:
    -Open Minstrel.json file for edit in your server file manager.
    -Add song file names to .json and save. (example above)
    -All the song files will go into directory: Oxide>Data
    -Reload plugin through Console System, Rusty, or RustAdmin programs per command listed below.
    -Songs are ready to be played via your character or by NPCs (HumanNPC plugin).

    Now this is only an example from the songs listed in the forum here. I'm pretty sure 4seti post this .json file too for download. To reload the Minstrel plugin through Oxide. (F1 Console System: type "oxide.reload Minstrel".
