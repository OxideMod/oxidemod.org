    Hey Matt, I think the problem is that you're not set as a Raid Alert admin. The mod automatically adds server moderators when they wake up, to an internal authorisation list. If you installed the mod while you were logged in and you haven't died, or gone to sleep and woken up, it won't have added you. Raid Alert admins aren't checked against perimeters, so they can fly through peoples' bases without triggering alerts.

    Try going to sleep and waking up again. You can see the mod's auth list by doing /rs auths, or rs.auths from the console. You should be in there after that.

    I'll have a think about a better way of adding moderators automatically, and get an update out later today.

    Let me know if you still have problems, as maybe there's a permutation of config options that's causing it, in which case it'd be handy to have copies of your config and data files.

    I'm definitely still thinking about an iOS version. A developer contacted me yesterday and said he might be interested. I'm going to put together instructions on the protocol and set up a test server for him later today, and see what emerges.

    I've not managed to get enough in donations to pay for the hosting this month, so any contributions, even if just a few cents will help. I don't mind covering the costs for now, but long-term it would be nice if it would pay for itself =)

    Many thanks for the interest.

    Let me know how you get on =)

  14. Joe Edmunds

    Joe Edmunds updated Rusty Sheriff Raid Alert with a new update entry:

    Added admin check on startup + alert fix

    Added a check on startup to ensure any moderators or owners are added to the Raid Alert authorised users list as admins. This ensures if the mod is installed while the server is running, logged in admins will automatically be added as opposed to when they wake from sleep; which was the old detection method.

    Fixed an issue where a player could jump between one perimeter and another within the perimeter check interval, and the mod wouldn't recognise the change until the player left the area.
