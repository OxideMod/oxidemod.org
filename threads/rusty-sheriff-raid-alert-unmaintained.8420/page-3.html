  20. Joe Edmunds

    Hey Quicken,

    As far as I'm aware there shouldn't be a problem with many users of the mod on a given server. I've not had any reports from existing server owners that would indicate to the contrary.

    I programmed the mod to spread the perimeter checks over 30 seconds by default, so you shouldn't see any real additional server load, and synchronisation with my Raid Alert server to upload any alerts only happens once every 30 seconds, instead of uploading on each detection. Alerts are only queued to be sent externally once upon breach of a perimeter, and then once again when the player has left, to keep things to a minimum.

    If you ever think the mod may be causing issues, type /rs disable to stop the automatic perimeter checks, and see if the problem persists.

    Let me know how you get on. If there's anything that needs fixing, or any additions/options etc you'd like, post them here.

    I'll be looking more into iOS bits and pieces over the weekend, and good luck with the server! =)
