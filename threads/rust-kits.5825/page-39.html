    - Stop the plugin
    - Go into data/Kits_Data.json
    - Delete all needed :
    Example :
    Code:
    "refunder": {
      "used": 1
    },
    - reload the plugin

    Work fine. I use use Sublime Text and regex / replace tool for that.
