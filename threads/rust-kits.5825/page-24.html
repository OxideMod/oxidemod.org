  13. Jikey#0327

    work? do not Recieving 0 / 2 ? or just Autokit it affects
    Wulf said: ↑
    I imagine the issue only happens with the autokits/welcome portion of Kits. The issues lies with the connection process. Likely plugins trying to give too early as they think that the player is already connected, but they aren't yet. This should be fixed in the dev branch of Rust and will be in tomorrow's update. Kits just happens to be what you are using that brings the issue to light, not the cause. I believe the dev branch changed the login process a bit to account for this and other network related client connection issues.

    To sum up the thread, the issue is NOT from:
    • Oxide or any Oxide extension
    • Any plugin (Kits, Better Start, Enhanced Ban System, etc.)
    • Any Rust server configuration files
    • A .NET library or other dependency missing
    As a temporary fix, remove any plugins that try to interact with a player with OnPlayerConnected, or possibly any early connection hooks. To properly fix the issue, either update to the dev branch of wait for tomorrow's Rust update that has the networking changes.
