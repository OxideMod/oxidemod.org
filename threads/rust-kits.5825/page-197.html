  19. TorQue[MoD]

    You ARE creating the kits inside the game right? The config file isn't used anymore.
    These are the exact steps you need to take:

    1 Launch the game and connect to your server
    2 with the console open (F1) use the items tab to give yourself all of the items you want your kit to include
    3 press ENTER to open the chat window
    4 type /kit add "kitname" - with the quotes
    5 with the items in your inventory (not equipped or wearing) type /kit items

    Repeat steps 1-5 for each kit you want to create.

    To check to see if the kits are created properly, open the chat window and type /kit
    if the kits aren't working, it won't list anything and will give an error. If the kits are working, they will show in a list. If you forget step 5 ( /kit items ) then the kits will be empty and won't give anyone anything.

    Also, you should install the Rewards plugin so that your kits can be added to a store and people can buy them.

    Also, its completely moot since you don't use the config file anymore, but you're closing your code with 4 curly braces and 1 braces in the second line. They should all be curly braces.
