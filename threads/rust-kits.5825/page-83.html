    Thats only for vip and donator groups you set them for you need to set them for these groups then also each member who is in them groups
    You can grant a user permission by using:
    oxide.grant user <username> <permission>
    To create a group:
    oxide.group add <groupname>
    To assign permission to a group:
    oxide.grant group <groupname> <permission>
    To add users to a group:
    oxide.usergroup add <username> <groupname>
    To remove users permission:
    oxide.revoke <userid/username> <group> <permission>
