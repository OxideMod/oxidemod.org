    It would be on the VIP group, but you would need to add each member you want that kit for the VIP group so in other words they would be a VIP. You can call the group anything you want does not have to be VIP but to get the kit they would need to be part of that group.


    /kit add "vipautokit"
    /kit items description "This is the vipautokit" authlevel 2
    /kit hide true
    /kit permission vip

    Note not tested best to try first
