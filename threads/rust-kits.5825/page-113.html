    This version will spawn guns with full ammo. However, for this to work on existing kits you will need to do the following for each kit that contains guns;

    Empty your inventory and give yourself the kit
    type '/kit edit kitname'
    type '/kit items'

    Basically you need to re-save each kit for the plugin to register the weapons.
