    well when you look it from this side. its like all the other plugins from him.
    /kit add NAME (starter)
    so. now you have a "empty" kit with a Name tag. nothing more.

    now you can use the options like in the other plugins

    /kit description "To Get started"

    the most important step. ( i also forgot it..)
    /kit items <<<<<<<<<<<<<
