  9. RustyForest

    when making the kit you have to add a permission for the kit so like /kit permission vip(adds the Kits prefix auto), then you should have a oxide group
    group add vip
    grant group vip Kits.vip

    then when you want to add someone to group you can do
    usergroup add name/steamid vip to make it easy
