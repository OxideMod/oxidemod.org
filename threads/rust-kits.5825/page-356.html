  14. Krazy J

    icosama said: ↑
    hi,

    thank you ^^
    Click to expand...
    Make a group called "steam"
    Add that group to the steamsgroups plugin config and reload.

    Add a kit named "steam" and put stuff in it.
    /Kit add steam

    Add a permission to the kit
    /Kit permission kits.steam

    The add the permission to the steam group.
    o.grant group steam kits.steam

  16. Kappasaurus

    Hello everyone, I've been working on Kits 4 recently, I have an alpha version that has all the core functionality down. Please note, if you decide to use this do not expect any plugins which integrate with Kits to work properly, the API is not fully implemented. Additionally, please note, certain attributes of items like ammunition and attachments won't be saved yet. Lastly, do not expect your old kits data to work with the new version. In order for the new version to work properly, the old data files must be deleted. I appreciate all issues reported, please follow this guide Reporting bugs and issues, and if possible, submit them as GitHub issues. All documentation can be found in the README file at jacobmstein/Kits. The plugin is available at https://github.com/jacobmstein/Kits/releases/download/4.0.0-alpha/Kits.cs. Thank you!
