  1. Mughisi

    You need quotes:
    /kit add Starter "Items to get you going"

    You can also mention the options you'd like after that
    -maxXXX => max numbers of time a kit is allowed to be used
    -cooldownXXXX => cooldown in SECONDS of the kit
    -authlevelX => required authlevel

    So if you want the starter kit to have a 10 minutes cooldown you would use
    /kit add Starter "Items to get you going" -cooldown600
