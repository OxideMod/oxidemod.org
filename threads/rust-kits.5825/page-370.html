    It really is very easy say you are making a vip kit first in game you get all the item you want from the f1 menu then in game chat you copy the first line I posted
    /kit add "vip"

    Then you copy the next line and paste it in game
    /kit items description "VIP reward" permission vip

    it will auto add the permission vip and if you look at the file with notepad++ it will show it as
    "permission": "Kits.vip",
    it puts the word kits in front of every permission so when adding the permission to your vip group you add kits.vip

    oxide.grant group vip kits.vip
