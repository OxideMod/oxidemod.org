    Be aware that installing some of these packages exposes your servers for hackers. E.g. Postfix is a MTA (Mail-Transfer-Agent) which opens a socket and only secures it with a default password.


    About your problem:

    Did you insert the necessary informations in the rustservers script and in the server.cfg?
    If yes we need the proper log files and nost just the informations of `./rustserver details`
    [DOUBLEPOST=1493340818][/DOUBLEPOST]Another small thing:
    /var isn't really there to install programs like steam. It's a normal convention with lgsm to create a new user per server entity & install everything in it's home directory. If you want to install it in a seperate directory it's a normal convention to install programs which are not part of the distribution in /opt.
