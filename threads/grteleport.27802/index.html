    carny666 submitted a new resource:

    GrTeleport - GrTeleport allows you to teleport using the new grid reference. e.g: n5 or p14

    GrTeleport allows you to teleport using the map's grid reference.

    Chat Commands:
    /grt X#

    examples:
    /grt N10
    /grt p12
    /grt s4

    Console Commands:
    grt.nextspawn
    - Teleports player to next spawn available point.

    Permissions:
    GrTeleport.use
    GrTeleport.admin

    API:
    int TeleportToGridReference(BasePlayer player, string gridReference, bool avoidWater = true)
