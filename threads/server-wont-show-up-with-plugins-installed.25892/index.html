    It looks like your host misunderstands how it works then, as plugins have nothing to do with your server being listed or not and Oxide being installed doesn't change your server being listed or not like I stated previously.

    What server listing comes down to:
    • Your server being fully started and online
    • Your server having its main port accessible and queryable by Steam
    • Your client being able to access and query the latest listings from Steam
    Rust's servers are categorized on your client based on something called Steam tags. When Oxide is not installed, you have various tags that add various specs such as player count and such. When you install Oxide, it adds the "oxide" and the "modded" tag, which the "modded" tag is what Rust uses to list your server under the Modded section. Aside from that, there is nothing that is changed with Oxide and the same for plugins.

    The only plugin that could potentially be changing it if you are running a sketchy plugin that fakes online player counts or messing with other Steam tags, but unlikely and you'd know if you installed one of those.
