  8. Colon Blow

    Now that the default 'player' group has changed to 'default'
    have you guys changed the permission to that?

    To set all players to have access use:
    Code:
    oxide.grant group default firearrows.allowed
oxide.grant group default firearrows.ball.allowed
oxide.grant group default firearrows.bomb.allowed
    if you are getting normal arrow every time, this means permissions are not setup for that person / group.
