  6. mattymcrae

    Update: So i appear to have solved my problem. Here is how i did it....

    1. Unloaded Cornucopia and deleted both config and .cs file
    2. Installed new copy of the plugin.
    3. Unloaded again and updated the animal paths in both .cs and config
    4. Loaded the the plugin again
    5. Ran cornu.spawn via rcon
    6. Went in game and used "del assets/rust.ai/agents/" to clear all current animals
    7. When they spawned back on the natural tick cycle they behaved fine.
    Hope this helps and works some other people.

    Update 2: Scratch that, it appears that when the plugin adds additional animals into the game after 15 mins it makes them all weird again. Sorry
