    RA1NF3AR said: ↑
    Ive just been on a server using this plugin and somehow the admin had it set so barrels/crates spawned all over the map can someone tell me how to do that please?
    Click to expand...
    Just look at the default config you'll find a spot where there are barrels and trashcans. You can set the minimum amount there to a value higher than your natural count to get more of them. To find out your natural count, let the plugin run with the default config for like an hour or so and run /cdump and check RCON, you'll find the current count of everything and can adjust from there.
