  10. recon

    Can also recommend OVH.
    Check their newly released Game series: Serveurs GAME | Serveurs Dédiés
    The i7-4790K (MC-32) should be perfectly fine for your needs and has a strong single core performance. They don't list Windows in the OS list but you can install it by yourself with vKVM and boot up a Windows install ISO.

    Otherwise you could go for their enterprise series: Serveurs dédiés ENTERPRISE | Serveurs Dédiés
    Does include Windows by default and has a decent CPU as well. Has no SSD but that's not really needed for Rust Servers.
    500Mbit/s guaranteed and 1Gbit/s burst.

    Also has a decent permanent DDOS protection and quick support responses.
    If you order, take one from the SBG datacenter.

    I'm renting 3 dedis there and had a single HDD failure until now which was fixed instantly.
