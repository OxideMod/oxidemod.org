  3. Mughisi

    Mughisi updated Information Announcer with a new update entry:

    Update

    Changed the name of the plugin from 'Server Information Announcer' to 'Announcer' to reduce the text printed to the Oxide logs;
    Added a default chat prefix color for the prefix;
    Split the join/leave messages so they can be enabled/disabled separately;
    Added an ExcludePlayers option to the config where you can add SteamIds of the players that you don't want to see announced when they join/leave the server;
    Added an option to show the broadcasts in order instead of always a random order;
