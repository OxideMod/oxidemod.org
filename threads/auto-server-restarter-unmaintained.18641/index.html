  1. Feramor

    Feramor submitted a new resource:

    Auto Server Restarter - Automatically restart server every given time to reduce memory usage

    You need to use a start script which loops starting server as given example (_start-example.bat).

    This plugin will execute saveworld and shutdown even there is no player in server.

    Default Settings
    Code:
    {
  "Chat Announcement": true,
  "Chat Announcements Will Occur Every x Seconds": 15,
  "Server Will Restart Every x Seconds": 14400,
  "Start Chat Announcements Just Before x Seconds": 60,
  "Version": "0.2"
}
