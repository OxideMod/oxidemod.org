    Kayzor submitted a new resource:

    LustyPort - A simple teleportation plugin!

    Simple plugin which allows Admins to add locations that players can teleport to.

    Chat Commands:
    /tp <zone> - teleports you to zone
    /tp_back - teleport back to original location
    /tp_list - lists all available zones

    Admin Chat Commands:
    /tp_add <name> - Adds a new zone set to your current location
    /tp_del <name> - Deletes a zone

    Note: When teleporting must remain still for 10 seconds (to prevent using the tp command to escape!)
