    LocalTimeDamageControl - Allows admin to control when damage can be done by non-owners using local server time.

    LocalTimeDamageControl allows admin to control when damage can be done by non-owners using local server time.

    User Chat Commands:
    /lset start 08:00 - sets start time at 8 am
    /lset minutes 160 - sets duration for 160 minutes
    /lset hours 2 - sets duration for 120 minutes
    /lset on - turns protection on .
    /lset off - turns protection off.

    Permissions:
    LocalTimeDamageControl.admin

    Quick Start:
    -Install Mod
    -Set admin permission ie: oxide.grant user username...

  16. carny666

    There are no console commands for this plugin.

    To use the LSET command you must grant yourself admin privs for the plugin, not admin privs on the server per se:

    Code:
    oxide.grant user <username> LocalTimeDamageControl.admin
    or

    Code:
    oxide.grant group <admingroupname> LocalTimeDamageControl.admin
    otherwise you must make sure the config file has the start time in 12 hour format, including the AM/PM specifier.

    Please repost your config.
