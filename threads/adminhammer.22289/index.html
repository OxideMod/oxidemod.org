    AdminHammer - Use the hammer to display info about entities/authorization

    Equip the hammer (default tool) and point and right-click at any building/object to display ownership/authorization information.

    You can change ToolUsed to any other item in the Config file. Use the item's shortname - find the item list here: Oxide API for Rust

    Permissions
    This plugin uses Oxide's permission system. To assign a permission, use grant user <name or steam id> <permission>. To remove a permission, use revoke...

  4. mvrb

    albator4242 said: ↑
    I do not see how the plugin works. A small video will be cool thank you
    Click to expand...
    I will see if I can do that.

    1. Type /adminhammer (This will enable the tool for you)
    2. Hold the hammer in your hand (if you haven't changed the ToolUsed value in the Config file)
    3. Look at a building/player-made object and right-click
    4. It will show you the owner of the object. If it's a Cupboard or AutoTurret, you will see a list of authorized players.
