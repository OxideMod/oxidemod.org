    If you wiped the primary EntityOwner data file that contains KnownPlayers, yes. Prod will not work for any structures until the player logs in again to re-add them to the KnownPlayers list. Deleting the primary EntityOwner data file will not delete the ownership data, only the reference to it until the player logs back in.
