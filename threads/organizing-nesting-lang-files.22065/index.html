  5. Mughisi

    All language files created/maintained/used by the Lang API use a Dictionary<string, string> so nesting isn't possible, you can however just alter the names of your keys to something like this:
    Code:
    {
    "menu-msg1": "bla",
    "menu-msg2": "bla",
    "menu-msg3": "bla",
    "chat-msg1": "bla",
    "chat-msg2": "bla",
    "chat-msg3": "bla",
}
     
    Mughisi, Nov 21, 2016
