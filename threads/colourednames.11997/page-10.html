  14. PsychoTea

    DazeR said: ↑
    how can i give myself the colour permissions like where and what do i type. wont work for me
    Click to expand...
    To give yourself the "use" permission type, in the server console or RCON:
    Code:
    oxide.grant user YOUR_USERNAME colourednames.use
    Replace "YOUR_USERNAME" with your ingame name.

    Then, to use the plugin type "/colour {colour}" in chat. Replace "{colour}" with the colour you want to use
     
    PsychoTea, Aug 17, 2017
