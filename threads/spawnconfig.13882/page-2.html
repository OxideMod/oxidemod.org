  7. CaptainNebulous

    1zwo said: ↑
    We are just 2 People on the Server, how do i make the Crate-Spawnrate higher in the Radtowns?
    Click to expand...
    In note pad open up the LootConfig.JSON file in Oxide/Config (assuming you have reloaded the plugin or restarted server after uploading the file to Oxide/Plugins)

    Use the FIND feature in the EDIT menu

    Search for "crate"

    In the vicinity you will see the min/max spawners and maximum spawn timer

    Edit to your liking (however I believe there is only 80 or so crate spawns)

    Save

    Delete old SpawnConfig.JSON from oxide/config folder on server

    Upload new SpawnConfig.JSON to same folder

    In game chat type /reload SpawnConfig

    DONE!
     
    CaptainNebulous, Feb 1, 2016
