  1. JakeKillsAll

    JakeKillsAll submitted a new resource:

    NoGreen - Simply removes the green name away from admins in-game

    This plugin does exactly what the name says. It changes the chat format so that every player's name is the normal blue, including admins, owners, and moderators.

    The main reason I wrote this is because it just simply works. You don't need to configure anything, and there is currently no config file.

    One thing I will likely add in the future, once I learn how, I will add a config file and make it so that you can control your chat's username and message colors independently.

    There are no compatibility issues in how you send a message. If you want to send the avatar, you need to use a method that allows that such as rust.BroadcastChat in order to specify the userId for the avatar. SendReply and PrintToChat are essentially the same thing, no difference in what they do really.

    As far as people not seeing messages, it shouldn't be the cause, but try removing the if (string.IsNullOrEmpty(Message)) return false; line.
