  6. Mughisi

    OnPlayerInput is a horrible way to do this. A hook was actually added in the past to allow you to capture hits of the hammer (Oxide Documentation) as Sanlerus also mentioned above.
    Code:
    void OnHammerHit(BasePlayer player, HitInfo info)
{
    // Your code here...
}
    To obtain the object the user hits you just need to grab the target out of the HitInfo object and check if it is a building block, no need to do any raycast either.
