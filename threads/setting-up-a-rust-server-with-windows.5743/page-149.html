  14. Samsonite

    DerlustigeLappen said: ↑
    When i try to connect it says Wrong Connection Protocol after Update Server and Oxide.....i have installed it with
    app_update 258550 validate is it right or what is right ? Thanks for Help me
    Click to expand...
    Add the following to steamapp 258550 for the corresponding server versions.
    e.g. "258550 -beta experimental"

    -beta experimental is the stable Rust experimental server.
    -beta legacy is the legacy dedicated server
    -beta development is the bleeding-edge build that mirrors changes on @RustUpdates as soon as the builds can be compiled and uploaded to Steam -- which can be as many as 20+ times a day.
    -beta development-debug is an alternate development branch which the devs maintain, and on occasion it contains changes and fixes before the development and 'default' branches get them.
