    anyway
    fix for server not saving items, inventories, buildings, props etc
    when my save plugin manually saves files it saves to Procedural Map_<Map Seed>
    however the servers auto save saves to Procedural Map_<Size>_<Map Seed>

    so when i used -load "C:\my_rust_server\server\FirstServeryay\save\Procedural Map_<Map Seed>/0.sav" it would set me at my previous spawn point but however would still wipe all inventories, props, buildings etc

    however when i used -load "C:\my_rust_server\server\FirstServeryay\save\Procedural Map_<Size>_<Map Seed>/0.sav" this then worked for me, loaded me at my sleeping bag and with my inventory so yay! thank you for the help : )
