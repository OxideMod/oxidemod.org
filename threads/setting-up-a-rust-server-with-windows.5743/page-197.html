    The -cfg and -datadir do not work or exist with Rust, they were only used in Rust Legacy. I would also remove -autoupdate too. The actual config is under your server/identity/config folder. I think you have a couple other settings wrong too, maybe maxplayers and max.fps, but I'd have to double-check. The "cd ../" at the end shouldn't be there either, at least not in the same line.

    I'm not aware of any bugs with Oxide, but if you let us know we can fix them.
