    Before the recent Rust update, everything was running fine, no problems, nothing. Then the Rust update happened. I updated it through Steam CMD and ran the server again. Nothing appeared in the server window. So I made a completely new installation through Steam CMD and following the tutorial on this website. Same problem. I tried this 3 times with no solution.     @echo off
    RustDedicated.exe -batchmode +server.port 28015 +rcon.port 28016 +rcon.ip 0.0.0.0 +rcon.password "changeme"
