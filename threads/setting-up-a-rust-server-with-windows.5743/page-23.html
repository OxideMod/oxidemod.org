    You just extract the contents of the snapshot file into that folder, the snapshot contains the changed files & required files for oxide so when it asks to overwrite certain files you say yes.

    Regarding the bat file, just put it in the same folder and use something like this:
    Code:
    @echo off
set IDENTITY="myserver"
start RustDedicated.exe -batchmode -server.hostname "<YOUR-SERVER-NAME>" -server.port 28015 +server.identity %IDENTITY% +server.maxplayers 100 -cfg "server/%IDENTITY%/cfg/server.cfg" -logFile "server/%IDENTITY%/logs/rust_%DATE:~-7,2%-%DATE:~-10,2%-%DATE:~-4,4%.txt" -server.seed <YOUR-WORLD-SEED> -server.worldsize <YOUR-WORLDSIZE> +rcon.port 28016 +rcon.password "<YOUR-RCON-PW-HERE>" -server.globalchat true
    Replace the things between <YOUR-...> with your values, server name & rcon password are pretty straight forward, world seed should be any number (really any number ) and worldsize can be anything between 2000 and 8000.
