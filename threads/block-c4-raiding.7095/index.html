  3. swiderek

    I have installed the ItemConfig plugin.

    After config file is created open it, go to the bottom and change all values to 100 for each type, Twig, Wood, Stone and Metal like this:
    (example)
    "DamageProtections": {
    "BlockTwig": {
    "Generic": 100.98,
    "Hunger": 100.0,
    "Thirst": 100.0,
    "Cold": 100.0,
    "Drowned": 100.0,
    "Heat": 100.0,
    "Bleeding": 100.0,
    "Poison": 100.0,
    "Suicide": 100.0,
    "Bullet": 100.9999,
    "Slash": 100.99,
    "Blunt": 100.99,
    "Fall": 100.0,
    "Radiation": 100.0,
    "Bite": 100.99,
    "Stab": 100.98,
    "Explosion": 100.3,
    "RadiationExposure": 100.0,
    "ColdExposure": 100.0

    You can also change the maxHealth for each item to any number you wish. I did it to 99999 just like this:
    (example)
    "wall.window.bars": {
    "grades": {
    "0": {
    "maxHealth": 99999.0,
    "canRotate": true,
    "costToBuild": [
    {
    "itemDef": "wood",
    "amount": 300.0,
    "itemid": 3655341.0


    Restart server and now all home parts should be indestructible no matter what weapon you will use
    Works for me and all my players are very happy.
