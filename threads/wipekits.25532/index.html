  1. Ryan

    Ryan submitted a new resource:

    WipeKits - Puts a configurable cooldown from the wipe on each kit depending on their kitname.

    This plugin puts configurable cooldowns on certain kits after a map wipe. This is useful for only allowing players to use kits until # hours/days after a wipe. Forcing players to farm instead of redeeming a kit to build a base after a wipe.

    This plugin hasn't been tested on the GUI version of Kits.

    Default Configuration File (oxide/config/WipeKits.json)
    Code:
    {
  "Kit Names & Cooldowns - Cooldowns (minutes)": {
    "kitname1": 5.0,
    "kitname2": 5.0
  }
}
