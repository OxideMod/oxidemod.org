    Mattparks submitted a new resource:

    XpRevived - A plugin that brings back a XP system.

    XpRevived is a plugin that adds a XP system back into Rust. This system can be heavily customized. This plugin is recommended to be used with NoWorkbench to disable workbench craft dependencies, and with Crafting Controller to disable the crafting for Workbenches, Research tables and items that can be crafted with unlimited components enabled.
    This plugin can also be used to disable components from loot boxes and crafting, listing what can be leaned from a level is found using "/level #",...

    The way I created this XP system is that is is to replace the BP system, so you might need to be level 15 and have 11 XP to learn a salvaged axe. The point was to not force a player to go out and search for a item that has a low chance of spawning and relying on pure luck, but instead to reward people who stayed on and played on the server.
