    k1lly0u updated ImageLibrary with a new update entry:

    2.0.28

    Added methods to remove and restore images when the CommunityEntity ID has changed.

    The CommunityEntity ID is used by clients to find images in file storage. When this ID changes (usually after a wipe) the client can no longer find images using the image identifier. This would result in servers having to re-download all stored images from the internet.
