  4. RAYs3T

    Axial said: ↑
    Im not sure about the ECC-Memory how can i check that?
    Click to expand...
    You can run the cmd
    Code:
    wmic MEMORYCHIP get DataWidth,TotalWidth
    If value TotalWidth greater than DataWidth you got ECC-Ram (ECC stands for Error-correcting code memory) in your box.
    In the case that you got ECC-Ram, we can exclude the ram as failure point...but if not, you schold give Memtest86+ a try. just to exlude this as possible failure point.
