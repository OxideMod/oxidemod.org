    DylanSMR submitted a new resource:

    Smart Homes - Control your home and make it smart

    Smart home is a system that allows you to control various things including:
    • Turrets(Toggle On/Off)
    • Lights(Toggle On/Off)
    • Doors(Open/Close)
    To make this less over powered this has a configuration option that only allows people to create a interact able object within a radius of there base.

    Configuration:
    • ActivationDistance = 30 - Distance a player can interact with a object from his home.
