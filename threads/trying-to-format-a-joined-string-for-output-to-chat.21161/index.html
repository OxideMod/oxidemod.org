  4. sqroot

    KDRGUI isn't spacing its columns with spaces.
    In the font Rust uses, digits are monospaced, spaces are not, and as such, using spaces for proper spacing isn't possible (unless the width of a digit is divisible by the width of a space, but that's difficult )
    Maybe you can try using tabs ("\t") and see if that allows you align things.

    Adding leading 0's should be easily possible.
