  1. Absolut

    Absolut submitted a new resource:

    TrollTax - Places a tax on players killed. Once the Taxer dies his/her tax payer list is cleared.

    This is a very simple plugin to implement. It works as follows:

    Admin loads the plugin and sets the tax rate value in the config file (default is 10%).

    A player places a large wooden box or wooden box and is prompted whether to make this box the players tax box.

    If this player kills another player they will begin receiving taxed portions of resources (stone, wood, sulfar, metal, HQM, etc.). The taxed portion will automatically go in the designated tax box for the player.
