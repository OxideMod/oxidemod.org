  9. Calytic Community Admin Community Mod

    Just letting people know who might search for this, I managed to figure a fix using the following code:

    Code:
    var p = plugins.Find(pluginName);
if(p !== null) {
        this.Plugin.Manager.RemovePlugin(p);
        this.Plugin.Manager.AddPlugin(p);
    }
