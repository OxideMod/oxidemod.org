  10. yt0k

    Same thing just happened to a player on my server. Player is legit as well.

    Edit: On the players side they are seeing "Render Texture Create failed : format unsupported"

    Found this fix via Facepunch @ https://facepunch.com/showthread.php?t=1340931

    Right click rust in steam, go to properties -> set launch option and type this:

    -force-feature-level-9-3
    -nolog
    -force-feature-level-11-0
    -high
