  11. Das JÄGER

    Thank you for the awesome plugin, however it overrides Zlevels so I had to make some changes.

    Open up your Timber.cs go to lines 36 & 37 comment them out.

    Code:
                     //public float harvestStanding = 0.5f; // The rate of gather in fallen trees.
                //public float harvestFallen = 2.805f; // The rate of gather in standing trees.

    Now go to line 133 and comment it out until line 142.

    Code:
                
/*Changes the harvest amount in objects that are falling or despawning.
            if (fallDefined || despawnDefined)
            {
                item.amount = (int) (item.amount * configs.options.harvestFallen);
            }
            else if (dispenser.gatherType == ResourceDispenser.GatherType.Tree)
            {
                item.amount = (int) (item.amount * configs.options.harvestStanding);
            }
            */
    Now save the plugin and it should auto update the pluing for you after you save it. You are now done with those steps. If you want to add a bit of realism to the falling trees, and only get wood from trees once the tree is on the ground follow these settings below.

    Ok first go to Timber.cs, only comment out line 37

    Code:
                      public float harvestStanding = 0.5f; // The rate of gather in fallen trees.
                 //public float harvestFallen = 2.805f; // The rate of gather in standing trees.
    Next only comment out line 136

    Code:
               
                    //Changes the harvest amount in objects that are falling or despawning.
            if (fallDefined || despawnDefined)
            {
                //item.amount = (int) (item.amount * configs.options.harvestFallen);
            }
            else if (dispenser.gatherType == ResourceDispenser.GatherType.Tree)
            {
                item.amount = (int) (item.amount * configs.options.harvestStanding);
            }
    Final step now go into your Timber.json make sure your "harvestStanding" looks like mine.

    Code:
    {
  "options": {
    "harvestStanding": 0.0,
    "hitsStanding": 10,
    "hitsFallen": 10,
    "despawnLength": 30.0,
    "screamPercent": 0.03,
    "includeCacti": true,
    "logToPlayer": false
  }
}
    That is it!

    Regards,
    Jager
