    So i put this in as I was interested in using it for obvious reasons. Turns out even with a set config, as soon as you gain TC rights you can destroy other peoples walls etc with the hammer.

    Config below

    {
    "AllowDemolishAfterServerRestart": true,
    "AllowRotateAfterServerRestart": true,
    "ClanCanDemolish": true,
    "ClanCanRotate": true,
    "DemolishTime": -1.0,
    "FriendsCanDemolish": true,
    "FriendsCanRotate": true,
    "MustOwnToDemolish": true,
    "MustOwnToRotate": true,
    "RepairDamageCooldown": 1.0,
    "RotateTime": -1.0
    }
