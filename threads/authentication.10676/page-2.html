    1.0.0

    Changelog (Version 1.0.0) See the main page for descriptions:

    Most of the plugin's code has been rewritten to enable some new features to be added, this is the first update to have them.
    • Features:
    • Ability to toggle plugin on/off
    • Prevent player from chatting before authentication
    • Prevent player from typing the password in the chat
    • New Commands:
    • /auth toggle (on/off)
    • /auth timeout
    • /auth timeout [new timeout]
    • /auth status
    • Permissions:
    • auth.edit has been replaced by authentication.edit
    • Fixes:
    • Plugin won't stop working when the server is reloaded;
