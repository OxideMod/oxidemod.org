  19. Mughisi

    Mughisi updated Teleportation System with a new update entry:

    Fixes and new command

    Fix for the issue where servers don't have the server.worldsize parameter set which would cause them to limit teleports for admins between 0 and 0 (x, z values).
    New '/tpn <player>' admin command that will teleport you near the supplied playername, also accepts '/tpn <player> <distance>'.
    New admin command to wipe saved home locations (for after a wipe), '/wipehomes', be careful with this command as it will erase all the saved homes for all players and they can't be recovered!
