  19. Mughisi

    That's how I intended it yes, basically the idea is that when you're just playing and then suddenly someone asks for your help you tp to that player and then you use /tpb to go back to your previous location and continue playing where you left off. It will always remember the first location you were so when you're playing at location a and multiple players need your help so you tp to location b, and then to location c, d, e and f, you will still return to location a when using /tpb. This location is saved in the data file so if you forget to use /tpb that location will still be saved and the location will not be saved again until the command is used.
