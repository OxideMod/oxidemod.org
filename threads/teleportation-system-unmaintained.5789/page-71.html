  17. k1lly0u

    jbarrett said: ↑
    is there a way to remove the sethomes inside a base? i thought what used to happen was it wouldn't let anyone set a home in a base if they were not friends, but that seems to not be the case anymore.
    Click to expand...
    You need the friends api and building owners plugins to make it so you can only set home in yours or a friends base. Both are very useful plugins, but if they aren't on your server since your last wipe all existing buildings prior to you adding them will not be affected by it (may change if a player adds a new part to a existing building but I'm not 100% sure)

    As for removing set homes, if your trying to remove ones people have placed in others bases specifically I think you will have some trouble unless you know who it is and where they are going exactly. Otherwise there is /wipehomes which will remove all home locations.

    If you don't already have building owners it's possible to add it and change owner of houses to the players that actually own them using /changeowner "player name" (depends how many houses you have and how keen you are to do it)
    Then use /wipehomes to clear home list, then only players who own the house can set home there. (Or their friends if you have friends api)

    Also check the config for tp as I can't remember if there are options to turn this stuff on or not?? (Providing you have the plugins)
