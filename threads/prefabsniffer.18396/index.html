  1. Ayrin

    Ayrin submitted a new resource:

    PrefabSniffer - Sniffs the game files for prefab file locations

    This small plugin is currently set up to sniff and give you a list of the prefabs available for use with the Build plugins /plant command.

    To use it simply type prefabs in your RCON of choice, and then go to your ~/oxide/logs folder and you'll find a file named PrefabsForBuild_ followed by the current date.

    The purpose of this plugin is to allow people an up-to-date list of available prefabs when they need it.
