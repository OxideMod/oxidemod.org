    Loot Spawn Lists - Allows modification of the loot tables.

    Loot Spawn Lists
    Allows modification of the loot tables (drops from zombies, crates and supply drops).

    How to use
    1. Once the plugin is installed, launch or restart the server.
    2. Inside your server's data directory (see http://oxidemod.org/threads/how-to-install-and-use-plugins-on-your-server.70/), the file "loot_tables.txt" should have been created. Download it.
    3. This file contains the default loot tables included within rust, in JSON format....
