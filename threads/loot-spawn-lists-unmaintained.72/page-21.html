  2. Gliktch

    DG2 said: ↑
    Hello,

    Can anyone tell me how to edit code to original drop after the recent Rust patch (E.g. when red bears and wolves appear instead of zombies now).
    Click to expand...
    If you want to revert to original (vanilla, unchanged from Rust defaults) drop lists, you can rename or empty the txt config, or uninstall the plugin.. If you want a particular drop list (e.g., wooden boxes) to go back to defaults, you can remove that sublist (in this case, JunkSpawnList) from your txt config and reload or reboot.

    If you want a loot lists config which recreates the defaults or some other variation on this, you can look in the new 'Configurations' section of this forum. Good luck with your server
