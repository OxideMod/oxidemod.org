  3. Caffeine

    Honestly in this day and age, Ubuntu is usually the best distro for anything server side, I know people can be passionate about their distro flavors, but being completely practical, Ubuntu is one of the easiest and most supported. It's global reach is massive and I find it fights with me less than some other distros. To give you some examples, Steam Linux focused Ubuntu first, Rasberry Pi os is Ubuntu based, it's also the most popular cloud based OS(on Amazon AWS and Rackspace). So I'd go with Ubuntu personally, it's tremendously easy to find solutions and help in regards to it with a simply google search where as other distros you could get lost.
