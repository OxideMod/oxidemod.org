Also worthy of note is that in the config there's a cooldown for the groups. so if Moderator group in the config has 60 as a cooldown, moderators can only use the economy plugin once every 60 seconds per player. Hope this helps!

  13. FlamingMojo

    Cataclysme said: ↑
    Hello , I wonder if permisions in adminmenu are the same as that of the plugins that is name is .
    Example: A modo didnt permission on the economics but it may give money from the adminmenu
    Click to expand...
    Hi, sorry about the confusion - the adminmenu.allow.economy permission should only be given to those you trust to use the economics plugin - as Economics does not have it's own permissions, the adminmenu has to bypass it to allow non-admins to use it at all. In short - anyone with adminmenu.allow.economy can give or take money from players. If you want, you can set a cooldown for their access per player in the config, i.e. so they can't spam give the same player money
