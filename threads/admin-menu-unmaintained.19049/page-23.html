    looks good, but i kinda still have to pass. For me its missing the ability to add own commands. i got a few selfmade plugins and would want a option to add those into the menu.
    Click to expand...
    There is the ability to add your own selfmade plugins so long as your own plugins have callable console commands. Just add the following to the config file under ActionMenu if the command is player specific or ToolMenu if it is a general command:
    Code:
        "YourPluginName": {
      "Command_Name1": {
        "bcolour": "black_tint",
        "cmd": "yourplugin.command param1 param2...",
        "perm": "yourplugin.allow.xxx",
        "pos": 1,
        "tcolour": "white_text"
      },
    For the Params you can use:
    Code:
    {ADMINNAME} --Replaces this with calling admin name
{PLAYERNAME} --replaces this with target player name
{ADMINID} -- replaces this with calling admin UserID (STEAMID64)
{PLAYERID} -- replaces this with target player UserID
{PLAYERXZ} --replaces this with target player's X and Z co-ords  with a space between
{PLAYERXYZ} --replaces this with target player's X and Y co-ords  with a space between each
    For permission you can use a specific permission or IsAdmin to require authlevel >0

    for exanple for rain of fire it uses in the config:
    Code:
        "RainOfFire": {
      "RainFire": {
        "bcolour": "red_tint",
        "cmd": "rof.onposition {PLAYERXZ}",
        "perm": "IsAdmin",
        "pos": 1,
        "tcolour": "white_text"
      }
    },
