    Slack - Plugin API for sending messages and notifications to Slack

    Slack is a plugin API for sending messages and notifications to Slack channels. There are a few message templates provided, and more will likely be added as the plugin develops.

    Configuration
    You can configure the settings in the Slack.json file under the oxide/config directory.
    Code:
    {
  "BotName": "OxideBot",
  "HookUrl": "",
  "IconUrl": "",
  "LinkNames": true,
  "ServerInfo": true
}

    The HookUrl is the URL you get from creating an incoming webhook in your Slack community.
    https://YOURSLACKNAME.slack.com/apps/A0F7XDUAZ-incoming-webhooks

    The IconUrl is optional, it's just used to override what is set as default when you make your webhook.
