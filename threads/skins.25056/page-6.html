    I think the administrator should decide independently which skins to add, which ones don't. And remember that the number of skins for one subject is limited to 29 (default skin is required).
    1. I don't recommend to open skin command to all players only for vip or admins;
    2. I don't recommend to add alot of skins from workshop, probably just your own skins with server name if you have.
    3. And keep in mind, not optimezed skins adds lags to players. (All optimzed skins is available by default in this plugin, any added skins is not optimezed.)
