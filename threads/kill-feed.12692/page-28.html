    You can set 2.1 formatting to "§{initiator}{hitBone}§{distance}{hitEntity}§". That should do the trick. Additionally you can just add spaces like so if you need them:
    Code:
    "§{initiator}    {hitBone}§{distance}    {hitEntity}§"
