    Kill Feed - Displays a basic kill feed on screen



    Displays a basic Kill Feed, like the ones you see in Counter-Strike games, on screen.

    Features:
    • Only displays players who either have been wounded or killed while not being wounded!
    • Will display a headshot icon if a player receives a headshot!
    • Works with animals!
    • Position, icons and more can be customized through a configuration file!

    You should be able to adjust the position by configuring Position > x/y in the configuration file. I chose that particular positon because it interferes the least with the existing UI, only overlapping slighty with the chat-bar.
    Hitbox and distance probably won't be included in this plugin. I want to stick to the classic Kill Feed functionality without much fuzz, though I might work on a plugin extension that adds your requested functionality in the future.
