  13. Tuntenfisch

    thomaspaste said: ↑
    "1.3.2 player deaths": false (still report when people suicide) - Seems like it's ignoring the config in some places.
    Click to expand...
    if 1.3.2 player deaths is set to false it will only display players who either have been wounded or killed while not being wounded by default! If you suicide you have been killed while not being wounded, so it will be displayed. If you want to avoid suicides from showing up you can remove the URL pointing at the suicide image under 3.3 Damagetype Files.
    [DOUBLEPOST=1467979746][/DOUBLEPOST]

    2.6.1 x defines the horizontal position. 0.0 will put the Kill Feed all the way to the left hand side of the screen. 1.0 will put the Kill Feed all the way to the right hand side of the screen. Keep in mind that the anchor point for the position is the center of the Kill Feed, so a value of 0.0 will probably result in half the Kill Feed being outside the screen to the left.
