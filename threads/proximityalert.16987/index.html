  1. k1lly0u

    k1lly0u submitted a new resource:

    ProximityAlert - Displays a UI warning message when players get within a set radius of the user

    As requested by @Tenpo89

    Displays a warning message to the user when other players get within a set radius, and another message once they leave

    Chat Commands
    /prox - Activates/Deactivates the function for the player using the command

    Config
    Code:
    {
  "GUI_X_Dim": 0.5,
  "GUI_X_Pos": 0.25,
  "GUI_Y_Dim": 0.12,
  "GUI_Y_Pos": 0.12,
  "TriggerRadius": 50.0 // The range of detection
}
