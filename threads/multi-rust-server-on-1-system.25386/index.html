    You can't start and stop them unless you use commands like restart or quit via RCON, however you can manage the servers after that. If you'd really like to manage them at that level you'd need your own program or to find one that'd suite your needs.

    To further expand upon restart and quit, those won't restart your server unless your batch file is setup to do so, you can do that by adding :x to the top, and goto x to the bottom.
