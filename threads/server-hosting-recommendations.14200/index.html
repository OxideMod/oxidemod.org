  15. Tekoud

    I find it very hard to find a list of required specs somewhere. Wanted to run my own server so i just ordered from Strato.com a windows 2012 vps, cheapest package for 9 euros a month with 1 core and 2 gygs internal to see how it would work. That really didnt run well to my wishes with like 25 mods, 50 players and a 8k map. I was totally new that month to Rust server and you gotto start somewhere i thought so lets just try. It started lagging quickly but memory leaks occured also i think which blew up performance back then. I saw people in forums seeing their server blow up to 26gygs internal memory occupied of the 32gygs so i think it was a software issue also then.

    So i upgraded the package to the second tier for 16 euros a month with 4 cores and 4 gyg to see how well that would behave, bearing in mind i would probably upgrade again next month because overall you hear 8gygs in your server is quite important for performance. I have learned how to keep entities low now (For me its just installing twigsdecay setting decay on twig wood and stone and making a round removing all raided and old bases using R-remover each day myself, some days the entity count is even lower then 4 days earlier, been in the 150k-200k entities range all save now.

    So i dont see the need for the 8gygs internal memory version yet, highest amount of players i had was 26 while its the third month so i was amused by that. While i had 26 players there was about 150k entities on the 6400 size map and server was running on ~230 serverfps so that was quite ok. I restart every like 48 hours now just as a precaution. Usually a restart cleans about 25k entities and fps will go like 40 higher.

    Hope this info helps you, and personally i think its more fun configuring a complete windows12 server to your own needs. You can access all directories and files easily and do not have to cope with some "provider dashboard" because its your own server. On the other hard through a provider you can get it all running very easy. A friend of mine uses multiplay for a 20 player server and hes quite satisfied, installed a fully modded server without too much ict knowledge.
     
    Tekoud, Jan 26, 2016
