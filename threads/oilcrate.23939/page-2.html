    the disappearing oil even if you don't have the permission is a known bug waiting on a fix. I have it too. Now as for the permissions. how are you trying to set permissions ?

    There are two ways you can assign permissions: to Player or to Group
    usage of permission: oilcrate.allow

    grant user <steamid64/playername> oilcrate.allow
    grant group <groupname>oilcrate.allow
    **Note** player has to be added to group to get the permission**

    if you're not doing this way it will not work.

    If you change your config this way,
    Code:
      },
  "3. Oil gahter rate (lower is faster)": {
    "1. Min (0.0 to ~200.0)": 0.0,
    "2. Max (0.0 to ~200.0)": 1.0
  }
    you'll get the maximum amount.

    When you have done this you must reload the plugin and use the command /oilcrate update.
