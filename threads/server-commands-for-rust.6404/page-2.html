  19. Mughisi

    TheRipper 2 said: ↑
    where is that?
    Click to expand...
    There is actually a minor bug in the loading of the configuration file in Rust as it isn't loaded from the server identity folder. It is instead loaded from the root directory (the one that has the folders server, config, logs, RustDedicated_Data, ...). You want to create a new folder "cfg" in here and in that folder you create a file "server.cfg". And in that file you place all the server commands you want to execute when the server loads.
