    ignignokt84 submitted a new resource:

    BedShare - Allow public beds to be set up

    BedShare allows server admins to set beds to be available to all players. Implemented as an overlay on the respawn screen which will display shared beds similarly to regular beds.

    The command prefixes /bag and /bed are interchangeable - since bag/bed is ambiguous in Rust, it made sense to allow either to be used.

    Chat Commands:
    /bed share: Share the bed/bag being looked at
    /bed unshare: Unshare the bed being looked at
    /bed status: Display...
