  1. 4seti

    4seti submitted a new resource:

    AutoWipeReply - This plugin check player's chat for "wipe" word and reply with info given by admin.

    I was freaked by players always asking about "Next wipe date", so:
    This plugin check player's chat for "wipe" word, reply with info about last and next wipe (setted up by admin).

    Admin commands:
    /awr l "Last wipe infoText"     - set info about last wipe (date or smth)
    /awr n "Next wipe infoText" - set info about next wipe (date or smth)
    /awr w "word" - add new word (or phrase) to check disctionary (default is "wipe" and "вайп")

    That's all.
