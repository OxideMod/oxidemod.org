  6. testro

    Here is another way to find maps.

    Just go here. http://playrust.io/

    Pick any server and look at their map. When you find one you like just ask the admin of the server for the seed. Try the rustafied servers and rustopia. Those admins will give you the seeds if you like them and want it.

    Any server with players on it will show a map that is current for the last update - since it it being used now. So no old maps.
