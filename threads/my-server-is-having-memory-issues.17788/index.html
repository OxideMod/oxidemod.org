    Code:
            var itemm = item.GetHeldEntity();
                            if (itemm) itemm.KillMessage();
    A fairly bad workaround is dropping the item at invalid coordinates, so really low (or probably really high, or anything else that is just invalid) and it will forcefully destroy it.

    EDIT: Searching through the assembly I found
    Code:
    protected void RemoveFromWorld()
  {
    BaseEntity worldEntity = this.GetWorldEntity();
    if ((Object) worldEntity == (Object) null)
      return;
    this.SetWorldEntity((BaseEntity) null);
    this.OnRemovedFromWorld();
    if (this.contents != null)
      this.contents.OnRemovedFromWorld();
    if (!BaseEntityEx.IsValid(worldEntity))
      return;
    worldEntity.Kill(BaseNetworkable.DestroyMode.None);
  }
    I think you could just copy this code and use it.


    EDIT Again:
    Confirmed that the code above works, provided you drop the item first.
