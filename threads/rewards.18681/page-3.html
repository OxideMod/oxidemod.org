    In addition to what he said, you can disable one of them (or both as well of course).
    For example if you only want distance multiplier, you set DistanceMultiplier_Enabled to "true" and WeaponMultiplier_Enabled to "false"
    Then you will be rewarded (default reward * Distance reward)

    If you want both you just set both to "true"
