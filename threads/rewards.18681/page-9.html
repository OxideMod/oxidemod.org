    To stop this plugin (Rewards) from giving Activity Rewards (which is different from welcome reward) then set ActivityReward_Enabled to false.

    ActivityReward rewards players an amount of money (ActivityReward) each x minutes they're online (ActivityRewardRate_minutes).
    Welcome Reward rewards players only the first time they join the server.
