  7. Mughisi

    A few things you could do with this plugin:
    - Add a command to give all players a random item
    - Add an option to automatically run the command on a set interval
    - Add console commands as well
    - Replace Rust's the find function with your own to allow players to use partial, case-insensitive names and when multiple names are found they can select a player from a list by number in that list (you could check the inventory viewer command for this)

    A few things you should consider to do:
    - Show a message to the player using the command when the playername specified doesn't return a player
    - Use the functions available in Rust to give an item instead of using the inv.giveplayer console command as that is from the give plugin and not available to a server unless they have it (or you add that plugin as a dependency and you just immediately call that plugin instead of running inv.giveplayer )
     
    Mughisi, Apr 10, 2015
