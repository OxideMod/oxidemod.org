    Shows the current time info:
    /settime

    Modifies the time:

    Freeze the time:
    /freezetime

    Unfreeze the time:
    /unfreezetime

    Set the day length:
    /setdaylength <length in minutes (5-720)>

    Set the night length:
    /setnightlength <length in minutes (5-720)>
    Console commands:
    Code (Text):
    Shows the current time info:
    env.time

    Modifies the time:
    env.time <hour (0-24)>

    Freeze the time:
    env.freeze

    Unfreeze the time:
    env.unfreeze

    Set the day length:
    env.daylength <length in minutes (5-720)>
