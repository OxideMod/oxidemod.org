    Click to expand...
    Make sure to set both the night and day lengths, if both are not set it won't do anything. And you should not edit the lua file, the sunrise/sunset values are constant values that can't be altered and are determined by the time of day system that is used in rust, modifying these values will mess up the timer that runs to modify the day and night length.
