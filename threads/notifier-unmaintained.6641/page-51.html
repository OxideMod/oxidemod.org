  13. sLUDGE

    1. in console, oxide.unload notifier
    2. delete your old version of plugin notifier.py on the ftp.
    3. delete your old config file notifier.json on the ftp.
    4. upload the new version of notifier.py, it loads automaticly and generates a new config file notifier.json.
    5. download the new generated config file and edit it.
    6. upload your edited config file, and replace the existing one.
    7. in console, oxide.reload notifier
