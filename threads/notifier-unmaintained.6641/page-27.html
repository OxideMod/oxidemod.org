  19. NobodyFTW

    SkinN õ.Õ'.|. updated Notifier with a new update entry:

    Notifier | Admin Talk + Improvements!

    [ CHANGELOG ]
    • Code improvements in overall.
    • Adverts interval works now in minutes instead of seconds.
    • Greatly improved how Admin Tags work.
    • Board messages now use the prefix on the title instead of the plugin name.
    • Admin tags will now only show in chat messages.
    • Added Admin Talk feature.
    • Added Admin names colors.
    • Added Admin Tags colors.
    • Added {players}, {maxplayers}, {sleepers} format names for advert messages, more info below.
    • Added Portuguese and French rules command triggers. (To mostly work as examples)
    • Removed console messages on Unload.
    • Fixed rules commands description not showing on HelpText.
    • Minor tweaks and fixes.
    - The issue with the Joining messages was an Oxide issue regarding webrequests, it requires the latest Oxide build to fix it.
    - Various settings or messages names have been changed/re-named, as result their values will be forced to their defaults.
    - New Advert messages format names:
    {players} - Number of players online.
    {maxplayers} - Number of slots of the server.
    {sleepers} - Number of sleepers of the server.
    Usage Example: Players Online: <lime>{players} / {maxplayers}<end> Sleepers: <lime>{sleepers}<end>
    - To use the Admin Talk feature you must be an Admin and start talking in the chat with @.
    The @ is the trigger of the Admin Talk, though, you may change the trigger on the plugin settings, along with the Admin Talk tag, tag color, and text color, and you may also change the Auth level that is able to use the feature.
