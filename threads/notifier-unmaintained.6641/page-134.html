  1. NobodyFTW

    SkinN õ.Õ'.|. updated Notifier with a new update entry:

    3.1.0

    [ CHANGELOG ]
    • Added the following name formats for use in messages:
      • {player.level} - The player level
      • {player.progress} - Amount in percent of the current level progress
      • {player.xp} - The player experience
      • {player.title} - The player primary group title (Requires Better Chat | Oxide)
      • {player.titlecolor} - The player primary group title color (Requires Better Chat | Oxide)
      • {player.namecolor} - The player primary group player name color (Requires Better Chat | Oxide)
    • Added BetterChat support
    • Added new Advert Messages random cycling system
    • Added Enable HelpText Support setting
    • Tweaked Welcome Message to show exactly when the player wakes up after connecting to the server
    • Tweaked Scheduled Messages to scan messages right when the minute starts
    • Tweaked player country web-request algorithm for improvements
    • Tweaked English and Portuguese default rules strings
    • Tweaked most player side name formats, check the Notifier for Rust - FAQ | Oxide for more info
    • Tweaked /rules <country> argument so its possible to use country names instead of just country codes (e.g: english, french, german, etc...)
    • Tweaked {server.ip} to show as 0.0.0.0 when returns nothing
    • Tweaked Simple Color Format to remove <size=%> and </size> formats in console messages
    • Fixed Scheduled Messages not working at all
    • Fixed Incoming Helicopter message not working
    • Removed Message Of The Day feature
    Click to expand...
    - This update requires you to delete any older configuration file in order to work properly. I am sorry for the inconvenience
    - This update does major changes to the Players Database, therefore the plugin will now save the data in a new Notifier_PlayersData_v2 file, making the old file now obsolete
