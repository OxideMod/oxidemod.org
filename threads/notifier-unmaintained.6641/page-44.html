    Hey Botmon,
    I had the same problem and solved it the "hard way":
    1.) Download the latest version of Notifier
    2.) Stop your Rust server
    3.) Delete the notifier.py from your server directory
    4.) Upload the new file
    5.) Restart the server

    It's kinda nasty you have to restart the server, but at least this solved the problem for me, maybe it helps you as well.

    In the config file around line 120 (depends how many user defined adverts you added at the beginning of the file), look for the setting
    Code:
    "ADVERTS INTERVAL": 5,
    The 5 is for 5 minutes, change it to the amount of minutes you want.
