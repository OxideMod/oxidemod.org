  3. NobodyFTW

    SkinN õ.Õ'.|. updated Notifier with a new update entry:

    Notifier | Message Of The Day Feature!

    [ CHANGELOG ]
    • Added Message Of The Day feature
    • Added MOTD timer, to set the time in minutes for the MOTD interval
    • Added Enable MOTD Cycle to settings section, which enables/disables the MOTD cycle
    • Added /motd command, for users to see the MOTD, or for Admins to change the MOTD message
    • Added /motd <message> option for Admins only, to change the MOTD message in-game
    • Added MOTD to colors section
    • Added MOTD Prefix to settings section, for a customized prefix for MOTD
    • MOTD uses the same full list of Name Formats available for Adverts and Scheduled Messages
    • Rules lines are no longer enumerated automatically
    • Configuration dictionary is now kept in a different variable in the plugin class initialization for more flexibility working with the Update System
    • Minor tweaks
    Click to expand...


    - This update only does some additions to the configuration file, no resets required
    - The Admins /motd <message> changes the message in-game and saves the new message in the config file, this way Admins don't need to go within the plugin config to change it. It is also possible to add a message as you would in the config file using the color system that Notifier features.
    - This update also provides a fix for some configuration files that are missing the Scheduled Messages section
