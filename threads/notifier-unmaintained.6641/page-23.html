    There is a difference, the prefix will only appear in global messages (messages that are shown to all players at once, like connected, disconnected and adverts) and player/system messages (messages specifically to inform or notify something to a player, like welcome messages or seed command). And then there are the messages which I like to call board messages, which mostly are from commands like rules or players list that show a row of messages, in these board messages there is a title instead of the prefix it self which includes the plugin name, and that cannot be changed.
