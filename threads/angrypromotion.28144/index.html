    AngryPromotion - Automatically add users to a group if they have a specific word/phrase in their steam name

    View attachment 42179

    Do you want to be able to give rewards for people who have your server or website name in their steam name? Then this is the plugin for you.

    All you have to do is create a new group, give it permissions (if you want to) then put that group name into the config under "Group", then set the word/phrase, and you're done! For more information check out FAQ
