    k1lly0u submitted a new resource:

    Airstrike - Call an airstrike using a supply signal/chat command

    Call an airstrike either by throwing a supply signal or using a chat command (for players with permission &/or admin)

    ** This will over-ride the supply signal!, but only for the players with permission. If you want to disable the override set "Use supply signals to call strike" to false in the config

    Chat Commands
    callstrike - Will call a strike on your position
    callstrike "playername" - Will call a strike on a player

    Permissions
    can.airstrike -...
