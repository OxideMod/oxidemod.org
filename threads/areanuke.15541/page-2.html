  2. Noviets

    Objects created with the hammer are different, as they use the ConstructionManager. So be careful around buildings. If you remove the construction manager, it will remove the entire building (All construction attachments within that ownership cell). It will remove all non-hammer placed objects at distance correctly, but buildings are using the ConstructionManager. The first construiction object placed within an owership cell will create the ConstructionManager if one does not exist (created with the hammer). However things on the floor or objects placed from the hotbar etc will be fine.

    Hope that answers your question mate.
     
    Noviets, Oct 2, 2016

  4. Noviets

    Noviets updated AreaNuke with a new update entry:

    1.0.2

    Fixed:
    AreaNuke now checks for disabled/destroyed networkviews of gameobjects instead of listing gameobjects. This will fix getting strange results when destroying objects, especially when doing so many times within the same area.

    Items within the range will now display and destroy correctly.

    New Feature Added:
    An additional paramter can now be supplied to destroy a certain object within a range.
    For example: /areanuke 10 Roach
    Will destroy all roaches within 10 meters of you.
