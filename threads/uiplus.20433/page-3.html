    Sorry for the time in between responses I was away for a little. If you just use the default config file without changes that I don't have to see it.

    The problem seems to be that when downloading the icons it has problems to keep the connection going. This can have many issues.

    The solution is to download the icons locally and store them there. The download links are; the player icon (http://i.imgur.com/UY0y5ZI.png), the sleeping icon (http://i.imgur.com/mvUBBOB.png), and the clock icon (http://i.imgur.com/CycsoyW.png). Now you have downloaded them locate your data folder. It is located in the same folder as the folders to the plugin scripts and config files. Create a folder called "UiPlus" inside the data folder and drop the three icon files in there. Now all you have to do after that is change the uri address in the configuration file to the same names of the icons. For example; "clock.png".
