    Server Tags - Adds specified tags to the existing server tags


    Server Tags is a mostly useless plugin that is mainly here as a basic example of the BuildServerTags hook. It adds the specified tags to the existing server tags. The addition of these tags won't really do anything or show for normal Rust clients, but could potentially be used by server listing sites and such that parse the tags from each server.
