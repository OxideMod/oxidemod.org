    Hey Oxidemod community!

    Im Exel80, 21 Year old Finland guy.
    I found this forum ~2 year agon, but made my first plugin 2 day ago.

    I have pretty big project, but dont know is it worth to do it. That's why i made this thread, to ask you opinion!
    My project is to make "Global Banning System". Yes, i got idea from mcbans.com & Enhanced Ban System.

    Every server which use this system, share them bans to other server. If one server catch and global ban cheater/hacker, then they can't join to other servers which use this Global Banning System. Of course there will be Local and Global bans, more about those later.

    My first name idea was OxideBans.org, but Wulf suggested a more generic name.
    So we also need think new name.

    You don't have to worry about hosting large database or banlists. Only what you need is to upload plugin & register your server to get API key.

    Main Features:
    • Going to support all Oxidemod games! (Rust, Rust Legacy, Hurtworld, 7 days to die, etc.)
    • Add own community (Server owner can add multiple server to own "community"/group)
    • Admin chat commands: /ban /tempban /mute /unmute /unban /kick
    • Local ban: Only banned at server/community/group
    • Global ban: Banned from all servers which use this system

    Those are the MAIN features, i haven't planned this any further.
    As you can think, this cant work without dedicated host server and it cost.
    So what you think, should this cost per month or should i just delete this project and continue my others?

    Thank you and sorry for long post.
