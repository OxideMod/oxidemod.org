    SPooCK submitted a new resource:

    Anti-Dupe - Prevent player to stack-duplicate items

    Anti Item Stack Duping
    Recently i notice a stack-dupe exploit with was increasing the players Wood, Stones, Metal and etc (mostly all stacking items) by double each time. Which was not only making the player cheat-exploit the stack items but also was causing lag and was the reason for some server freezes. So I've coded this plugin for my server and I think is a good idea to share with the community so we all can have some protection against this...
