  2. Quansheng

    you would use the killbyblueprint or killbytype command as follows...

    /killbyblueprint [blueprint] (radius) ("silent")
    rok.command.objects.killbyblueprint
    Kills all entities in a radius from you. Everything with the blueprint specificed will be destroyed. Only objects in loaded pages will be killed.

    /killbytype [type] (radius) ("silent")
    rok.command.objects.killbytype
    Kills all entities in a radius from you. Everything with the type specificed will be destroyed. Only objects in loaded pages will be killed.
