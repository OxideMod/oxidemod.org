    NoCost - Choose a materials that won't be consumed while crafting.

    About
    This is my first "real" plugin I've ever made and this will prevent game consuming a item(s) on crafting. The list of items is configurable. You can use item shortname i.e: metal.fragments. Plugin supports any speed of crafting (i.e instant craft, normal craft). I've added a item duplicate prevention like that the already refunded materials will be taken from the player if he cancels the crafting task so he cannot abuse the original "refund" system to have crazy amount of some...
