  4. Nogrod

    change the time entry of the blueprint, here from 120 to 60:
    Code:
    {
  "amountToCreate": 1.0,
  "defaultBlueprint": true,
  "ingredients": [
    {
      "amount": 300.0,
      "shortname": "wood"
    }
  ],
  "isResearchable": true,
  "rarity": "Common",
  "targetItem": "spear_wooden",
  "time": 60.0
}
