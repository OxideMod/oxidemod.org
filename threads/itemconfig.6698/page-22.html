  14. Deicide666ra

    Crushed, you have to understand these are from the Rust datafiles and these are shared between the client and server versions of Rust. Some of the settings are used by the client only like speed, range and throwable, others only server side, etc. Plugins can generally only affect the server side of things which explains why several of these have no effect. They are nonetheless interesting values to see. For instance, I have a plugin on my server that uses the swing speed values to alter the gathering rate on the salvaged axe. It doesn't change that value but it uses it to calculate a ratio compared to the hatchet.

    I haven't tried adding/removing mods, but I assume the same applies here. If you add a "Gathering" mod to a piece of clothing for example, it will have no effect because the clothing class does not use that mod (even if it's loaded). This is just a silly example, but what I'm trying to say is that the data does not dictate the behavior.
     
    Deicide666ra, Sep 5, 2015
