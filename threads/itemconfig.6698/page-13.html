  12. Nogrod

    In the config there is saved the game protocol version as "Version" and the config version as "ConfigVersion" which is checked on load of the plugin. On an update (new game version or new config version) the config is moved to .old and an new one is created since the old is not compatible with the game anymore...but you can still merge the two configs with a program like WinMerge and see what was changed...
