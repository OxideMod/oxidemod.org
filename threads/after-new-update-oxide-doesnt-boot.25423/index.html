  8. Wulf Community Admin

    When you see errors/warnings such as "The referenced script on this Behaviour (Game Object 'mp5.viewmodel') is missing!", that generally means you've mixing Rust DLLs (generally Assembly-CSharp.dll as we patch that). This can happen by not updating Rust before installing a newer Oxide build meant for a newer Rust version, or using an old Oxide build on a newer Rust version.
