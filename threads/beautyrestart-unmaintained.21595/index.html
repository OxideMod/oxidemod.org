    Reynostrum submitted a new resource:

    BeautyRestart - Restart the server with a cooldown GUI timer

    BeautyRestart plugin allow admins and players with permission to restart the server with a cooldown GUI timer.

    This plugin use the default "quit" command. It means that if your your .bat file is well configurated it will restart the server. If not, it will shutdown and terminate the process.

    Permissions
    This plugin uses Oxide's permission system. To assign a permission, use grant user <username|steamid> <permission>. To remove a permission, use revoke user...
