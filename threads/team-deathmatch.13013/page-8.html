  1. k1lly0u

    @GSTYLEZ, I am posting my response to your pm here so others can see

    The way multiple spawn files works has changed. There is instructions for this on the EM overview. When making your spawn files they need _a and _b at the end of the file names. When setting your auto events you need to set the TEAM A spawn file. TDM, and other multiple spawn file events will take that spawn file name and search for the matching team b file.
    ex. Set team A spawnfile name to 'tdm_spawns_a', TDM will automatically search for 'tdm_spawns_b'. If its not found the game will not run
