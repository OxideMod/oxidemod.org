  10. ♠✪Misty✪♠

    Misty from rustled gaming here, had script errors when using /remove with various additional HUD plugins installed, to fix it not showing up and getting Parent Overlay issues,
    Change:
    Code (Text):
    ""parent"": ""Overlay"",

    To

    Code (Text):
    ""parent"": ""HUD/Overlay"",

    Just find the line with the top one and add HUD/ into it, will auto refresh no need to oxide.reload plugins.
