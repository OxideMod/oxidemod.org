  14. FrankNedz

    About building blocks refunds (based on percentage), I thought the plugin did not work as expected, but it does. The docs are just not extremely clear, in that, when you have a "sub item" specified, it will refund on AMOUNT. Here's what I mean:
    stone: 50, -> will refund 50% stone
    But
    Code:
    "Metal": {
    "metal fragments": 50
}
    will refund 50 frags !

    So for those you just have to remove the "detail" sub-item, like so:
    Code:
    "Metal": 50,
"TopTier": 50,
and so on..
    and it'll refund 50% instead of 50 units.

    Cheers
