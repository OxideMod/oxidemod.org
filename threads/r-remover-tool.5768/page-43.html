    I think this has to do with the REFUND feature.

    Code:
    "Remove - Refund": true,
  "Remove - Refund - Deployables": true,
  "Remove - Refund - Percentage (Structures Only)": {
    "0": "100.0",
    "1": "80.0",
    "2": "60.0",
    "3": "40.0",
    "4": "20.0"
  },
  "Remove - Refund - Structures": true
    If refund is TRUE then the player will receive X% resources back when removing depending on which grade it is.

    0 = twg (100% resources back)
    1 = wood (80% resources back)
    2 = stone (60% resources back)
    3 = metal sheet (40% resources back)
    4 = armor (20% resources back)
