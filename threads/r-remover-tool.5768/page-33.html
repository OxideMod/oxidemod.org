  8. Xtadeus

    @maksiyshuk : are you writing to me ?
    I made the change on the .cfg file but people can still use /remove for 30 seconds.

    The only solution ,as admin, that I found and that can help others is to modify this in the .cfg file :


    Code:
      "RemoveTimer": {
    "default": 0,
    "max": 120
    With this 0 : nobody can use it. ( even the admin ) but you can modify when necessary, reload, do the remove, then change the value, and reload again.
