  17. Brengun

    Rather than getting mad, just ask how.
    Getting mad doesn't fix it and remember people here are helping for free.
    Also reading and looking at configs will help.

    Ensure you have RUSTIO installed and use the clans plugin: Rust:IO Clans for Rust | Oxide for RUSTIO to auto add clan mates in
    You will need to use the latest file sqroot created on page 70 and it is best to only make these sort of changes on a fresh map wipe as the new feature sqroot added for remove without Building Owners requires a wipe after the RUST update from last week.
    After making these changes do some testing to make sure it all works.
    In the config for remover choose these options

    "Remove - Access - Use RustIO & BuildingOwners (Building Owners needs to be true)": true,
    "Remove - Access - Use ToolCupboards": false,
