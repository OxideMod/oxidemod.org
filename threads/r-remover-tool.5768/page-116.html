    Click to expand...
    EntityOwner for Rust | Oxide Is a Admin tool, to provide Ownership details and such about buildings.

    Building Owners for Rust | Oxide Is a plugin that changes the default ownership of buildings in the game.

    Using EntityOwner option, merely uses the Rust Default ownership as explained below

    Remove - Access - Use Entity Owners
    This will allow players to remove what they BUILT THEMSELVES. This is not the same as being the Building Owner. So if you build a wall inside someones base, you will be able to remove it, but not the other parts of the building (obviously)

    using BuildingOwner option uses Building Owners for Rust | Oxide plugin as explained below

    Remove - Access - Use Building Owners (You will need Building Owners plugin)
    Only let the Building Owner to remove building parts. The Building Owner is the one that first built the building. So someone that builds on his base, he will be able to remove what was built.

    For Friend Support at this time you must have RustIO installed, but it seems Reneb is updating possibly
