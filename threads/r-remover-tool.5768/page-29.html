  12. insane8472

    Thought i'd upload this for anyone else that wants to use it. After studying and learning the basics, I fixed a few things. May or may not be the cleanest code, but it works all the same. Working just fine on my own server.

    - cantRemove message no longer used. noToolCupboard and noToolCupboardAccess remain unchanged. noFriend message now used for RustIO friends errors.
    - Players can no longer remove resources. No error message is displayed and they will gather resources normally by hitting them.
    - Players can now remove deployed items such as crates and furnaces when using RustIO friends permission. These items still and always require tool cupboard access to remove, regardless of permissions.
    - You can now use cupboard access and RustIO friends permissions together, both conditions are checked. The appropriate error messages will be shown for each.
