    @Reneb, error is being thrown because of the timer change, it used to just be a "Rcon Exception, Ambiguous matchin in method resolution". But the timers have been modified to throw actual errors and destroy the timer when it runs in to an exception. Only happens the first time the plugin is loaded and is caused by
    Code:
        timer.Once(0.1, function()
        nulVector3 = new(UnityEngine.Vector3._type,nil)
    end )
    So by removing this from the init the current problem is solved, but you still need your nulVector so I moved it to OnServerInitialized() but due to some bug (not sure if it's Oxide or Rust, the first time you try to create the null vector it fails so I tossed in some lua error handling.
    Code:
    function PLUGIN:OnServerInitialized()
    status, err = pcall( new, UnityEngine.Vector3._type, nil)
    nulVector3 = new( UnityEngine.Vector3._type, nil )
end
