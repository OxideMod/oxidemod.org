    Click to expand...
    In the config change line
    Code:
    {% raw %}
    "Remove - Auth - AuthLevel - Normal Remove": 0,
    {% endraw %}
    To
    Code:
    {% raw %}
    "Remove - Auth - AuthLevel - Normal Remove": 1,-OR-"Remove - Auth - AuthLevel - Normal Remove": 2,
    {% endraw %}
