  4. kenna

    Hi, you havent installed the dependency modules.

    You can use npm install (this will installed the required dependency from package.json)
    or you can do this command:
    Code:
    npm install discord.js request
Than try to start it:

node app.js
    If you get an error login inncorrect, you have typed in wrong token (Do not use client secret, client secret is only for inviting the bot)
    If you also are thinking about install it as a service, do the above step first, see if its starts, than you need to install the required dependency
    Code:
    npm install -g node-windows
npm link node-windows
node installSVC.js
    Hope this helps
