  2. Wulf Community Admin

    Plugins would not affected your server listing or not, that is managed by the Rust client, Facepunch, and Steam. If your server isn't listed, then either the server is set to not use EAC/VAC (server.secure 0), it's blacklisted, or Steam can't see it.
