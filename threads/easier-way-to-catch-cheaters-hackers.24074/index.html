  1. Chief Greg

    As a server owner I know the struggle of trying to find evidence of a player hacking. The game doesn't allow admins to use the necessary tools to tell whether someone is cheating or not. While the spectate system is a good tool it doesn't do any justice unless the cheater is blatantly cheating (Spin-bot ect.) There needs to be a way to see a first person point of view from the spectate menu when you hit F3. Otherwise cheaters can pretty much get away with anything. I'm not sure if a plugin can be made to make cheating easier to detect or if a first person spectate camera could be made into a plugin. I honestly think this should be a Vanilla feature in the game.
