    A note on Anticheat and Antihack.
    So far with months of 1000's of players using this on my server, the Rust Anti-Cheat system does not seem to flag it. No player has reported any issues to me.

    But, the Server AntiHack settings will see players movement as a noclip violation and start adding up violation points. By default, server settings do not enforce this or kick/punish for it. But I usually turn off the antihack via server settings for noclip.

    But in the end, use your own judgement. do the research on Server Antihack settings and Rust Anticheat and decide for yourself
