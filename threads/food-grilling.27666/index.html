    Food Grilling - Add physical food on the grill

    This is a pretty simple plugin it just adds meat onto the grill. Inside the config you can set it to put any meat on the grill, but it won't fit except pork.




    Config:
    Code:
    {
  "Grill": {
    "allChicken": true
  }
