    Mughisi submitted a new resource:

    Increased Building Rate - Basic plugin to increase the building rate.

    Increased Building Rate - 0.1.0
    This plugin allows admins to modify the building rate exponentially, just modify the building rate with /build 3 to increase the building rate to 3x normal!

    While there is no real limit to the rate you can set this, be aware if you set it too high errors will occur and if you set it to more than 9 the buildings will instantly be level 2 or higher depending on the multiplier.

  20. jf3000

    hmmm I had to edit the users.cfg file on the server and add

    ownerid steamid "steamname" "Owner"

    Find Steam 64 ID: http://www.steamidconverter.com

    To add a owner:
    ownerid STEAM64ID "Players In Game Name" "Owner"


    To add an Moderator:
    moderatorid STEAM64ID "Players In Game Name" "Moderator"
