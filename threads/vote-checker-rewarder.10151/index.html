    Pho3niX90 submitted a new resource:

    Vote Checker - Give rewards to users for voting on rust-servers.net

    Checks votes on http://rust-servers.net and rewards users for their votes. The more they vote the more they receive.

    To setup your api key use
    /rewardconf api youServerApiKey

    To setup your serverID use
    /rewardconf serverid yourServerID

    To get items
    /itemlist - This shows all items that you can reward

    To setup your reward method use
    /rewardconf rewardtype 0
    0 = Reward users based on their total monthly votes for the current month
    1 = Reward users...
